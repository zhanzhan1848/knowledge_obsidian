## 研究目标

基于 Alexeev 流体方程建立湍流通道流的解析框架，关注流向和横向速度分量的耦合行为。

## 方法

### 解析框架
1. **平均流向速度**: 层流（抛物线）分量 + 非线性湍流贡献的叠加
2. **横向速度分量**: 简化形式的控制方程

### 验证范围
- **Reynolds 数范围**: 3×10³ ≤ Re ≤ 3.5×10⁷
- **偏差**: 中等 Re 下约 1%，最高 Re 下约 3%

## 核心发现

### 1. 扭结型解（Kink-Type Solutions）
- 流向湍流分量承认一族扭结型解
- 特征：局部单调过渡，分离近乎均匀速度区域
- **解释为条纹结构的解析表示**

### 2. 条纹属性预测
| 属性 | 预测 |
|------|------|
| 间距 | ✓ 一致 |
| 厚度 | ✓ 一致 |
| 强度 | ✓ 一致 |
| 流向范围 | ✓ 一致 |

### 3. 物理机制
横向速度波动 → 相干流向结构（条纹）涌现

## 创新点

1. **统一描述**: 平均速度剖面 + 二次流 + 条纹形成
2. **解析方法**: 超越传统经验模型
3. **实验验证**: 与通道和管道流实验数据一致

## 控制方程

基于 Alexeev 方程：
- 简化的 Navier-Stokes 近似
- 壁面边界条件
- 湍流分量非线性耦合
