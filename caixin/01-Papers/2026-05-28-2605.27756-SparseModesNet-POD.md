## 核心贡献

> 提出 SparseModesNet：结合 POD 编码和 LassoNet 解码的稀疏模态选择框架

1. **线性编码 + 非线性解码**：POD 模态线性编码，LassoNet 非线性解码
2. **层次稀疏约束**：通过残差连接和线性跳跃层强制层次稀疏
3. **同时模态选择和映射学习**：最小化重构误差的同时选择信息性 POD 模态
4. **湍流通道流验证**：Re_τ=5200 重构误差降低 51-78%

---

## 技术方案

### 核心思想

解决 POD 在缓慢衰减 Kolmogorov n-width 问题（如对流主导和湍流）上的局限。

### 框架结构

```
Input → [Linear: POD Modes] → [Nonlinear: LassoNet Decoder] → Reconstruction
              ↓
        Hierarchical Sparsity (residual + skip layers)
```

### 关键技术

| 技术 | 说明 |
|------|------|
| POD (Proper Orthogonal Decomposition) | 能量最优线性子空间 |
| LassoNet | 层次稀疏网络结构 |
| Sparse Mode Selection | 同时选择和信息性模态 |
| Reconstruction Error Minimization | 端到端优化 |

## 公式

### POD 编码

```math
\mathbf{a} = \mathbf{U}^T \mathbf{u}
```

### LassoNet 解码器损失

```math
\mathcal{L} = \|\mathbf{u} - \mathbf{U} \mathbf{D}(\mathbf{a})\|^2 + \lambda \|\mathbf{w}\|_1
```

## 实验结论

- **Benchmark 验证**：对流主导和混沌流的基准测试匹配或超越 SOTA
- **湍流通道流**：Re_τ=5200 重构误差降低 51-78% vs 多项式流形方法
- **可解释性**：通过物理有意义的模态选择保持可解释性
