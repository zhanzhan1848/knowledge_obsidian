## 研究背景

### 稀薄气体动力学 (Rarefied Gas Dynamics)
```
Kn = λ/L  (Knudsen 数)
  > 0.1: 稀薄, DSMC 必需
  < 0.01: 连续介质, N-S 可用
```

### DSMC (Direct Simulation Monte Carlo)
Birds (1960s) 开创的粒子方法:
```
真实分子 → 模拟粒子 (稀疏采样)
碰撞: 随机抽样
演化: 拉格朗日跟踪
```

### 粒子合并问题 (Particle Merging)
DSMC 中当粒子数过多时需要合并:
```
合并触发: 本地粒子数 > N_max
目标: 减少计算量
挑战: 合并误差会传播并累积
```

## 核心问题

### 传统合并方法的缺陷
```
1. 简单权重平均: 
   丢失速度矩 (→ 宏观量误差)
2. 原始权重:
   引入数值噪声

→ 需要: 同时保持多个物理矩
```

### 理想目标
保持以下物理量 (碰撞不变量):
```
1. 质量:   Σ m   (零阶矩)
2. 动量:   Σ mv  (一阶矩)  
3. 能量:   Σ ½mv² (二阶矩)
4. (可选) 更高阶矩
```

## 核心创新

### 非负最小二乘 (NNLS) 框架
将粒子合并建模为优化问题:

```
min ||W_new * N_new - Σ_i W_i * N_i||²
s.t.
  N_new ≥ 0 (非负粒子数)
  矩约束: A * N_new = b (保持指定矩)
```

其中:
- W: 粒子权重
- N: 粒子数
- A: 矩矩阵 (质量/动量/能量行)
- b: 目标矩值

### 任意矩保持的推广
```
可保持的矩:
  - 空间矩: Σ x^n
  - 速度矩: Σ v^n
  - 混合矩: Σ x^i v^j

→ 用户可选择保留哪些矩
→ 灵活性与精度的权衡
```

### 碰撞率保持扩展
两种模式:
```
1. 精确碰撞率保持:
   碰撞率 = 精确 DSMC 公式
   (但可能违反其他矩)
   
2. 近似碰撞率保持:
   放松约束, 允许小幅偏差
   (更好的整体精度)
```

## 数学框架

### 问题定义
```
M 个父粒子 → N 个子粒子 (N << M)
父粒子: 权重 w_i, 速度 v_i, 位置 x_i

寻找: 子粒子权重 w'_j, 速度 v'_j

约束:
  Σ_i w_i = Σ_j w'_j (质量守恒)
  Σ_i w_i v_i = Σ_j w'_j v'_j (动量守恒)
  Σ_i w_i |v_i|² = Σ_j w'_j |v'_j|² (能量守恒)
```

### NNLS 求解
```
使用 Lawson-Hanson 算法求解 NNLS
→ 迭代求解约束最小二乘
→ 收敛快, 适合在线计算
```

## 数值验证

### 测试算例

#### 1. 均匀松弛
```
初始: 非平衡分布
演化: DSMC 碰撞 + 合并
验证: 宏观量 (密度, 温度, 速度) 保持
```

#### 2. 激波结构 (Sod 问题)
```
配置: 1D 激波管
Kn = 0.1 (稀薄)
验证: 合并误差不累积
```

## 关键结果

### 误差对比
```
NNLS 合并 vs 传统方法:
  → 关键宏观量误差显著降低
  → 无合并误差传播累积
  → 长时间模拟稳定性提升
```

## 计算成本分析

| 方面 | 描述 |
|------|------|
| 合并频率 | 粒子数超阈值时触发 |
| NNLS 求解 | O(M·N²) per merge event |
| 精度 | 任意矩保持可调 |
| 额外存储 | 矩矩阵 A, 目标向量 b |

## 创新点总结

1. **首次**将 NNLS 引入粒子合并问题
2. **任意矩保持**: 用户可指定保持哪些物理矩
3. **碰撞率保持**: 精确/近似两种模式
4. **稳定性**: 消除合并误差累积
