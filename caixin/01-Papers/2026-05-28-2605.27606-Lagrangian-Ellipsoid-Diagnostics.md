## 核心贡献

> 提出 Löwner-John 变形云图方案作为不可压缩随机流的拉格朗日诊断工具

1. **变形粒子云诊断**：通过最小体积包围椭球的惯性张量和粗粒化速度梯度总结粒子云
2. **随机流测试**：在 2D 等谱不可压缩高斯-赫尔德有限时间相关速度场验证
3. **降阶建模**：将 aspect ratio 动力学分解为对齐应变源和 Löwner-John 残差
4. **物理约束闭合**：应变和涡量作为尺度依赖随机驱动，von-Mises 偏置表征对齐

---

## 技术方案

### 核心思想

粒子云在紫外尺度释放，每个时刻用两个对象总结：
- 最小体积包围椭球的惯性张量
- 该椭球上粗粒化的速度梯度

### 关键技术

| 技术 | 说明 |
|------|------|
| Löwner-John Ellipsoid | 最小体积包围椭球计算 |
| Ornstein-Uhlenbeck Fourier Modes | 随机速度场生成 |
| Kolmogorov Scaling | 2/3 律，惯性范围 |
| von-Mises Distribution | 表征应变-椭球对齐统计 |
| Affine Feedback | 尺度依赖反馈闭合 |

---

## 公式

### 速度场生成

```math
\mathbf{u}(\mathbf{k}, t) = \text{Ornstein-Uhlenbeck process with } \langle u^2(k) \rangle \propto k^{-2/3}
```

### Aspect Ratio 动力学

```math
\frac{dA}{dt} = \underbrace{(\text{aligned strain})}_{\text{source}} - \underbrace{(\text{Löwner-John relaxation})}_{\text{sink}}
```

## 实验结论

- **Aspect Ratio 饱和**：统计饱和的椭圆率
- **尺度依赖**：感知梯度呈现清晰尺度依赖
- **应变-涡量平衡**：近似普通张量级别平衡
- **湍流应用**：提供从粒子云数据到可解释有限维随机动力学的便携路径
