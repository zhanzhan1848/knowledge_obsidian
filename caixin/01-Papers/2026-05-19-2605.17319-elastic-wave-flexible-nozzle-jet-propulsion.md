# 2605.17319 — Elastic Wave Propagation in Flexible Nozzles for Jet Propulsion

## 基础信息
- **arXiv**: [2605.17319](https://arxiv.org/abs/2605.17319)
- **日期**: 2026-05-18
- **分类**: physics.flu-dyn
- **关键词**: fluid-structure interaction, FSI, vortex formation, jet propulsion, elastic wave, Re=4000

## 核心创新

### 背景
受头足类喷水推进通过柔软漏斗的启发，研究柔软圆柱形喷嘴中的弹性波传播和能量交换。

### 方法
- **3D 双向流固耦合 (FSI) 模拟**
- 喷嘴刚度范围：$Eh = 75-500$ N/m
- 脉动射流入口，$Re \sim 4000$

### 关键发现

| 刚度 $Eh$ | 影响 |
|-----------|------|
| 增加柔度 | 变形波速降低（符合 Moens-Korteweg 标度）|
| 延长喷嘴膨胀阶段 | 增强射流卷吸和弹性储能 |
| 抑制早期剪切层卷起和涡形成 |  |
| 收缩阶段 | 储存弹性能释放，增强射流加速和涡形成 |

**最柔喷嘴结果**（vs 刚性喷嘴）：
- 主涡环环量：**+52.13%**
- 涡对流距离：**+9.00%**
- 峰值出口动能通量：**×4.62**
- 脉冲增强：**+61%**（原文截断）

## 湍流/ CFD 关联
- **流固耦合 (FSI)** — 双向耦合
- **涡环形成** — 射流涡动力学
- **柔体推进** — 生物启发设计
- 高 Re 数 ($Re \sim 4000$) 湍流

## 数值方法
- 3D FSI 模拟
- 脉动入流边界条件
- 弹性波 (Moens-Korteweg) 标度
