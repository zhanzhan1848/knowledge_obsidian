# Fully Discrete Active Flux Method for Compressible Euler Equations

- **arXiv**: [2605.12915](https://arxiv.org/abs/2605.12915)
- **Date**: 2026-05-12
- **Subjects**: math.NA, physics.flu-dyn
- **URL**: https://arxiv.org/pdf/2605.12915

## 核心创新点

1. **Active Flux 方法**: 完整的全离散实现
2. **输运声学增量**: 基于输运声学增量的 2D 可压缩欧拉方程
3. **ROE演化算子框架**: 基于 Roe 的演化算子 formulation

## 方法论

### 数值方法

| 组件 | 描述 |
|------|------|
| 单元平均值 | 通过unsplit通量求积更新 |
| 原始点值 | 通过声学和平流子求解器演化 |
| 声学增量重建 | Q2 场（单元内） |
| 对流足点 | 在目标点评估重建场 |

### 理论保证

- 对于常系数冻结情况，点更新简化为输运组合
- 消除加性分裂缺陷
- 当声学和平流生成元对易时，精确无分裂冻结演化

## 数值验证

| 测试案例 | 阶数/精度 |
|----------|----------|
| 混合傅里叶波包 | 三阶点精度 |
| 等熵涡旋平流 | 三阶收敛 |
| 非线性高斯声学脉冲 | 保持径向对称性，近似三阶衰减 |
| 低马赫剪切层 | 相干涡量演化，超低熵耗散 |
| 未解析Kelvin-Helmholtz | 无限制器演化至晚期时间 |

## 优势

- 保持 Barsukow (2025) 的精确局部线性化声学演化算子
- 紧凑模板
- 保守单阶段平均值更新
