# Lagrangian analyses in turbulent Rayleigh-Bénard convection

- **arXiv**: [2605.22326](https://arxiv.org/abs/2605.22326)
- **Date**: 2026-05-21
- **Subjects**: physics.flu-dyn
- **URL**: https://arxiv.org/pdf/2605.22326

## 核心创新点

三维DNS模拟 Lagrangian 框架下的湍流 Rayleigh-Bénard 对流 (RBC)，Ra: 10⁵ → 10¹⁰，揭示拉格朗日热传输的极端间歇性。

## 控制方程/方法

- **Rayleigh数**: 10⁵ ≤ Ra ≤ 10¹⁰
- **Prandtl数**: Pr = 0.7
- **域尺寸**: L:L:H = 4:4:1
- **分析方法**:
  - 粒子加速度
  - 拉格朗日热传输
  - Q-R 不变量拓扑分析
  - 粒子对散布
  - 尺度相关拉格朗日涡粘度
  - PCA (主成分分析)

## 关键发现

1. **极端间歇性**: 单粒子可携带高达全局平均500倍的对流热通量
2. **高阶矩趋向Gaussian**: Ra增加时
3. **Q-R拓扑特征**: Q>0, R<0象限揭示尘魔式对流涡旋特征
4. **散布标度律**: 
   - 短时 t⁵ 类似阶段 (羽流驱动)
   - 持续 Richardson t³ 标度

## 数值方法

- **谱元法** (Spectral Element Method)
- **Heisenberg-Yaglom 行为** 恢复
- **Kolmogorov 尺度** 完全求解
