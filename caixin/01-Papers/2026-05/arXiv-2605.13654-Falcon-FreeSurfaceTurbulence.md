# Free-surface Deformations Induced by Three-Dimensional Turbulence

- **arXiv**: [2605.13654](https://arxiv.org/abs/2605.13654)
- **Authors**: Eric Falcon
- **Date**: 2026-05-13
- **Subjects**: physics.flu-dyn, nlin.CD, physics.ao-ph, physics.class-ph
- **URL**: https://arxiv.org/pdf/2605.13654

## 核心创新点

1. **实验表征**: 使用傅里叶变换轮廓术（FTP）在射流驱动湍流池中，对自由表面变形进行时空测量
2. **线性标度关系**: 表面变形标准差与水下速度波动呈线性标度
3. **双机制共存**:
   - 瞬态相干结构（如上升流）→ 低频大尺度谱分量
   - 对水下湍流压力波动的被动响应 → 幂律谱标度
4. **谱指数**: 波数谱和频率谱均呈现 ~-2.5 的幂律指数（≈ -7/3）

## 理论模型

- 建立了从自由表面到湍流压力波动的线性响应模型（传递函数）
- 包含波-湍流阻尼（wave-turbulent damping）
- 被动响应主导，而非波浪生成

## 关键发现

| 现象 | 规律 |
|------|------|
| 表面变形幅值 | σ ∼ u'（线性） |
| 表面谱指数 | -7/3（频率/波数） |
| 低频大尺度 | 瞬态相干结构贡献 |
| 高频小尺度 | 被动压力响应 |

## 物理意义

- 适用于湍流速度低于表面破波阈值的情况
- 揭示了自由表面湍流的新物理机制
