# 核心创新点

首次研究壁约束湍流中解析相干粒子（cohesive particles）的聚集、破碎和尺寸依赖输运。

## DNS 配置
5 组直接数值模拟，湍流通道流加载有限尺寸相干粒子，逐步增加相干强度。

## 群体平衡方程 (PBE) 框架

### 全局闭合
积分过完整域时，PBE 仅由聚集和破碎闭合。

### 局部失衡发现
壁法向方向上平衡不成立：
- 存在净聚集生产区
- 存在净消耗区

### 补偿机制
尺寸依赖的壁法向粒子输运补偿失衡。

## 物理图景
```
通道中心 → 大颗粒优先产生 → 迁移向壁 → 破碎
壁区域 → 小颗粒输运离开 → 生长 → 重新进入循环
```

形成平均循环：聚集-输运-破碎-再聚集
