## 核心贡献

> 一句话总结这篇论文的核心创新点：提出统一框架强制不可压缩连续方程作为硬约束，通过可微谱Leray投影和基于旋度的前推构造，确保流体模拟的物理一致性和长期稳定性。

1. **统一硬约束框架**：为确定性和生成式模型强制执行不可压缩连续方程
2. **可微谱Leray投影**：基于Helmholtz-Hodge分解，将确定性模型投影到无散度子空间
3. **无散度高斯参考测度**：通过基于旋度的前推构造，确保概率流的子空间一致性
4. **精确物理一致性**：在2D Navier-Stokes方程上实现了离散化误差级别的精确不可压缩性

## 技术方案

### 核心思想

针对基于学习的流体动力学模型常在无约束函数空间中运行导致物理不许可和不稳定模拟的问题，论文引入了一个统一框架，强制不可压缩连续方程作为硬约束。该方法解决了惩罚方法无法提供结构保证的关键问题，避免了伪散度和长期崩溃现象。

### 关键技术

| 技术 | 说明 |
|------|------|
| **确定性投影** | 可微谱Leray投影，将回归假设空间限制在物理可许可的速度场 |
| **生成式模型** | 基于旋度前推的无散度高斯参考测度构造 |
| **子空间一致性** | 通过构造确保整个概率流保持子空间一致性 |
| **物理约束** | 硬约束而非软约束，确保长期数值稳定性 |

## 公式

核心的Helmholtz-Hodge分解：

```math
u = u_{irrotational} + u_{solenoidal}
```

其中：
- u_{irrotational} = ∇φ (无旋部分)
- u_{solenoidal} = ∇×ψ (无散度部分)

Leray投影算子：

```math
P(u) = u - ∇(Δ^{-1}∇·u)
```

概率流的前推构造：

```math
μ_{DF} = (ψ_*) ∘ μ_0
```

其中ψ_∗是基于旋度的前推映射。

## 实验结论

- **数据集**: 2D Navier-Stokes方程数值实验
- **基线**: 传统神经算子 vs 无散度神经算子
- **结果**: 实现了离散化误差级别的精确不可压缩性，显著提高了稳定性和物理一致性
