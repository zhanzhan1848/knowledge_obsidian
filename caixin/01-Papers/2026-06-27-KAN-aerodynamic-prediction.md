# Kolmogorov Arnold Networks (KAN) for Aerodynamic Prediction

## 论文信息
- **arXiv**: 2606.xxxxx (搜索结果中)
- **作者**: Miguel Jaraiz, Fermin Gutierrez, Pablo Yeste, Miguel Sánchez-Domínguez, Eusebio Valero, Gonzalo Rubio, Lucas Lacasa
- **提交日期**: 2025年6月25日
- **分类**: Fluid Dynamics, Aerodynamics

## 核心贡献
评估 Kolmogorov Arnold Networks (KANs) 在流体力学代理建模中的性能，并与 MLP 和 GNN 进行对比。

### 研究背景
KAN 是最近提出的一种神经网络架构，在自然语言处理和计算机视觉领域引起了广泛关注。本研究首次系统评估 KAN 在气动代理建模中的表现。

### 研究方法
- **任务**: 预测亚音速和跨音速翼型的表面压力分布（空气动力学中的标准任务）
- **对比模型**: 
  - KAN (Kolmogorov Arnold Networks)
  - MLP (多层感知机)
  - GNN (图神经网络)

### 关键发现
- KAN 在流体力学代理建模中展现出与 MLP 和 GNN 不同的性能特征
- KAN 提供了 MLP 和 GNN 之间的一种折中方案
- 在某些条件下 KAN 表现出独特的优势

## 技术亮点
- 亚音速和跨音速翼型绕流
- 表面压力预测
- 代理模型对比研究

## 应用价值
- 空气动力学设计优化
- 飞行器气动性能评估
- CFD 加速预测
