## 核心创新点

- `amerta`：开源 Python 库，求解一维无摩擦 Saint-Venant 浅水方程
- **MUSCL 重建** + **minmod 斜率限制器**
- **HLLC 近似 Riemann 求解器**
- **两阶段 SSP-RK**（强稳定性保持 Runge-Kutta）时间积分
- **Numba JIT** 编译加速性能关键内核

## 验证案例（4 个典型 Riemann 问题）

| 问题类型 | 描述 |
|----------|------|
| 湿底溃坝 (wet-bed dam break) | 下游有水层 |
| 干底溃坝 (dry-bed dam break) | 下游干燥 |
| 双稀疏波 (double rarefaction) | 左右两侧稀疏 |
| 双激波 (double shock) | 左右两侧压缩 |

## 核心数值方法

### 离散化
- 均匀 Cartesian 网格
- MUSCL (Monotone Upstream-centered Schemes for Conservation Laws)
- minmod 斜率限制器 → TVD 保证

### Riemann 求解器
- HLLC (Harten-Lax-van Leer-Contact)
- 近似而非精确 Riemann 解

### 时间积分
- 两阶段 SSP-RK
- 强稳定性保持

### 后处理（6 组件流水线）

1. **时空拓扑量化**
2. **误差范数 + 经验分位数分解**
3. **自相似性塌缩分析**（解析 Riemann 波）
4. **积分范数演化**
5. **边界通量修正的质量/能量诊断**
6. **相平面分析**（深度-速度）

## 验证结果

- **离散质量守恒**：浮点精度
- **离散熵容许性**：恒满足
- **波曲线几何**：亚厘米精度（深度-速度相平面）
