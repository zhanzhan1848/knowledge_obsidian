# Scalable Ewald-Free BIE Framework for Periodic Stokes Flow via Hierarchical Proxy Sums

## 核心创新点

- **无需 Ewald 求和**的周期 Stokes 流 BIE 框架
- 仅使用**自由空间格林函数**（free-space Green's function）
- **无扩展线性系统**（extended linear system）
- 基于 **KIFMM (Kernel-Independent FMM)** 的代理源构建辅助基
- 通过**分层代理和**（hierarchical proxy sum）捕获远场贡献
- **净力为零兼容条件**（net-force-zero compatibility condition）保证绝对收敛

## 核心算法组件

1. **边界积分方程 (BIE)**
   - Stokeslet, Stresslet, Rotlet
   - 仅需自由空间格林函数

2. **KIFMM (核无关快速多极方法)**
   - 代理源放置于等价表面
   - 核无关实现

3. **分层代理和**
   - 远场图像箱贡献
   - 净力为零兼容条件 → 绝对收敛

4. **周期化预计算**
   - 仅依赖周期盒几何
   - 与核函数和内部表面无关
   - 可跨 Stokeslet/Stresslet/Rotlet 复用
