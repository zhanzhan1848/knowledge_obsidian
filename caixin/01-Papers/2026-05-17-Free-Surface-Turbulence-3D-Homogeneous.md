# Free-Surface Deformations Induced by Three-Dimensional Turbulence

> **arXiv**: [2605.13654](https://arxiv.org/abs/2605.13654) | **Date**: 2026-05-13 | **Author**: Eric Falcon

## 核心创新点

- 实验表征: 三维均匀各向同性湍流产生的自由表面变形
- 使用 **傅里叶变换轮廓术 (Fourier transform profilometry)** 在射流强迫湍流池中测量
- 表面变形标准差与水下速度波动呈 **线性标度**
- 表面变形频谱揭示两种机制共存:
  1. **瞬态相干结构** (如上涌) → 低频、大尺度
  2. **水下湍流压力的被动响应** → 幂律标度
- 波数和频率谱呈现相似幂律指数 (**-2.5**)，暗示湍流结构在自由表面的平流

## 实验配置

| 参数 | 值 |
|------|------|
| 实验装置 | 射流强迫湍流池 (jet-forced turbulent tank) |
| 测量技术 | 傅里叶变换轮廓术 (Fourier transform profilometry) |
| 测量量 | 表面高程场时空数据 |
| 湍流强度范围 | 宽范围 |

## 理论模型

- 建立基于传递函数的 **线性响应模型**
- 传递函数: 自由表面 → 水下湍流压力波动
- 包含**波-湍流阻尼**
- 预测: 
  - 时空谱形状
  - 幂律指数 (-7/3)
  - 被动响应主导波生成

## 物理机制

- 湍流速度低于表面破碎阈值时:
  - 低频: 瞬态相干结构（上行）主导
  - 高频/被动响应: 湍流压力被动响应主导
- 波数-频率谱相似性 → 湍流结构的平流传输
