## 研究背景

铁颗粒以其非挥发性燃烧模式在整个燃烧过程中保持分散相，导致典型铁粉燃烧器中的流动为颗粒负载湍流。优先浓度（Preferential concentration）是这种湍流流动中的普遍现象，导致颗粒聚集。

## 方法

### 直接数值模拟 (DNS)
- **域**: 立方体域 + 强制均匀各向同性湍流
- **氧化产物**: FeO

### 参数范围
| 参数 | 值 |
|------|-----|
| Kolmogorov Stokes 数 | St = 1, 10, 50 |
| 湍流 Reynolds 数 | Re_λ = 5, 10, 20 |
| 全局当量比 | φ = 0.25, 0.5, 0.75 |

## 核心发现

### 1. 当量比效应
- 增加 φ 显著延长燃烧完成时间

### 2. 初始分布效应
- **Poisson 分布**: 燃烧更快，峰值温度更高
- **聚类分布**: 温度演化平滑，峰值较小
- **聚类分布燃烧时间可延长 8 倍**（Re_λ=20, φ=0.75）

### 3. Voronoi 体积分析
- 高密度区域（cluster）的颗粒燃烧时间更长
- 燃烧时间与 Voronoi 体积 V_norm 呈强指数依赖
- "void" 区域呈渐近行为

### 4. 宏观结构重要性
- 多个聚类的接近程度（macrostructure）对燃烧时间延长有重要影响

## 物理机制

```
高密度区域 → 局部 O₂ 耗尽 → 燃烧时间延长
         ↓
    多个聚类接近 → O₂ 扩散受限 → 进一步延长
```

## 数值方法

- **湍流生成**: 强制均匀各向同性湍流
- **粒子追踪**: Lagrangian 方法
- **反应模型**: 非挥发性燃烧（颗粒表面反应）
