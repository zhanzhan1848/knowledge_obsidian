# 2607.02203: Self-explainable Operator Learning for Discovering Spatial Patterns in Fluid Flow

## Metadata
- **arXiv**: [2607.02203](https://arxiv.org/abs/2607.02203)
- **Authors**: Amirhossein Arzani et al.
- **Date**: 2026-07-02
- **Subjects**: cs.LG, physics.flu-dyn

## Research Type
Machine learning for physics — interpretable operator learning

## Problem Description
Overcome opacity of neural operator architectures by introducing self-explainable operator learning for fluid flow problems.

## Method
Reformulate operator learning as **linear combination of generalized functional linear models** expressed through integral equations.

Key features:
- Divide input domain into subdomains
- Compute localized integrals to evaluate contribution of each region
- **Direct interpretability**: links specific input regions to corresponding output patterns

## Applications
- Function-to-scalar: blood flow problems
- Function-to-function: unsteady aerodynamics

## Key Findings
1. Operator prioritizes regions with **strong feature gradients** — physically meaningful
2. Provides **embedded explainability** (no external tool needed)
3. Qualitatively agrees with post-hoc explainability methods
