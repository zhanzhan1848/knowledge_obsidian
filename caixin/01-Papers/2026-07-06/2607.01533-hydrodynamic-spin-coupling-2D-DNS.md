# 2607.01533: Two-dimensional simulations of hydrodynamic spin coupling in a two-rotor corral

## Metadata
- **arXiv**: [2607.01533](https://arxiv.org/abs/2607.01533)
- **Authors**: Tsorng-Whay Pan et al.
- **Date**: 2026-07-01
- **Subjects**: physics.flu-dyn
- **MSC**: 76D05

## Research Type
DNS (Direct Numerical Simulation) of 2D incompressible viscous flow

## Problem Description
Study hydrodynamic spin coupling in a two-rotor corral using DNS of 2D incompressible viscous fluid flow. An active rotor driven at angular velocity W and a nearby torque-free passive rotor selects angular velocity w through hydrodynamic torque balance.

## Key Parameters
- **Gear ratio**: Γ = w/W (distinguishes corotation from counterrotation)
- **Reynolds number**: Re = |Ω|r²/ν
- **Corral sizes**: C = 3, 4.5, 6

## Numerical Method
- **DLM/FD method** (Distributed Lagrange Multiplier / Finite Difference)
- Planar phase diagrams of Γ(G, Re) computation

## Key Findings
1. **Gap route at Re=20**: intermediate counterrotation band, wide-gap transition to corotation, gear-ratio magnitudes ~10⁻²
2. **Vortex dynamics**: sequence of vortex attachment, detachment, and merger
3. **Reentrant-like gap structure**: small-gap corotation region
4. **High-Re boundary discrepancy**: planar model displaces boundary in gap and alters surface-stress mechanism
5. **Mechanism**: crossing occurs by redistribution of integrated planar torque (not shear-competition)

## Relevance to CFD
- DNS validation of hydrodynamic coupling
- 2D vortex dynamics in confined flows
- **Keywords**: DNS, vortex method, 2D turbulence
