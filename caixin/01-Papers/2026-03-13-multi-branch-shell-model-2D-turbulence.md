# Multi-branch Shell Models of Two-Dimensional Turbulence

## 元数据
- **arXiv ID**: 2603.11892
- **提交日期**: 2026-03-12
- **作者**: Flavio Tuteri
- **URL**: https://arxiv.org/abs/2603.11892
- **PDF**: https://arxiv.org/pdf/2603.11892

## 核心创新
提出多分支 shell 模型，首次在 shell 模型框架下实现二维湍流的双级联（能量逆级联 + 涡量正级联）。

## 关键技术点

### Shell 模型改进
- **多分支结构**: 包含跨尺度几何层级组织
- **克服限制**: 传统 shell 模型无法产生双级联
- **热谱匹配**: 与二维流体方程热谱一致

### 级联特性
- **能量逆级联**: 向大尺度传递
- **涡量正级联**: 向小尺度传递
- **统计定态**: 双级联达到稳态

### 研究内容
- 局部传递分析
- 自相似性研究
- 非高斯性研究

## 理论背景
传统 shell 模型失败原因：
- 无法复现正确的热谱
- 缺少几何结构信息

## 数值验证
- 热谱与二维流体方程一致
- 双级联统计定态实现
