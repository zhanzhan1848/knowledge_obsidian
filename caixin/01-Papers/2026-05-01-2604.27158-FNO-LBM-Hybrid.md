# 2604.27158 - Hybrid Fourier Neural Operator-Lattice Boltzmann Method

## 论文信息

## 核心创新点

### 研究目标
提出基于 **Fourier Neural Operator (FNO)** 和 **Lattice Boltzmann Method (LBM)** 混合的加速计算框架,用于稳态和非稳态弱可压缩流。

### 关键技术

#### 1. FNO初始化加速LBM稳态收敛
- 使用FNO进行初始化,加速LBM达到稳态
- **加速效果**: 
  - 密度收敛加速 **70%**
  - 压降精度保持 **40%** 以上提升

#### 2. Super-time-stepping混合策略
- 将FNO rollout嵌入LBM时间推进
- 适用于非稳态流动加速

#### 3. 轻量级模型有效性
- 2.6M参数FNO在纯自回归模式下发散
- 混合耦合后误差降低 **96-99.8%**
- 匹配11.2M参数大模型的预测能力

## 数值方法

### FNO-LBM 混合框架
```
LBM时间步 → FNO rollout → LBM时间步 → ...
     ↑_____________________________↓
```

### 误差评估
- 全局和时空误差度量
- 100条轨迹验证
- 2D通用流动测试

## 核心发现

### 为什么混合有效?
1. **抑制误差积累** - 纯FNO长期展开误差发散
2. **保持物理一致性** - LBM提供物理约束
3. **小模型大能力** - 混合使轻量模型达到大模型精度

### 加速比
| 场景 | 加速效果 |
|------|----------|
| 稳态密度收敛 | 70% |
| 稳态压降精度 | 40%+ |
| 非稳态混合 | 显著提升稳定性 |

## 应用场景
- 多孔介质流动
- 弱可压缩流
- 需要长期时间展开的流动
