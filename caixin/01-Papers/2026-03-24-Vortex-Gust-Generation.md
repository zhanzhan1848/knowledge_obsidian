# 涡流扰动生成: 俯仰-升沉翼型方法

## 元信息

- **作者**: Bingfei Yan, et al.
- **提交日期**: 2026-03-22
- **领域**: Fluid Dynamics (physics.flu-dyn)
- **PDF**: [arxiv.org/pdf/2603.21336](https://arxiv.org/pdf/2603.21336)

## 研究目的

开发**孤立涡流扰动**生成方法，用于研究涡-翼相互作用。

## 核心方法

### 涡流生成
- 对称翼型
- 在**升沉运动**中进行**快速俯仰**
- 适用于**数值模拟和实验**

### 涡流特性
- 传播轨迹**近似平行于来流**
- 尾迹从涡核**斜向延伸**

## 主要发现

### 1. 运动参数影响
尽管数值和实验存在Re、俯仰持续时间、涡结构差异:
- **一致趋势**: 涡旋转方向、强度、位置与运动参数的关系

### 2. 下游翼型升力响应
- 与尾迹相关的气动影响**不持续**长时间尺度

### 3. 可控性
- 方法可实现**具有预设特性的涡流扰动**受控生成

## 应用场景

1. **涡-翼相互作用系统研究**
2. **阵风响应分析**
3. **气动弹性研究**
4. **流动控制验证**

## 技术细节

### 运动学
```
h(t) = h0 sin(ωt)        // 升沉
θ(t) = θ0 f(t/t_pitch)   // 快速俯仰
```

### 涡强度
与俯仰幅度和速率相关:
```
Γ ~ C θ0 / t_pitch
```
