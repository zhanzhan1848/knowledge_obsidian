# 层流预混氢气火焰拉伸因子的直接测量

**arXiv**: [2603.18852](https://arxiv.org/abs/2603.18852)  
**作者**: Max Schneider  
**提交日期**: 2026-03-19  
**分类**: physics.flu-dyn

---

## 📌 核心创新

首次使用 **OH-PLIF 成像**实验配置，直接测量层流氢气火焰从准稳态到热扩散不稳定 regime 转换时的拉伸因子 $I_0$。

---

## 🎯 实验配置

### 火焰设置
```
杆锚定 V 型火焰
    ↓
┌─────────────┬─────────────┐
│ 近锚定区    │ 远端区      │
│ 准稳态      │ 热扩散不稳定 │
│ 光滑表面    │ 胞状结构    │
│ θ_s 角度    │ θ_u 角度    │
│ S_s 燃速    │ S_u 燃速    │
└─────────────┴─────────────┘
```

### 当量比范围
$$\phi = 0.35 \sim 0.40$$

---

## 🔬 测量方法

### 1. 火焰速度比
$$\frac{S_u}{S_s} = \frac{S_c}{S_L}$$

通过测量稳定与不稳定分支的火焰角度比直接确定。

### 2. 表面积增加

**三种互补方法**:
1. 图像处理 - 火焰轮廓提取
2. 几何重建 - 3D 表面估计
3. 统计平均 - 多次测量

$$\frac{A}{A_0} = \text{不稳定表面积} / \text{平滑参考面积}$$

### 3. 拉伸因子

$$I_0 = \frac{S_c}{S_L} \times \frac{A_0}{A}$$

---

## 📊 实验结果

### 拉伸因子趋势

| 当量比 φ | I₀ 范围 | 趋势 |
|---------|---------|------|
| 0.35 | 1.1 – 1.3 | 高 |
| 0.40 | 0.8 – 0.9 | 低 |

**单调递减**: 与理论预测一致

### 转换距离

```
当量比 ↑
    ↓
转换距离 ↓
（不稳定更早出现）
```
