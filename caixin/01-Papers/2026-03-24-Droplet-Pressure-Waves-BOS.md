# 背景纹影技术测量液滴内行进压力波

## 元信息
- **arXiv**: [2603.21905](https://arxiv.org/abs/2603.21905)
- **作者**: Sayaka Ichihara, et al.
- **提交日期**: 2026-03-23
- **领域**: Fluid Dynamics (physics.flu-dyn)
- **PDF**: [arxiv.org/pdf/2603.21905](https://arxiv.org/pdf/2603.21905)

## 研究背景

冲击波-液滴相互作用在以下领域重要:
- 航空燃料燃烧
- 微创医学治疗

**挑战**: 实验量化困难

## 核心方法

### BOS技术增强
1. **新型光线追踪校正**
2. **同步系统**
3. **投影背景**

### 测量对象
- 毫米级全氟己烷液滴
- 浸没在水中
- 水下冲击波传播

## 主要发现

### 1. 成功量化
- 密度梯度场
- 压力场
- 液滴内外冲击波

### 2. 与CFD对比
| 参数 | BOS测量 | 数值模拟 |
|------|---------|----------|
| 声速 | ✅ 吻合 | - |
| 冲击聚焦位置 | ✅ 吻合 | - |
| 最大压力 | ✅ 吻合 | - |

### 3. 相移捕捉
**首次实验验证**冲击聚焦前后的相移
- 之前仅存在于假设

## 技术创新

### 光线追踪校正
```
真实密度梯度 ← [校正算法] ← 原始BOS数据
```

### 同步系统
- 冲击波发射
- 背景成像
- 精确时序控制

## 应用场景

1. **燃料喷射** - 液滴破碎研究
2. **医学超声** - 冲击波治疗
3. **CFD验证** - 提供基准数据
