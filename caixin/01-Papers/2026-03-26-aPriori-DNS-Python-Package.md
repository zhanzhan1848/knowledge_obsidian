**arXiv**: [2603.24363](https://arxiv.org/abs/2603.24363)
**日期**: 2026-03-26
**作者**: Lorenzo Piu et al.
**领域**: CFD 工具 / 湍流分析

## 核心问题

DNS 生成的高保真数据规模大、复杂、异构，系统化后处理和数据复用面临挑战。

## 解决方案

开发 `aPriori` 开源 Python 包：
- **内存高效**：指针式数据管理，标准工作站即可处理大型数据集
- **用户友好**：简洁可复现的脚本
- **功能丰富**：3D 场处理、过滤、尺度分离、梯度计算、热化学分析、可视化

## 主要功能

| 功能模块 | 描述 |
|---------|------|
| 过滤 | 尺度分离操作 |
| 梯度计算 | 数值微分 |
| 热化学分析 | 化学反应分析 |
| 可视化 | 数据可视化 |
| CSP | 计算奇异摄动分析 |

## 应用场景

1. **a priori 模型评估** - SGS 模型验证
2. **数据驱动闭包开发** - ML 模型训练
3. **详细化学分析** - 燃烧研究

## 技术亮点

```python
# 指针式数据管理示例
import aPriori as ap

# 加载大型 DNS 数据（内存高效）
data = ap.load_dns_field("simulation.h5")

# 过滤操作
filtered = ap.filter(data, cutoff=32)

# 梯度计算
gradients = ap.gradient(filtered)
```
