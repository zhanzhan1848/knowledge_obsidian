# Multiresolution Analysis on Tessellation Graphs for Inertial Particle Dynamics

## 论文信息
- **arXiv**: [2605.19244](https://arxiv.org/abs/2605.19244)
- **日期**: 2026-05-19
- **标签**: #颗粒动力学 #小波分析 #Voronoi #湍流 #多尺度分析

## 核心创新点
1. 提出基于**镶嵌图(tessellation graphs)的多分辨率技术**
2. 使用**Delaunay镶嵌**定义图结构，Voronoi胞元体积满足体积守恒
3. 利用**小波变换**分解拉格朗日点粒子数据为尺度相关贡献
4. 从DNS数据提取惯性粒子速度散度的尺度相关统计

## 技术框架
```
粒子位置数据
    ↓
Delaunay镶嵌 → Voronoi图
    ↓
小波变换 → 尺度分解
    ↓
统计 → 速度散度功率谱
```

## 验证
- 2D随机分布粒子的合成数据
- 同质各向同性湍流DNS数据
- 与傅里叶方法对比验证

## 应用
- 湍流中粒子聚集表征
- 多相流多尺度分析
- 颗粒 clustering 定量描述
