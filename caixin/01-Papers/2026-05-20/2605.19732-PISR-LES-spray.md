# Large-Eddy Simulation of Moderately Dense Evaporating Sprays with Particle-Informed Super-Resolution

## 论文信息
- **arXiv**: [2605.19732](https://arxiv.org/abs/2605.19732)
- **作者**: Ruyue Cheng
- **日期**: 2026-05-19
- **提交**: Proceedings of the Combustion Institute (volume 42)
- **标签**: #LES #蒸发 #喷雾 #深度学习 #超分辨率 #燃烧

## 核心创新点
1. 提出**Particle-Informed Super-Resolution (PISR)** 方法
2. 用深度学习超分辨率重建液滴间局部气体场，改善蒸发率计算
3. 在中等密度喷雾的先验和后验测试中验证
4. PISR-LES能准确复现载相直接数值模拟(CP-DNS)的蒸发率
5. 模型对未见案例具有良好的泛化能力（变化空气温度、液滴直径、湍流雷诺数）

## 问题背景
- 密集喷雾或显著聚集喷雾的LES中，网格太粗时蒸发率不准确
- 单液滴蒸发模型的边界条件不真实
- 深度学习SR方法适合重建液滴间气体场

## 技术路线
```
LES网格 → PISR超分辨率 → 高分辨率气体场 → 修正蒸发率
                                    ↓
                           与CP-DNS对比验证
```

## 数值方法
- **大涡模拟 (LES)**
- **载相直接数值模拟 (CP-DNS)** 用于验证
- **深度学习超分辨率**
