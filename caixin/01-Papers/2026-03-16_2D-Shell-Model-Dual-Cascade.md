# Multi-branch Shell Models of 2D Turbulence: Dual Energy-Enstrophy Cascades

**arXiv ID**: 2603.11892v1  
**发布日期**: 2026-03-12  
**链接**: https://arxiv.org/abs/2603.11892

## 分类
- physics.flu-dyn (流体动力学)

## 摘要
提出多分支壳模型，包含跨尺度层次组织的几何结构，以克服经典壳模型无法展示二维湍流双级联（能量逆级联+熵正级联）的限制。数值验证了热谱与二维流体方程一致，以及统计稳态双级联的出现。

## 核心创新点
1. **多分支结构**: 跨尺度层次几何组织
2. **双级联**: 能量逆级联 + 熵正级联
3. **热谱一致**: 与二维流体方程热谱匹配

## 理论背景
- **经典壳模型**: 无法产生双级联（热谱不正确）
- **二维湍流特征**: 
  - 能量逆级联（大尺度）
  - 熵正级联（小尺度）

## 数值方法
- **模型**: 多分支壳模型
- **验证**: 热谱、双级联、局部传递

## 关键结果
| 特性 | 经典壳模型 | 多分支壳模型 |
|------|------------|--------------|
| 热谱 | ❌ 不匹配 | ✅ 匹配 |
| 双级联 | ❌ 无 | ✅ 有 |
| 自相似 | - | ✅ 研究 |
| 非高斯性 | - | ✅ 研究 |

## 技术要点
- **几何层次**: 跨尺度组织
- **局部传递**: 可研究局部传递特性
- **统计稳态**: 双级联稳态出现
