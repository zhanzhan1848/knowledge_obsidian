## 核心贡献

> 通过几何解析 DNS 研究垂直轴风力发电机近尾流动力学

1. **几何解析 DNS**：完整解析动态失速过程，包括大尺度涡生成、叶片分离、与近尾流相互作用
2. **叶片数效应**：1/2/3 叶片配置对比，发现叶片-涡相互作用影响动态失速
3. **新型互补机制**：直接涡撞击导致动态失速涡提前破裂
4. **自相似分析**：量化近尾流向钝体动力学的转变速率

---

## 技术方案

### 核心思想

使用 spectral/hp 元素方法框架 Nektar++ 和移动参考系 formulation 进行几何解析 DNS。

### 关键技术

| 技术 | 说明 |
|------|------|
| Nektar++ | Spectral/hp 元素方法 |
| Moving Reference Frame | 移动参考系处理转子 |
| Dynamic Stall | 动态失速全分辨率捕捉 |
| Blade-Vortex Interaction | 叶片数增加引入新的相互作用 |
| Self-Similarity Analysis | 近尾流自相似性分析 |

---

## 主要发现

- **三叶片配置**：叶片-涡相互作用与动态失速早期阶段重合
- **层流分离泡**：动态失速涡尺寸减小
- **提前破裂**：直接涡撞击导致动态失速涡提前解体
- **尾流转变**：近尾流更快速地向钝体动力学转变
- **叶片数影响**：比尖速比更影响尾流转变速率
