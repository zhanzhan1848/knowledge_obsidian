# 2607.15078 - Thermodynamically Consistent Manifold Model for Flames

## Metadata
- **arXiv**: [2607.15078](https://arxiv.org/abs/2607.15078)
- **Subjects**: Fluid Dynamics (physics.flu-dyn)
- **Date**: Thu, 16 Jul 2026
- **Authors**: John Boerchers

## Core Innovation
Manifold-based turbulent combustion model for LES with **thermodynamic consistency**:
- Iterative procedure to ensure consistency between model and flow solver
- Unified framework for deflagrations AND detonations
- Reproduces temperature, radical species, source term profiles

## Key Methods
- **Large Eddy Simulation (LES)** framework
- **Manifold-based combustion model**
- **Thermodynamic consistency** procedure
- Validated against 1D and RDE-like data

## Applications
- Rotating detonation engines
- Supersonic combustion systems
- High-speed reacting flows

## Key Findings
- Outperforms existing approaches across deflagration and detonation regimes
- Captures both deflagrative and detonative behavior in single framework
- Foundation for future compressible reacting flow studies

## Significance for CFD
✅ Major advance for LES of high-speed combustion, scramjet applications
