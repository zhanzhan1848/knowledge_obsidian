## 核心贡献

> 将 POD-NN 代理模型从个案研究扩展为可复用的研究软件平台

1. **POD-NN 工作流封装**：参数采样、Fluent 自动化、数据提取、降阶模型构建、NN 训练、验证、预测
2. **Python API + GUI**：脚本化 API 和可选桌面 GUI，支持可重复研究
3. **多输出支持**：标量、表面场、体积区输出
4. **数字孪生开发**：为 CFD 代理建模和数字孪生开发提供可重用平台

---

## 技术方案

### 核心思想

封装 CFD 仿真全流程，通过 POD (Proper Orthogonal Decomposition) 和 Neural Network 构建快速代理模型。

### 工作流

```
Simulation Inputs → DOE Sampling → Fluent Batch → Data Extraction → POD-ROM → NN Training → Validation → Prediction
```

### 关键技术

| 技术 | 说明 |
|------|------|
| POD (Proper Orthogonal Decomposition) | 模态分解，能量最优基 |
| Neural Network | 非线性解码器学习映射 |
| ANSYS Fluent API | 批量仿真自动化 |
| Design of Experiments | 参数采样策略 |
| Python API + Desktop GUI | 双重接口 |
