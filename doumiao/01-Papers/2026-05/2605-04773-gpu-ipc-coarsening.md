# Adaptive In-Solve Algebraic Coarsening for GPU IPC

## 核心创新点
- **方法**: 代数自适应 in-solve 粗化
- **突破**: 在隐式时间积分的牛顿求解内动态减少自由度，无需显式拓扑修改
- **效率**: GPU 导向，IPC 屏障能量的无缝集成

## 技术细节

### 方法流程
1. 从细网格开始
2. 将适应性表达为选择性边缘折叠过程
3. 使用 warp 级哈希映射方案并行聚合
4. 受保护边缘保留局部细节
5. PCG 求解粗系统后延长回细网格

### 性能提升
- 高达 **3x 加速** (相比 SOTA GPU IPC 求解器)
- 视觉无差异结果

### 适用场景
- 刚性材料模拟
- 大变形
- 隐式时间积分
