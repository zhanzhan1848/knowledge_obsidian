# Mixwell - 2D 流体画笔
**日期**: 2026-07 (SIGGRAPH 2026)
**类型**: 交互式流体渲染
**会议**: SIGGRAPH 2026 **Best Paper**
**作者**: Doug James, Ethan James

---

## 核心贡献

Mixwell 是实时、分辨率无关的**物理基础流体混合工具**，通过 GPU 加速的解析方法实现。

### 关键创新

1. **势流建模**: 源自圆柱齿周围的势流推导
2. **无网格方法**: 每个采样点无网格或中间重采样计算漂移
3. **渐进式混合**: 支持任意分辨率的实时混合和渲染
4. **极低数值耗散**: 解析方法避免传统数值方法的耗散问题

---

## 技术细节

### 物理模型
```
漂移评估: 基于势流理论的解析解
- 无需网格离散化
- 无中间重采样
- 数值耗散可忽略
```

### GPU 加速
- 利用 GPU 并行计算能力
- 每个样本独立评估 drift
- 支持任意分辨率输出

### 应用场景
- 创意流体工具
- 实时交互模拟
- 数字艺术创作
