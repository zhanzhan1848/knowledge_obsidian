## 🎯 核心问题

**流体-颗粒流动模拟**的挑战：
1. 连续流体 + 离散颗粒介质的**强非线性耦合**
2. 大规模场景难以同时实现**物理保真度**和**计算效率**

---

## 💡 创新方法

### 统一框架架构
```
┌─────────────────────────────────────┐
│         自适应多级 HOME-LBM          │
│    (流体: Lattice Boltzmann)        │
├─────────────────────────────────────┤
│              双向耦合层              │
├─────────────────────────────────────┤
│         MPM 颗粒求解器               │
│    (颗粒: Material Point Method)    │
└─────────────────────────────────────┘
```

### 关键技术

#### 1. 自适应块式多级 HOME-LBM
- 基于**固体几何结构**的块划分
- 多级格子分辨率
- 高效内存使用

#### 2. 一致的矩重标度律
- 宏观量在**细化界面**间准确传递
- 保证物理一致性

#### 3. GPU 动态维护
- 多级块随**粒子运动**动态调整
- 全 GPU 加速算法

#### 4. 双向耦合
- 所有 MPM 粒子位于**最细流体节点**内
- 精确的流体-颗粒相互作用

## 🔬 技术细节

### Lattice Boltzmann Method (LBM)
- 基于微观动力学的流体模拟
- 天然适合并行计算
- HOME-LBM 变体提供自适应能力

### Material Point Method (MPM)
- 粒子法表示颗粒介质
- 适合大变形问题
- 支持：沙、雪等材料

### 耦合策略
```
流体节点 ←→ MPM 粒子
    ↓           ↓
速度场传递   力反馈
```

## 📝 实现要点

### 渲染管线建议
```
LBM 网格数据 → 体积渲染 (烟雾/水雾)
     ↓
MPM 粒子数据 → 粒子渲染 (沙/雪)
     ↓
表面提取 → 网格渲染 (水体表面)
```

### 优化策略
1. 使用 GPU 计算 LBM 碰撞/流动
2. 粒子排序优化 MPM 更新
3. LOD 策略处理远处区域
