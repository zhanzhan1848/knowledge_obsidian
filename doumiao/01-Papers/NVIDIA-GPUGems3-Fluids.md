## 技术背景

### 流体模拟挑战
- **计算成本高**: 传统上属于高质量离线渲染领域
- **粒子系统局限**: 虽然能产生吸引人的结果，但无法匹敌流体模拟的真实外观和行为
- **数据格式问题**: 模拟产生的体积数据不容易适配到标准的光栅化渲染范式

### GPGPU方法基础
- **通用GPU计算**: 将GPU应用于传统光栅图形以外的问题
- **数学物理基础**: 需要理解底层数学和物理
- **数值计算**: 偏微分方程的离散化处理

## 流体运动方程

### 基本假设
- **无粘性**: 更像水而非油的流体
- **不可压缩**: 体积随时间不变化

### 动量方程
```
∂u/∂t + (u·∇)u = -∇p/ρ + f + ν∇²u
```

### 不可压缩性约束
```
∇·u = 0
```

其中:
- u: 速度场
- p: 压力
- ρ: 质量密度
- f: 外力（如重力）
- ν: 粘性系数
