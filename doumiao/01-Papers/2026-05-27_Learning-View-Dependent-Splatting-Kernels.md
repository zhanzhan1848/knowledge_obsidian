# Learning View-Dependent Splatting Kernels
**日期**: 2026-05-25
**类型**: 体积渲染 / 神经渲染
**会议**: SIGGRAPH 2026
**链接**: https://arxiv.org/abs/2605.25426

---

## 核心贡献

可微分框架自动学习**视图相关的 2D 内核**，提升新型视图合成的重建质量和表示效率。

### 关键创新

1. **体积图元定义**: 包围椭球 + 3D 内核潜在向量
2. **投影网络**: 输入椭球属性和 3D 内核潜在向量，输出 2D 内核潜在向量
3. **Mahalanobis 距离**: 使用 Mahalanobis 距离定义径向对称 2D 内核
4. **联合优化**: 神经网络与每图元属性联合优化

---

## 技术实现

### 管道
```
输入: 椭球属性 + 3D 内核潜在向量
    ↓
投影网络 → 2D 内核潜在向量
    ↓
解码器 → 径向对称 2D 内核 (bounded by projected ellipsoid)
```

### 方法优势
- 与状态-of-the-art 相比表现良好
- 可扩展到一般 2D splatting 和图像表示
- 支持分析和学习内核
