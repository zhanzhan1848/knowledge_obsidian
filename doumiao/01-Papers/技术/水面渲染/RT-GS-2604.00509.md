## 核心创新

### 问题
- 标准 3D Gaussian Splatting 难以同时建模：
  1. **镜面反射** (Specular Reflections)
  2. **半透明表面后的物体** (Transmittance)

### 解决方案
提出 **RT-GS**，统一框架：
- **微面元材质模型** (Microfacet Material Model)
- **光线追踪** (Ray Tracing)
- 联合建模镜面反射和透射

### 关键技术

#### 独立 Gaussian 基元
- 使用**独立的 Gaussian 基元**分别处理反射和透射
- 反射基元 → 建模远处反射
- 透射基元 → 重构半透明表面后的物体

#### 可微分光线追踪
- 可微分路径追踪框架
- 优化 Gaussian 参数以匹配物理过程

## 应用场景

| 场景 | 说明 |
|------|------|
| 水面反射 | 镜面反射 Gaussian |
| 玻璃/水折射 | 透射 Gaussian |
| 汽车漆面 | 多层材质 |
| 透明材质合成 | 反射+透射组合 |

## 与水面渲染的关系

### 水面渲染技术链
```
水面几何 + 法线
    ↓
RT-GS 反射基元 ← 镜面反射
    ↓
RT-GS 透射基元 ← 水下透射/次表面
    ↓
3DGS 渲染管线
    ↓
最终水面图像
```

### 优势
- 物理基础：基于微面元 BRDF
- 端到端可微：支持优化
- 统一表示：与现有 3DGS 兼容

## 局限性
- 需要光线追踪，计算量较大
- 不适合极端实时应用
- 依赖场景几何先验
