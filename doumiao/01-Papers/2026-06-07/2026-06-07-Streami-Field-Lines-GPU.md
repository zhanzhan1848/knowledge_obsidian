# Streami: GPU-Accelerated Field Lines Computation for Fluid Flows

## 核心贡献

### 概述
Streami是一个可扩展的GPU加速库，用于在高性能计算机上计算流体流动中的**场线 (Field Lines)**。

###关键特性
1. **高性能计算**
   - GPU加速
   - MPI并行
   - 适合高性能计算环境

2. **接口设计**
   - 薄层接口 (Thin Layer)
   - 支持事后分析和原位分析 (In-Situ)
   - 可与现有MPI应用接口

3. **应用场景**
   - 快速原型设计
   - 交互式种子点放置
   - 流体场可视化

## 渲染方法
- **类型**: 体积渲染 / 场线可视化
- **技术**: GPU并行计算 + MPI并行

## 性能预期
- **帧率**: 高吞吐量
- **GPU需求**: 多GPU
- **内存占用**: 取决于流体场规模

## 实现建议
- **着色器复杂度**: 低
- **管线要求**: MPI + GPU混合并行
- **推荐度**: ✅✅

## 技术架构
```
应用层 → Streami API → MPI通信 → GPU计算
```

## 创新点
1. 首个针对流体场线的GPU加速MPI库
2. 支持事后和原位分析
3. 可扩展性强

## 应用场景
- 科学可视化
- 流体动力学分析
- CFD后处理
- 场线渲染
