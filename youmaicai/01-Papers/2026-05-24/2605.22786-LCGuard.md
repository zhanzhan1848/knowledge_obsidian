## 核心贡献
1. 提出 **LCGuard** 框架：在多 Agent LLM 系统中实现安全的 KV-based 潜在通信
2. 将共享 KV caches 视为潜在工作内存，在跨 Agent 传输前学习表示级转换
3. 形式化表示级敏感信息泄漏：通过对抗解码器从共享 cache artifact 重构敏感输入

## 问题背景
- 多 Agent 系统通过潜在通信（特别是 transformer KV caches）提高效率
- KV caches 编码了上下文输入、中间推理状态、Agent 特定信息
- 敏感内容可能在没有显式文本披露的情况下传播

## 方法
- **对抗训练**：对手学习重构敏感输入，LCGuard 学习保留任务相关语义的转换
- 表示级转换在 cache artifacts 跨 Agent 传输前执行
- 验证：对抗解码器能否从共享 cache 中恢复敏感输入

## 实验结果
- 多模型家族和 multi-agent 基准测试
- 一致降低基于重构的泄漏和攻击成功率
- 保持与标准 KV 共享基线相当的任务性能

## 核心创新点
- 首次系统性处理 KV cache 潜在通信的安全问题
- 表示级安全 vs 文本级安全
