## 核心贡献
1. 发现**累积消息效应**（AMEL）：先前对话历史极性偏置后续判断
2. 75,898 次 API 调用跨 11 模型、4 提供商
3. 负向不对称性：negative histories 产生 1.62x 更强偏置

## 主要发现
- 模型向对话 prevailing polarity 偏移（d = -0.17, p < 10^-46）
- 高熵项目更受影响（d = -0.34）
- 5 和 50 个历史 turn 产生相同偏移（不随上下文增长）
- **Negativity asymmetry**：negative histories 产生 1.62x 更强偏置

## 机制分析
- Token 概率分布连续转移，非阈值式
- 负向不对称有 token 级和语义成分
- 位置无关：5 个偏置 turn 放在 50-turn 历史任意位置效果相同

## 缓解建议
- 每个评估项使用新鲜 context
- 必须 batching 时平衡历史

## 核心创新点
- 系统性揭示 LLM 作为自动化评估器时的历史偏置问题
- 提供实用缓解策略
