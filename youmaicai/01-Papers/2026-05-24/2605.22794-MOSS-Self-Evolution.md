## 核心贡献
1. 提出**源代码级自我进化**作为 Agent 适应性进化的更通用介质（图灵完备，是文本可 mutation 范围的超集）
2. MOSS 系统：通过多阶段 pipeline 实现基于自动 curation 的生产故障证据进行自重写
3. 在 OpenClaw 上，单次 cycle 将四任务平均 grader 分数从 0.25 提升至 0.61

## 模型架构
- 使用外置 coding-agent CLI 处理代码修改
- MOSS 保留阶段排序和 verdicts
- 通过 replay 验证候选版本（ephemeral trial workers）
- 用户同意后 in-place container swap，health-probe rollback

## 核心创新点
- 文本可 mutation 的内容（skill files, prompts, memory schemas, workflow graphs）vs 源代码（routing, hook ordering, state invariants, dispatch）
- 源代码级适应：更通用、可确定性生效、不受 long-context drift 影响

## 局限性
- 依赖外部 coding-agent CLI 的能力
- 需要足够的安全防护防止有害进化
