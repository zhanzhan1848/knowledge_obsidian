## 核心贡献
1. 提出**通道级 erase gate (b_t)** 和 **通道级 write gate (w_t)**，解耦线性注意力中的擦除和写入操作
2. 统一了 Gated DeltaNet 和 Kimi Delta Attention (KDA)
3. 1.3B 参数模型在 100B FineWeb-Edu tokens 上训练，长上下文 RULER needle-in-haystack 基准表现最优

## 模型架构
- 线性注意力替代 softmax 注意力，使用固定大小 recurrent state
- 将序列混合降至线性时间，解码为常数内存
- **Gated Delta Rule-2**：分离 erase 和 write 角色

```math
\text{新状态} = f(\text{旧状态}, \text{输入})
```

### 与之前工作的关系
- 当 erase gate = write gate 时，退化为 KDA
- 当 channel-wise decay 也 collapse 时，退化为 Gated DeltaNet

## 实验结果
- 1.3B 参数，100B tokens 训练
- 长上下文 RULER needle-in-a-haystack 基准：多键检索设置最优
- recurrent 和 hybrid 设置均保持强劲

## 核心创新点
- 解决标量绑定问题：之前工作用单一标量 gate 同时控制 erase 和 write
- 通道级衰减 + 通道级门控
