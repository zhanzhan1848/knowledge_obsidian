# RAGP: Prompt Compression via Lévy Walk-Guided Graph Pruning

## 核心贡献

1. **RAGP (Redundancy-Aware Graph Pruning)**：将 prompt 压缩建模为多路图上的冗余感知图剪枝
2. **Lévy Walk**：利用重尾分布平衡局部利用和全局探索
3. **LongBench 4x 压缩率下达到 49.3 分**，超越 LongLLMLingua 的 3x 压缩率（48.8）

## 方法

### 多路图表示
- **细粒度边**：基于注意力的依赖关系
- **粗粒度边**：语义关系
- 联合建模文本中的局部句法依赖和全局语义关系

### Lévy Walk 图剪枝
- 高效识别密集局部子图和稀疏全局连接中的非冗余节点
- 重尾步分布自然平衡局部探索和全局探索

## 实验结果

| 方法 | 压缩比 | LongBench 得分 |
|------|--------|--------------|
| LongLLMLingua | 3x | 48.8 |
| **RAGP** | **4x** | **49.3** |

- 在多个任务上超越基于视觉的文本压缩范式
