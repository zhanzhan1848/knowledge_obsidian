## 核心贡献

1. **RuleChef**：使用 LLM 生成可执行规则的框架，适用于文本分类、NER、关系抽取等 NLP 任务
2. **人机协同**：结合人类反馈和 LLM 自动迭代改进
3. **快速、可确定、可检查的规则系统**：推理时无需 LLM

## 方法

### 规则生成流程
1. 基于任务描述和标注样本生成初始规则
2. 基于额外样本和人类反馈迭代改进
3. 基于 held-out split 测量失败并补丁

### Bootstrap
- 可利用任何现有模型的输入-输出对来引导规则生成

### 特点
- LLM 仅在训练时使用
- 推理时：快速、确定性、可解释

## 局限性

- 初步评估，完整benchmark结果未展示
- 复杂任务上规则表达能力可能受限

## 建议

- **是否推荐**：是（代码已开源）
- **适用场景**：需要可解释性的 NLP 任务、规则化 domain knowledge 注入
