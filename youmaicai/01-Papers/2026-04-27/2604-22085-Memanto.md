# Memanto: Typed Semantic Memory with Information-Theoretic Retrieval for Long-Horizon Agents

|------|-------|

| arXiv | arXiv:2604.22085 |
| 分类 | cs.AI |

## 核心贡献

1. **Memanto**: 通用 Agent 记忆层，挑战"知识图谱复杂性是实现高保真 Agent 记忆必要条件"的假设
2. **13 种预定义记忆类别**: 类型化语义记忆模式
3. **自动化冲突解决**: 自动冲突解决机制
4. **时间版本控制**: Temporal versioning
5. **无索引语义数据库**: Moorcheh IT Search engine，<90ms 确定性检索，无摄取延迟

## 评估结果

| 基准 | Memanto | 之前最佳 |
|------|---------|---------|
| LongMemEval | 89.8% | 超越所有混合图和向量系统 |
| LoCoMo | 87.1% | 超越所有评估系统 |

## 关键优势

| 特性 | Memanto | 知识图谱方法 |
|------|---------|------------|
| 检索查询数 | 1 | 多查询管道 |
| 摄取成本 | 0 | 高（LLM 实体提取） |
| 架构复杂性 | 低 | 高 |
| 检索延迟 | <90ms | 更高 |
