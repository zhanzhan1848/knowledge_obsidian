## 核心贡献

1. **Multi-Agent Deliberation (MAD)**：研究 LLM 在法律推理任务中的多 Agent  deliberation 方法
2. **两种新框架**：受法庭程序和法律论证启发的多 Agent 框架
3. **互补性发现**：多 Agent 框架与基线 LLM 整体性能相当，但产生显著不同的答案；能成功解决基线失败案例，反之亦然

## 关键发现

- 多 Agent 框架更适合需要多角度批判性思考的问题
- 在需要从多个角度进行批判性思考的问题上表现更好
- 与 monolithic 方法形成互补
- 法律启发的多 Agent 方法是 AI 法律应用的 promising 方向
