## 核心贡献
1. **SPEAR (Sandboxed Prompt Engineer with Active Roll-back)**: 自由形式 agentic prompt 优化器
2. **四个工具**: evaluate, python, set_prompt, finish — 自主决定使用方式和时机
3. **Python sandbox**: 优化器可编写执行任意 Python 代码进行结构化误差分析
4. **两个 guardrails**: auto-rollback on metric regression + optional guard metric floor
5. 在 13 个工业 judge 任务和 BBH/GSM8K 上全面超越现有方法

## 方法
- 将 CodeAct 的 code-as-action 范式引入 APE
- Python tool 执行结构化误差分析（confusion matrices, error clustering, per group metrics）
- auto-rollback 防止指标退化
- guard metric floor 保证最低性能

## 实验结果
| 任务 | SPEAR | GEPA | TextGrad |
|------|-------|------|----------|
| tool-selection (κ) | 0.857 | 0.359 | - |
| filter-relevance (F1-macro) | 0.815 | 0.763 | - |
| extraction (κ) | 0.254 | 0.218 | - |
| BBH-7 avg | 0.938 | 0.628 | 0.484 |

## Ablations
- Python tool 是最大单一杠杆: +0.79 κ (5-class tool-selection), +0.35 κ (hardest extraction dimension)
- 核心贡献: class-pair confusion aggregation
