# GraphDx: Cost-Aware Knowledge-Enhanced Multi-Agent Framework for Sequential Diagnosis

## 核心贡献
1. **问题**: 现有LLM方法存在knowledge-reasoning gap：编码丰富医学知识但在成本约束下难以系统推理，常过度测试
2. **GraphDx框架**:
   - **创新1**: 自动pipeline利用LLM构建医学诊断知识图谱(MDKGs)
     - 量化的典型性
     - 以动作为中心的拓扑
     - 双重目标属性（诊断相关性+成本敏感性）
   - **创新2**: 三个协作Agent
     - Perception Agent: 语言理解和生成
     - Reasoning Agent: 在MDKG上进行确定性证据评分和成本感知规划
     - Decision Agent: 语言理解和生成

## 实验结果
| 指标 | 原始 | GraphDx |
|------|------|---------|
| 诊断成功率 | 50-68% | 79-93% |
| 测试成本降低 | - | 20-54% |
| 测试模型 | DeepSeek-V3, Kimi-k2, Llama-3.3 | |

## 关键洞察
- 医学诊断知识图谱+多Agent协作有效弥合knowledge-reasoning gap
- MDKG提供可解释的诊断推理轨迹

- **是否推荐使用**: 是
