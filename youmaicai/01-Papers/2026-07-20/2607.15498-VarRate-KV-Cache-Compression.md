# VarRate: Training-Free Variable-Rate KV Cache Compression for Long-Context LLMs

## 核心贡献
1. **问题诊断**: 指出现有两种训练-free KV压缩方法的局限：
   - Token-selection方法(SnapKV, Ada-KV): 驱逐低分token不可逆，查询无关重用时准确率崩溃11-15点
   - 均匀低秩编码: 每个token保留但预算均分，浪费资源
2. **核心洞察**: rank应该被分配，而不是被驱逐
3. **VarRate**: 训练-free KV codec，根据query salience为每个token分配可变低秩预算，保持每个token非零rank

## 方法
- **核心思想**: 秩分配而非token驱逐
- **关键创新**: 无需训练即可实现自适应秩分配
- **优势**: 不丢弃任何token，避免不可逆信息损失

## 实验结果
| 模型 | 性能 |
|------|------|
| 20% budget on LongBench (16 tasks) | VarRate保持在uncompressed model的0.8分内 |
| Llama-3.1-8B | 平均最强matched-memory compressor |
| Qwen2.5-7B | 平均最强matched-memory compressor |
| vs KVzip | 精度相当，prefill开销仅为1/8 |
| 精度损失 | 仅3.5-5.5点（query-aware selection崩溃11-15点） |

## 关键公式/概念
- Query salience-based rank allocation
- Variable low-rank budget per token
- Non-zero rank preservation for all tokens

## 建议
- **是否推荐使用**: 是
- **适用场景**: 长上下文LLM推理优化、内存受限部署
