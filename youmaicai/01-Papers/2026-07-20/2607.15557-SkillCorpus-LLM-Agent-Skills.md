## 核心贡献
1. **问题**: Agent skills (SKILL.md文件) 分散、冗余、质量不均
2. **SkillCorpus框架**:
   - 从~821,000个爬取的skills中筛选出96,401个
   - 组织为16类分类法 + 3个质量维度（utility, robustness, safety）
   - 配合微调的检索-选择stack匹配任务相关skills

## 评估结果
| 基准 | 提升 |
|------|------|
| SkillsBench | +7.5 pp |
| GDPVal | consistent gains |
| QwenClawBench | consistent gains |

## 关键发现
- 收益来自coverage boundary和harness boundary
- 首次端到端说明：何时社区语料库改善真实agent任务，何时不改善
- 数据集、模型和代码将在接受后发布

## 建议
- **是否推荐使用**: 是
- **适用场景**: LLM Agent技能管理、Skill检索、RAG系统
