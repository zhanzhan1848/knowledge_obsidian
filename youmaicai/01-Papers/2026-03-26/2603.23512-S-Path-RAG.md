## 核心贡献
1. **Semantic-Aware Path Retrieval**: 语义感知的最短路径检索框架，改进知识图谱上的多跳问答
2. **Hybrid Weighted Strategy**: 混合加权 k-shortest、beam 和约束随机游走策略
3. **Differentiable Path Scorer**: 可微分路径评分器 + 对比路径编码器 + 轻量验证器
4. **Neural-Socratic Graph Dialogue**: 迭代式神经苏格拉底图对话循环，支持自适应检索

## 模型架构

### 核心组件
```
1. Path Enumeration (路径枚举)
   - Hybrid weighted k-shortest
   - Beam search
   - Constrained random walk
   
2. Path Learning (路径学习)
   - Differentiable path scorer
   - Contrastive path encoder
   - Lightweight verifier
   
3. LLM Integration (LLM 集成)
   - Cross-attention 注入
   - Soft mixture of path latents
```

### Neural-Socratic Graph Dialogue Loop
```
LLM generates diagnostic message
    ↓
Map to graph edits or seed expansions
    ↓
Adaptive retrieval when model uncertain
    ↓
Updated graph state
    ↓
Repeat
```

## 关键创新

### 1. 语义加权路径
- 不同于一次性、文本密集的检索
- 枚举有界长度、语义加权的候选路径

### 2. Token 高效 + 拓扑感知
- 检索机制既节省 token 又保持知识图谱拓扑
- 保留可解释的路径级追踪

### 3. 自适应检索
- 当模型表达不确定性时，通过诊断消息触发针对性检索

## 实验结果
- 在标准多跳 KGQA 基准测试上验证
- 在答案准确性、证据覆盖率和端到端效率方面一致改进
- 超越强大的图基线和 LLM 基线

### 消融分析
- 语义加权 vs 验证器过滤 vs 迭代更新的权衡
- 提供受约束计算和 token 预算下的部署建议

## 适用场景
- 多跳知识图谱问答
- 复杂推理任务
- 需要可解释路径的系统
