## 核心贡献
1. **Privacy-Preserving Training**: 使用合成临床数据训练医学编码 LLM，无需暴露受保护的健康信息 (PHI)
2. **Llama 3-70B Fine-tuning**: 在 Llama 3-70B 上微调，实现专家级医学编码
3. **F1 大幅提升**: 零样本基线 F1=0.18 → 微调后 F1>0.70（绝对增益 52%+）
4. **复杂类别保持高性能**: 在需要多步临床推理和代码组合的复杂类别上保持高性能

## 背景

### 医学编码挑战
- 异构记录
- 细致的编码指南
- 长尾分布
- 基础模型未经医学编码训练，零样本效果差

### 解决方案
- 使用 EHR 接地模板和编码策略生成合成数据
- 临床笔记 + 金标准代码配对
- 隐私保护 + 任务特定优化

## 方法论

### 数据生成
```
EHR-grounded templates (EHR 接地模板)
    ↓
Coding policies (编码策略)
    ↓
Synthetic corpus (合成语料库)
    ↓
Clinical note + gold code pairs (临床笔记+金标准代码对)
```

### 模型训练
- 基础模型: Llama 3-70B
- 训练数据: 合成语料库
- 任务: ICD-10-CM 和 CPT 精确代码预测

## 实验结果

### 性能对比
| 设置 | Exact-Match F1 |
|------|----------------|
| 零样本基线 | 0.18 |
| 合成数据微调 | >0.70 |
| 绝对增益 | 52%+ |

### 复杂类别表现
- Advanced Illness (晚期疾病): 保持高性能
- Frailty classes (衰弱类别): 保持高性能
- 多步临床推理: 有效处理
- 代码组合: 有效处理

### 额外发现
- 模型在医学理解任务上保持性能
- 无灾难性遗忘

## 核心洞察
1. **合成数据有效**: 合成、策略感知数据可以高效教授通用 LLM 精确医学编码
2. **隐私保护可行**: 无需暴露 PHI 即可训练任务特定模型
3. **实际应用路径**: 为安全、迭代地训练特定任务的编码 Agent 提供实用路径
