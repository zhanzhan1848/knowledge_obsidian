## 核心贡献
1. **Domain-Agnostic Framework**: 领域无关的框架，通过自适应追问测量 LLM 的知识深度
2. **Adaptive Probing**: 基于模型实际提到的概念生成后续问题
3. **On-Demand Fact Verification**: 从权威来源按需进行事实验证
4. **Survival Statistics**: 每个深度级别保持恒定样本量的生存统计
5. **Expected Valid Depth (EVD)**: 引入 EVD 指标量化知识深度

## 方法论

### 三大创新
```
1. Adaptive Probing (自适应探测)
   - 生成基于模型实际提及概念的后续问题
   - 不是预定义的固定问题序列
   
2. On-Demand Verification (按需验证)
   - 从权威来源实时验证事实
   - 无需预构建测试集
   
3. Survival Statistics (生存统计)
   - 每个深度级别保持恒定样本量
   - 确保统计有效性
```

### 评估特性
- 可部署于任何具有公开可验证事实的知识领域
- 不需要预构建测试集或领域特定专业知识
- 结果相对于用于答案检查的评估器模型

## 实验结果

### 四个领域验证
- Medicine (医学)
- Constitutional Law (宪法)
- Ancient Rome (古罗马)
- Quantum Computing (量子计算)

### 五个前沿模型测试
- EVD 范围: 3.45 - 7.55（跨模型-领域组合）
- 模型排名因领域而异，无单一模型在所有领域占优

### 关键发现
| 发现 | 说明 |
|------|------|
| 领域差异 | 模型排名因领域大幅变化 |
| 成本≠深度 | 昂贵模型不一定达到更深知识 |
| 隐藏性能 | 标准基准无法揭示的深度依赖性能变化 |

## 核心洞察
1. **领域特定评估更重要**: 对于专业应用中的模型选择，领域特定评估比聚合基准更具信息量
2. **比较评估工具**: DepthCharge 是比较评估工具，而非绝对准确性认证
3. **揭示隐藏性能**: 揭示标准基准隐藏的深度依赖性能变化

## 适用场景
- 专业领域模型选型
- LLM 知识深度评估
- 模型能力分析

## 局限性
- 结果相对于评估器模型
- 需要公开可验证的事实来源
