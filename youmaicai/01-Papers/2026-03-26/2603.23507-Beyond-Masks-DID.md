## 核心贡献
1. **Deletion-Insertion Diffusion (DID)**: 首次将 token 删除和插入严格形式化为离散扩散过程，替代传统 Masked Diffusion Language Models (MDLMs) 中的 mask/unmask 范式
2. **计算效率提升**: 消除 MDLMs 中两个主要计算开销源：(1) 非信息性的 `<MASK>` token 计算 (2) 变长序列中的 `<PAD>` token 填充
3. **灵活性增强**: 原生支持变长序列（无需固定长度填充），且通过插入操作实现内在的自校正机制

## 模型架构

### 核心思想
传统 MDLMs 使用 mask/unmask：
- Mask: 将 token 替换为 `<MASK>`
- Unmask: 从 `<MASK>` 预测原始 token

DID 使用 deletion/insertion：
- Deletion: 直接删除 token，序列变短
- Insertion: 在位置插入新 token，序列变长

### 关键创新
```math
\text{MDLM}: \quad x_0 \xrightarrow{\text{mask}} x_t \xrightarrow{\text{unmask}} \hat{x}_0
```
```math
\text{DID}: \quad x_0 \xrightarrow{\text{delete}} x_t \xrightarrow{\text{insert}} \hat{x}_0
```

### 训练目标
- 设计基于 score 的方法，为 token 插入操作分配分数
- 训练目标涉及子序列计数问题
- 使用并行化动态规划算法高效求解

## 训练方法
- 基于 score 的扩散模型训练
- 并行化动态规划求解子序列计数
- 无需超参数调优即可超越基线

## 实验结果
- 在固定长度和变长设置下均优于 MDLM 基线
- 在建模性能、采样质量和训练/推理速度方面均有提升
- 与现有基于插入的 LM 相比也有优势
