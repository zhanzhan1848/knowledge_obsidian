## 核心贡献

1. **问题识别**：现有 Agent 假设用户是专家（偏好已明确），当任务未充分指定时默认使用澄清问题，但用户往往缺乏领域知识来形成完全明确的偏好
2. **CoPref 模型**：基于 Search-Experience-Credence 框架，形式化用户如何根据 Agent 对话动作构建偏好
3. **CoShop benchmark**：交互式评估 benchmark，Agent 与 CoPref 用户对话并推荐，用户表现取决于 Agent 能否帮助用户获取指定任务所需知识

## 关键洞察

- 当被问及某个特征的偏好时，用户可能无法回答，除非 Agent 帮助用户学习形成该偏好所需的领域知识（例如通过示例或解释）
- **核心观点**：Agent 应该帮助用户构建偏好，而不仅仅是 eliciting them

## 实验发现

- 评估五个前沿模型，**无一超过 56% 准确率**（5轮交互）
- 失败原因不在于 Agent 找到物品的能力，而在于交互对用户所知所想扩展很少

## 建议
- 是否推荐使用：**是**，对 Agent 设计有重要指导意义
- 适用场景：对话式推荐系统、用户偏好建模、人机交互
