## 3. [Single-Agent vs Multi-Agent: Multi-Hop Reasoning](https://arxiv.org/abs/2604.02460)

### 元信息
| 标题 | Single-Agent LLMs Outperform Multi-Agent Systems on Multi-Hop Reasoning Under Equal Thinking Token Budgets |
|------|------|
| 作者 | Dat Tran, Douwe Kiela |
| 链接 | [原文](https://arxiv.org/abs/2604.02460) |
| arXiv | arXiv:2604.02460v1 |
| 领域 | cs.CL, cs.MA |

### 核心贡献
1. **信息论分析**: 基于 Data Processing Inequality，在固定推理 token 预算下，单智能体系统信息效率更高
2. **受控实验**: Qwen3, DeepSeek-R1-Distill-Llama, Gemini 2.5 三大家族
3. **方法论贡献**: 识别 API 预算控制的 artifacts 和标准基准问题

### 关键发现
- 当推理 token 固定时，单智能体持续匹配或超越多智能体系统
- 很多多智能体优势源于未计量的计算和上下文效应
- 多智能体在上下文利用降级或增加计算时才有竞争力

### 评估
- 适用场景: 多智能体系统设计、推理效率优化
- 推荐指数: ⭐⭐⭐⭐⭐

| arXiv | arXiv:2604.02605v1 |
| 领域 | cs.AI, cs.SD |
| 备注 | CVPR Findings |

### 核心贡献
1. **首个 AVLLM 机械可解释性研究**
2. **跨层音频-视觉特征演化分析**
3. **模态偏倚根源追踪**: 训练数据中音频监督对齐不足

- AVLLM 音频行为与视觉-语言基模型强匹配，表明额外音频对齐有限

### 评估
- 适用场景: 多模态 LLM 训练、音频-视觉融合研究
