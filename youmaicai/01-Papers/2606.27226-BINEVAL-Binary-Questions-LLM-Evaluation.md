## 核心贡献
1. 提出 **BINEVAL** 框架：将评估标准分解为原子二值问题，聚合为可解释的多维度分数
2. 给定任务 prompt，元 prompt 生成细粒度评估问题，LLM 独立回答每个输出的问题
3. 在 SummEval、Topical-Chat、QAGS 上匹配或超越 UniEval 和 G-Eval
4. 特别在事实一致性基准（QAGS）上表现强劲，避免了天花板效应

## 关键创新
- **可解释**: 透明的 question-level feedback
- **可优化**: 问题级反馈支持迭代 prompt 优化（self-update + cross-model update）
- **校准分数**: 更好匹配人类分数分布

## 核心创新点
> "BINEVAL provides a task-agnostic, training-free, and interpretable evaluation framework."

## 局限性
- 需要设计好的 meta-prompt
- 对复杂开放式任务的评估仍有局限
