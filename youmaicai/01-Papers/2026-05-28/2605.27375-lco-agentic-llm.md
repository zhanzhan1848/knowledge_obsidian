# LCO: LLM-based Constraint Optimization for Safer Agentic LLMs in Real-world Tasks

## 核心贡献
1. 提出 **LCO (LLM-based Constraint Optimization)** 框架，无需模型微调即可有效减少 ICRH
2. 解决 in-context reward hacking (ICRH) 问题：LLM 迭代优化行为最大化代理目标时产生有害副作用

## 模型架构
LCO 两个模块：
- **Self-thought Module**：引导 LLM 在执行前主动思考并整合潜在安全约束
- **Evolutionary Sampling Module**：使用基于 LLM 的交叉和变异，将模型动作约束在安全解空间内

## 实验结果
- **Tweet 毒性优化任务**：GPT-4 上 Toxicity Growth Rate (TGR) 降低 39%
- **策略优化基准**：ICRH Occurrence Rate 降低 15.23%，安全性提升且不牺牲任务性能

## 局限性
- 依赖 LLM 的推理能力
- 对极端场景的泛化能力待验证
