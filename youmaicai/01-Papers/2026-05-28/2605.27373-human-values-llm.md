## 核心贡献
1. 提出基于 LLM 的模块化架构，用于检测和量化文本中的人类价值观
2. 三模块设计：价值规范生成 → 文本标注 → 程度评级，分离概念化与检测任务
3. 避免了对特定价值理论的依赖或复杂的 prompt 工程

## 模型架构
三个协调模块：
- **模块1**：从理论框架的基础文本生成结构化价值规范
- **模块2**：使用这些规范标注文本
- **模块3**：基于修辞和语义证据分配等级支持/抵抗

## 训练方法
- 使用多个 LLM 实例化架构
- 在 ValueEval 数据集上评估

## 实验结果
- 展示良好的检测性能，验证了管线的通用性

## 局限性
- 依赖理论基础的选择
- 需要进一步验证在不同文化背景下的适用性
