# 🥬 LLM 论文分析：Transformer with Temporal Middle-Layer Recurrence

## 基本信息
- **标题**: Transformer with Temporal Middle-Layer Recurrence (T2MLR)
- **arXiv**: [arXiv:2607.15178](https://arxiv.org/abs/2607.15178) [cs.CL, cs.AI]
- **链接**: [原文](https://arxiv.org/abs/2607.15178)
- ** Submitted**: 2026-07-16

## 核心贡献
1. 提出 **T2MLR**：一种基于 Transformer 的潜在推理架构，解决自回归解码过程中中间推理状态无法跨时间步持久化的问题
2. 将前一 token 的**中间层表示**缓存并直接融合到当前 token 的**较早层**，使抽象中间计算能以很少的推理开销跨解码步骤持久化
3. 仅对局部中间层块（网络 20%）应用 recurrence 往往优于全层 recurrence
4. **无需从头预训练**：可将 recurrent pathway 改装到现有 1.7B 预训练 Transformer 并微调即可显著提升数学推理

## 模型架构 / 方法

### 核心问题
自回归解码反复通过 token 空间压缩丰富的隐藏计算，使中间推理状态难以跨时间步持久。

### T2MLR 机制
```
前一个 token 的中间层表示 → 直接融合到当前 token 的较早层
```
- 缓存中间层激活
- 跨解码步骤传递抽象推理状态
- 推理开销小

### 关键发现
- 只需 20% 网络层的局部 recurrence 即可优于全层 recurrence
- 证明了有效潜在推理不需要遍历所有层

## 实验结果
- 自然语言预训练和多跳推理微调一致优于参数匹配的 Transformer 基线
- 1.7B 模型微调后数学推理能力显著提升

## 局限性
- 仅在 1.7B 参数规模验证
- 更深层架构的 recurrence 策略未充分探索

## 建议
- **是否推荐使用**：是
- **适用场景**：需要复杂推理的 LLM 应用、长对话上下文建模、数学推理
