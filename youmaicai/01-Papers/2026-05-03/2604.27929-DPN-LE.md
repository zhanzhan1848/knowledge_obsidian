## 核心贡献
1. **冗余性问题发现**：现有神经元编辑方法修改大量神经元（~20000个/特质），导致模型通用能力显著下降
2. **双向人格神经元定位**：利用对抗性人格特质之间的互斥性，通过对比高/低特质样本的MLP激活差异精准定位人格特异性神经元
3. **稀疏干预策略**：仅干预约0.5%的神经元，实现人格控制与能力保留的平衡，比NPTI减少96%以上修改量

## 模型架构

### 核心发现
- **多功能神经元**：神经元同时关联人格特质和通用知识
- **互斥表征模式**：对立的人格特质（如高外向vs低外向）在表征空间形成明显可分簇
- PCA分析显示LLaMA-3-8B-Instruct在第12层开始出现明显的人格分离

### 方法流程（DPN-LE）
```
1. 收集对比样本对 D = (x_i^+, x_i^-)，分别代表高/低特质
2. 构建层-wise steering vector：
   s_l = (1/N)(Σh_{i,l}^+ - Σh_{i,l}^-)
3. 双重过滤选择神经元：
   - Cohen's d 效应量过滤：|d_l| > τ_d (e.g., 0.8)
   - 激活幅度过滤：|s_l| > τ_q (分位数阈值)
4. 推理时稀疏线性干预：
   DPN-LE: h~_{i,l} = h_{i,l} + γ·s_{i,l}
   DPN-LEw: h~_{i,l} = h_{i,l} + γ·s_{i,l}·w_{i,l} (加权版本)
```

## 关键公式

### Steering Vector
```
s_l = (1/N)(Σ_{i=1}^N h_{i,l}^+ - Σ_{i=1}^N h_{i,l}^-)
```

### Cohen's d 效应量
```
d_l = (1/N)(Σh_{i,l}^+ - Σh_{i,l}^-) / σ_pooled
```

### 稀疏干预
```
DPN-LE: h~_{i,l} = h_{i,l} + γ·s_{i,l}, i ∈ N
DPN-LEw: h~_{i,l} = h_{i,l} + γ·s_{i,l}·w_{i,l}, i ∈ N
```

## 实验结果

### 人格控制效果（LLaMA-3-8B-Instruct）
| 方法 | 人格得分 | 修改神经元数 |
|------|---------|-------------|
| NPTI | 9.43 | ~20000 |
| DPN-LE | 9.11 | ~1000-1500 (0.5%) |

### 通用能力保留（γ=0.8）
| 任务 | NPTI 高特质 | NPTI 低特质 | DPN-LEw 高特质 | DPN-LEw 低特质 |
|------|------------|------------|---------------|---------------|
| GSM8K | -16.00% | -40.79% | -7.08% | -5.93% |
| HotpotQA EM | -1.46% | -1.46% | <1.04% | <1.04% |
| TriviaQA EM | -5.12% | -6.46% | -3.98% | -5.86% |

### 超参数
- LLaMA应用层：12-31层
- Qwen应用层：14-27层
- 分位数阈值：q=0.995（top 0.5%）
- Cohen's d阈值：τ_d=0.8
- 每层约70个神经元

## 局限性
- 仅在LLaMA-3-8B-Instruct和Qwen2.5-7B-Instruct上验证
- 低特质方向（尤其Extraversion）仍有较大能力下降（-17.89%）

## 建议
- **是否推荐使用**：是，尤其适合需要人格控制且要求保持推理能力的场景
- **适用场景**：社交AI、角色扮演、个性化对话系统
- **改进方向**：针对低特质方向的能力保留
