# 🥬 LLM 论文分析：EEVEE - Test-time Prompt Learning

## 基本信息
- **标题**: Towards Test-time Prompt Learning in the Real World for Self-Improving Agents
- **作者**: Weixian Xu et al.
- **发表**: cs.LG, 2026-06-09
- **链接**: [arXiv:2606.11182](https://arxiv.org/abs/2606.11182)

## 核心贡献

1. **问题**: 现有方法设计用于单数据集设置，但现实应用需要模型处理来自多数据集、多领域、多任务分布的异构输入流

2. **解决方案**: 提出 **EEVEE**，首个多数据集 test-time prompt learning 框架

3. **核心设计**:
   - **Router**: 将输入分配到任务集群，分配给合适的 prompt 配置
   - **Router-prompt co-evolution**: 交替进行 router 和 prompt 学习阶段，解决相互依赖

4. **跨数据集干扰缓解**: Router-partition incoming inputs into task clusters

## 实验结果

| 模型 | 提升 |
|------|------|
| Qwen3-4B-Instruct | +10.38 多基准平均分 |
| DeepSeek-V3.2 | +24.32 多基准平均分 |

- 超越 SOTA 方法 GEPA 和 ACE 高达 37.2% 和 48.2%
- 提高异构数据流下的鲁棒性
- 保持单基准学习能力和效率

## 方法论

### Router-Prompt Co-evolution
```
交替优化:
1. Router learning phase
2. Prompt learning phase
```

### Task Clustering
- 输入按任务聚类
- 分配到合适的 prompt 配置

## 局限性

- 需要多数据集场景
- Router 训练开销

- **是否推荐使用**: 是

- **优先级**: 中
