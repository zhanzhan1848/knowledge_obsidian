# 🥬 LLM 论文分析：HiViG - History-Aware Visually Grounded Critic

## 基本信息
- **标题**: A History-Aware Visually Grounded Critic for Computer Use Agents
- **作者**: Jaewoo Lee et al.
- **发表**: cs.AI, 2026-06-09
- **链接**: [arXiv:2606.11078](https://arxiv.org/abs/2606.11078)
- **代码**: https://github.com/G-JWLee/HiViG

## 核心贡献

1. **问题**: 现有 Computer Use Agents (CUAs) 的 test-time critic 存在两个关键局限：
   - 短视决策循环（忘记早期行动）
   - 缺乏视觉 grounding（无法检测点击错误 UI 元素等失败）

2. **解决方案**: 提出 **HiViG**，History-aware Visually Grounded test-time 框架

3. **核心设计**: 
   - 多模态 critic，在真实 GUI 轨迹上训练
   - 将过去交互抽象为紧凑记录
   - 用视觉 grounding 评估行动

4. **test-time 集成**:
   - **Macro-action history**: 总结策略的已完成成就
   - **Visually grounded critique**: 验证原始执行坐标与当前截图，拦截执行前错误

## 实验结果

| 模型 | 提升 |
|------|------|
| Qwen3-VL-32B | +5.8% |
| Gemini-3-Flash | +9.0% |

- 跨 web、mobile、desktop 基准一致优于现有 scalar 和 verbal critics
- 展示强跨平台泛化能力

## Ablation 发现

- Macro-action history 缓解短视规划
- Visually grounded critique 减少执行错误
- 两个组件对长视野 GUI 任务的 test-time scaling 都至关重要

## 局限性

- 限于 GUI 任务
- 计算开销未详细评估

## 建议

- **是否推荐使用**: 是
- **适用场景**: Computer Use Agents、GUI automation
- **优先级**: 中高
