## 核心贡献

1. **发现 α-Law**: 首次识别 LLM 概率更新的乘性缩放定律
2. **稳定性理论**: 证明 α < 1 是渐近稳定的充要条件
3. **架构差异**: 揭示 GPT-5.2 和 Claude 在信任比例上的系统性差异

## 核心创新

### 1. α-Law 数学表达

```math
P_{t+1}(y|x) \propto P_t(y|x)^{\alpha} \cdot E(y|x)^{1-\alpha}
```

其中:
- `P_t`: 第 t 步的先验概率
- `E`: 验证证据
- `α`: 信念修订指数 (belief revision exponent)

### 2. 稳定性条件

```
α < 1  →  渐近稳定 (推荐)
α = 1  →  边界稳定
α > 1  →  不稳定 (发散)
```

**理论贡献**: α < 1 是重复修订下渐近稳定的**充要条件**

### 3. 实验发现

| 模型 | 单步 α | 多步趋势 | 信任比例 |
|------|--------|----------|----------|
| GPT-5.2 | ~1.1 (略高于边界) | 递减 → 收敛 | 平衡先验与证据 |
| Claude Sonnet 4 | ~1.1 | 递减 → 收敛 | 略偏向新证据 |

## 实验设计

### 数据集
- **规模**: 4,975 问题
- **基准**:
  - GPQA Diamond (研究生级别)
  - TheoremQA
  - MMLU-Pro
  - ARC-Challenge

### 实验类型
1. **单步修订**: 观察初始 α 值
2. **多步修订**: 验证 α 递减趋势
3. **Token 级验证**: Llama-3.3-70B 确认

## 关键发现

### 1. 近贝叶斯行为
- 指令微调的 LLM 表现出**接近贝叶斯更新**的行为
- 单步修订略高于稳定性边界
- 多步修订自动收敛到稳定区域

### 2. 架构特异性
```
GPT-5.2:
- 平衡先验与证据
- 信任比例 ≈ 0.5

Claude Sonnet 4:
- 略偏向新证据
- 信任比例 < 0.5
```

### 3. 长期动力学
- α 随修订步数**递减**
- 产生**收缩性**长期动力学
- 与理论稳定性预测一致

## 理论意义

### 1. 可观测行为 vs 内部推理
- 本文刻画的是**可观测的推理时更新行为**
- 不声称模型内部进行贝叶斯推理
- 提供外部诊断工具

### 2. 诊断价值
```
α-Law 作为诊断工具:
- 监控更新稳定性
- 评估推理质量
- 比较不同架构
```

## 数学细节

### 贝叶斯更新对比
```math
\text{Bayes: } P(H|E) \propto P(H) \cdot P(E|H)

\text{α-Law: } P_{t+1} \propto P_t^{\alpha} \cdot E^{1-\alpha}
```

当 α = 0 时，完全依赖证据
当 α = 1 时，完全依赖先验

### 收敛性证明 (概要)
- 定义 Lyapunov 函数
- 证明 α < 1 时单调递减
- 收敛到唯一不动点
