## 核心贡献

1. **Contrastive Reflection framework**：面向 agentic IR 工作流的迭代 prompt 优化框架
2. **Task-centric quality definition**：QA agents 暴露检索或推理 traces，grading agents 暴露维度级分数和理由
3. **Error-anchored behavioral slices**：识别错误锚定的行为切片，从同一区域添加附近成功示例
4. **Tree-based slice selector**：用树结构选择切片，但核心贡献是 contrastive reflection loop 而非树本身

## 实验结果

在 HotpotQA 检索增强 QA 设置上：
- 单次 tree-selected contrastive repair：exact-match 从 **51.4% → 60.4%** (+9.0%)
- Failure-only 和 random-evidence 变体改进更少且破坏更多先前正确案例
- 与现代 prompt optimizer 比较：MIPROv2 达 59.4%，GEPA 达 57.0%

## 关键方法

1. **Contrastive repair loop**：识别失败案例 → 添加成功示例 → Teacher LLM 提出 targeted prompt edit
2. **Validation-driven**：候选编辑仅在验证性能改进时接受
3. **Interpretable**：使 prompt 修复更可检查、更验证驱动
