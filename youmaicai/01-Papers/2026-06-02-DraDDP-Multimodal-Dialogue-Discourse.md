# 🥬 DraDDP: Multimodal Multi-Party Dialogue Discourse Parsing

## 基本信息
- **标题:** DraDDP: A Multimodal Multi-Party Dialogue Discourse Parsing Dataset
- **作者:** Shannan Liu, Peifeng Li, Yaxin Fan, Qiaoming Zhu
- **链接:** [arXiv:2606.00012](https://arxiv.org/abs/2606.00012)
- **arXiv:** arXiv:2606.00012
- **分类:** cs.CL, cs.AI

## 核心创新点

1. **首个公开的多模态多轮对话话语解析数据集** — 基于美国电视节目
2. **495 个对话片段，6,374 个话语，9.1 小时并行视频内容**
3. **综合 benchmark** — 评估多模态信息对捕捉对话结构和关系类型的影响

## 问题背景

- 现有话语解析研究局限于文本模态或双人对话
- 实际场景需要多模态和多方的设置

## 数据集特点

| 特性 | 值 |
|------|-----|
| 话语数 | 6,374 |
| 视频时长 | 9.1 小时 |
| 对话片段 | 495 |
| 来源 | 美国电视节目 |
| 任务 | 话语依存结构 + 关系类型 |

## 实验结果

- 多模态信息对捕捉对话结构和关系类型有显著价值
- 将公开发布数据集、标注指南和代码

## 局限性

- 目前仅英文
- 数据来自电视剧，可能存在 domain gap

- **推荐程度:** ⭐⭐⭐⭐
- **适用场景:** 多模态对话理解、话语解析、视频理解
