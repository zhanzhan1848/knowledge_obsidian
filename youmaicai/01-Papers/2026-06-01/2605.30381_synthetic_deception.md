# 2605.30381: When LLMs Learn to Be Consistently Wrong

## 元信息
| 属性 | 内容 |
|------|------|
| 标题 | When LLMs Learn to Be Consistently Wrong: A Multi-Model Study of Linear Representations of Synthetic Deception |
| arXiv | [2605.30381](https://arxiv.org/abs/2605.30381) |
| 分类 | cs.LG, cs.AI |
| 作者 | 多模型研究 (Pythia-1.4B, Gemma-2-2B/9B, Qwen2.5-7B, Llama-3.1-8B) |

## 核心贡献
1. **合成不诚实检测范式**: 对5个Transformer模型（诚实/欺骗变体）用LoRA在同一问题分布上fine-tune，研究欺骗的表征基础
2. **线性探针检测**: 在第1-3层训练的线性探针即可检测合成不诚实，AUC ≥ 0.99（Gemma/Qwen/Llama），Pythia达0.705
3. **强泛化能力**: 在TruthfulQA上训练的探针可泛化到MMLU held-out主题，Delta AUC ≈ 0
4. **两类机制**: 表征崩溃（Pythia/Llama/Qwen） vs 高维保持（Gemma-2）

## 关键发现
- 不诚实方向在深层逐渐巩固
- 最优校准（ECE < 0.01除Pythia外）在第1-4层可达
- late-layer表征对Gaussian噪声具有强鲁棒性（Gemma-2尤为突出）
- 通过激活监测可检测模型是否被训练为持续输出错误答案

## 关键指标
- AUC ≥ 0.99 (4 architectures, layers 1-3)
- Pythia-1.4B peak AUC: 0.705
- ECE < 0.01 (except Pythia)
