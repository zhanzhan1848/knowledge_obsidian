## 🔥 重点推荐

### 1. Transformers are Bayesian Networks
- **arXiv**: 2603.17063
- **作者**: Gregory Coppola
- **领域**: 理论基础
- **核心贡献**: 
  - 首次证明 sigmoid Transformer 在任意权重下实现加权循环置信传播
  - 揭示 Attention = AND, FFN = OR 的布尔结构
  - 形式化验证了幻觉是"无概念操作的结构性后果"
- **意义**: 从概率图模型角度重新理解 Transformer

### 2. PRISM: Demystifying Retention and Interaction in Mid-Training
- **arXiv**: 2603.17074
- **作者**: Bharat Runwal 等
- **领域**: 训练策略
- **核心贡献**:
  - 跨 7 个模型家族的系统性 mid-training 研究
  - 27B 高质量 token 提升 math +15-40pts, code +5-12pts
  - 数据组合在 mid-training 阶段最重要，而非 RL 阶段
  - mid-training 密集重构 >90% 权重，RL 仅修改 ~5%
- **意义**: 为设计高效的训练流水线提供实证指导

### 3. Transformers Can Learn Rules They've Never Seen
- **arXiv**: 2603.17019
- **作者**: Andy Gray
- **领域**: 推理能力
- **核心贡献**:
  - 通过 XOR 元胞自动机实验证明 Transformer 可学习训练中未见过的规则
  - 排除了"最强插值假说"：Transformer 不能发现未见规则
  - 多步约束传播是关键：无展开时准确率 63.1%，软展开达 96.7%
- **意义**: 证明 Transformer 具备真正的规则学习能力

### 5. Do Understanding and Generation Fight? DPO for Unified Multimodal Models
- **arXiv**: 2603.17044
- **核心发现**: DPO 无法同时优化理解与生成能力
- **原因**: 理解与生成梯度近正交 (~0)，11-14x 幅度不平衡
- **瓶颈**: 离散 VQ tokenization

### 6. Knowledge Localization in MoE LLMs Using Cross-Lingual Inconsistency
- **arXiv**: 2603.17102
- **方法**: 利用跨语言不一致性定位 MoE 中的知识专家
- **效果**: 停用 ~20/6000 专家导致 >40% 问题回答错误
- **意义**: 可扩展的 MoE 可解释性方法

### 7. InfoDensity: Rewarding Information-Dense Traces for Efficient Reasoning
- **arXiv**: 2603.17310
- **核心思想**: 高质量推理轨迹应具有低不确定性收敛和单调进展
- **方法**: AUC-based + monotonicity reward + length scaling
- **效果**: 准确率相当，显著减少 token 使用

### 8. Efficient Exploration at Scale (RLHF)
- **arXiv**: 2603.17378
- **核心发现**: 在线学习算法可用 <20K 标签达到离线 RLHF 200K 标签效果
- **技术**: 小幅正向推动 + 认知神经网络 + 信息导向探索
- **意义**: 10x+ 数据效率提升
