# 2026-06-17 LLM/NLP 论文日报

## 今日共收录 7 篇论文

### 多模态与知识编辑
- **DECODE** (arXiv:2606.17057): 发现 MLLM 知识编辑中的模态解耦问题，提出针对模态特定神经元的编辑方法

### Transformer 与推理优化
- **KV Cache 可编辑性** (arXiv:2606.17107): 将 KV Cache 重新解释为预填充时写入的笔记，实现可编辑可组合的缓存

### BERT 与应用
- **InferBERT 模型选择** (arXiv:2606.17113): BioBERT 在药物警戒因果发现中优于更大模型，领域预训练是关键

### MoE 与量化
- **MODE** (arXiv:2606.17118): 模态分解的 MoE-MLLM 混合精度量化，W3A16 下性能损失 <2.9%

### Prompt 工程
- **PromptMN** (arXiv:2606.17164): 伪提示领域特定语言，提升人机交互清晰度和可复用性

### LLM 安全与遗忘
- **RepSelect** (arXiv:2606.17168): 通过表示选择性实现鲁棒 LLM 遗忘，抵抗微调攻击

### Agent 与记忆
- **MemSlides** (arXiv:2606.17162): 分层记忆框架，分离用户画像、工具记忆和工作记忆
