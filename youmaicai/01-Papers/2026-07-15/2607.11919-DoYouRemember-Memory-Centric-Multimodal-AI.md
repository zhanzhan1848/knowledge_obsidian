# 🥬 LLM 论文分析：DoYouRemember - Toward Memory-Centric Multimodal AI

## 基本信息
- **作者**：Xuguang Yu, Weigang Zheng, Minyue Yu
- **发表**：arXiv — 2026-07-07
- **链接**：[原文](https://arxiv.org/abs/2607.11919) | arXiv:2607.11919
- **主题**：cs.NE / cs.CV / cs.LG — 多模态 + 神经演化计算

## 核心贡献
1. **问题**：人类记忆是重建性的，但当前 MLLM 处理图像后丢弃内部表示——LLM "理解" 但不 "记忆"
2. **DoYouRemember 三阶段架构**：
   - VQ-VAE 压缩图像为离散 visual tokens
   - LoRA-fine-tuned LLM 联合处理 visual + text tokens
   - Diffusion Decoder 从 LLM hidden states 重建图像
3. **关键发现**：
   - LLM hidden states 含约零可恢复视觉信息（Decoder 从 VQ-VAE tokens 重建清晰，从 LLM hidden states 产生纯噪声）
   - 共享 memory matrix M 的 backpropagation 训练因梯度消失失败（O(1/√N) 衰减）
   - **Local EMA 更新解决三个根本原因**：每张图仅更新 top-8 slots（共64），保持 slot 间多样性
   - M（229K 参数，16x 压缩）接近 VQ 上界；1,024 slots 超越 VQ（LPIPS 0.056 vs 0.071）
4. **信息论框架**：记忆是有损压缩，回忆是解压，幻觉是有损解压的固有属性而非缺陷

## 核心洞察
> The LLM understands images but does not remember them. Hallucination is an inherent property of lossy decompression rather than a defect.
