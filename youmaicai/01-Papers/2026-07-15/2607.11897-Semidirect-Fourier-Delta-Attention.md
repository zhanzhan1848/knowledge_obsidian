# 🥬 LLM 论文分析：Semidirect Fourier Delta Attention (SFDA)

## 基本信息
- **作者**：Tiantian Zhang
- **发表**：arXiv — 2026-06-12
- **链接**：[原文](https://arxiv.org/abs/2607.11897) | arXiv:2607.11897

## 核心贡献
1. **SFDA**：Phase-controlled generalization of Kimi Delta Attention，用 block-rotational Fourier control 替代实对角衰减
   ```math
   S_t = (I - \beta_t k_t k_t^*) \Lambda_t S_{t-1} + \beta_t k_t v_t^*, \quad \Lambda_t = \diag(\alpha_t \odot e^{i\theta_t})
   ```
2. **Constructive Chunk-WY Factorization**：
   ```math
   A_t \cdots A_1 = \Gamma_t - Y_t M_t W_t^*
   ```
   固定 chunks 内秩增长有界 → exact affine chunk transfer、形式稳定性和复杂度界
3. **实验验证**：SFDA 在 toy state-tracking 实验中学习 cyclic memory，KDA baseline 仍接近随机

## 与 Linear Attention 的关系
- Linear attention 用固定 recurrent state 替代增长的 KV cache
- SFDA 通过相位控制增强长上下文记忆能力
