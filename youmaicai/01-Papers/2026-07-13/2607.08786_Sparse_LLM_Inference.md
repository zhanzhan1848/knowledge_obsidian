# Accelerating GPU Inference of LLMs with Moderately Unstructured Sparse Weight Matrices

## 元信息
| 标题 | 中度非结构化稀疏权重矩阵加速LLM GPU推理 |
| 作者 | Tao Lu, Haoyu Wang, Zonghui Wang, Keshen Xiang, Jiaheng Zhang, Wenzhi Chen |
| 链接 | [原文](https://arxiv.org/abs/2607.08786) |
| arXiv | arXiv:2607.08786 |
| 会议 | DAC 2026 |
| 领域 | cs.LG, cs.AI, cs.AR |

## 核心贡献
1. **问题**：现有GPU SpMM内核在50%稀疏度下无法超越密集矩阵乘法
2. **三层矩阵存储格式**：
   - **Sparse-TC层**：启用稀疏张量核加速SpMM
   - **Slot-Filling层**：使用并行微分距离进行矩阵压缩，支持低成本on-chip解码
   - **轻量级Residual层**：确保正确的SpMM计算
3. **SpMM内核设计**：联合利用稀疏张量核和CUDA核心

## 核心创新
- 首次在配备HBM的现代GPU上超越密集矩阵乘法
- 核级加速比SpInfer（EuroSys'25 Best Paper）高达1.64x
- 端到端加速比FlashLLM（VLDB'24）高达1.41x

## 实验结果
| 对比方法 | 加速比 |
|----------|--------|
| SpInfer (EuroSys'25) | 最高1.64x |
| FlashLLM (VLDB'24) | 最高1.41x |
