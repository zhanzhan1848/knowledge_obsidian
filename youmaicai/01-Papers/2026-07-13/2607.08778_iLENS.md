# iLENS: Interpretable LLM-Guided Mixture-of-Experts for Neuroimaging Survival Analysis

## 元信息
| 标题 | iLENS: 可解释LLM引导的MoE神经影像生存分析 |
| 作者 | Farica Zhuang, Seong Woo Han, Zixuan Wen, Shu Yang, Yize Zhao, Li Shen |
| 链接 | [原文](https://arxiv.org/abs/2607.08778) |
| arXiv | arXiv:2607.08778 |
| 领域 | cs.LG, cs.AI |

## 核心贡献
1. **iLENS框架**：结合LLM + MoE用于阿尔茨海默病（AD）转换预测的生存分析
2. **LLM合成能力**：使用LLM综合结构化神经影像测量和非结构化信息引导expert路由
3. **可解释性**：提供透明、生物学基础的路由决策解释，弥合高性能生存分析与可解释临床决策支持之间的差距
4. **患者亚型分析**：展示患者亚型能力

## 应用场景
- 阿尔茨海默病（AD） prodromal stage转换预测
- 临床决策支持

## 实验结果
- 竞争力预测性能
- 患者亚型能力

## 局限性
- 特定疾病领域，泛化性待验证
- LLM引入的额外计算成本

## 建议
- **是否推荐使用**：是（医学AI领域）
- **适用场景**：医学影像分析、临床生存预测、可解释AI医疗系统
