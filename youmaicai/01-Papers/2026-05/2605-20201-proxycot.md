# ProxyCoT: Long-Context Reasoning Through Proxy-Based Chain-of-Thought Tuning

## 核心贡献
1. **ProxyCoT 框架**: 将短 proxy context 的推理能力迁移到长 full context
2. **两阶段训练**: RL/蒸馏获取 CoT traces → SFT 接地到长上下文
3. **泛化能力**: 训练后的模型能将长上下文推理能力泛化到 out-of-domain 任务

## 核心问题
- 现代 LLM 支持长达 10 million tokens 的输入
- 但在需要复杂推理的长上下文任务上表现不佳
- 这些任务实际上只需要输入的一个子集 (proxy context) 即可解决
- 模型在 proxy context 和 full context 上存在显著性能差距

## 方法
```
Stage 1: Proxy Context Training
- 在 proxy contexts 上通过 RL 或蒸馏获取高质量 CoT reasoning traces
- 使用更大 teacher model 蒸馏

Stage 2: Grounding to Full Context  
- 使用 SFT 将生成的 traces 接地到完整长上下文
- supervised fine-tuning with proxy CoT traces
```

## 实验结果
- 在多个数据集上持续优于强基线
- 降低计算开销
- 展示跨任务泛化能力

## 关键洞察
**为什么有效**: 尽管共享相同的底层推理过程,模型在 proxy 和 full contexts 上表现差异大 —— 因为完整上下文引入的干扰和冗余信息影响了模型的推理过程
