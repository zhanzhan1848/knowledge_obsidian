# ReElicit: Dynamic Representations for Bayesian Optimization of System Prompts

## 核心贡献
1. **ReElicit 框架**: 提出基于"elicitation by embedding"的贝叶斯优化框架,用于优化系统提示词
2. **自适应语义表征**: LLM 根据任务描述和历史评估动态构建紧凑可解释的特征空间
3. **样本高效优化**: 在仅有聚合指标(无 per-example 标签)的情况下实现高效优化

## 核心思想
**问题设定**: 
- 系统提示词是控制 AI 行为的核心机制
- 但优化只能获得聚合反馈(aggregate metrics),而非 per-example 标签或错误

**方法**:
1. 给定任务描述 + 历史评估的 prompts + scalar scores
2. LLM elicits 一个紧凑、可解释的特征空间并映射 prompts 到该空间
3. 使用高斯过程 surrogate model + acquisition function 选择目标特征向量
4. LLM 将特征向量实现并 refine 成可部署的系统提示词
5. 随着新评估到达,Re-eliciting 特征空间以适应观察到的 prompt-score 历史

## 技术细节
```math
\text{Gaussian Process Surrogate} \rightarrow \text{Acquisition Function} \rightarrow \text{LLM Feature Realization}
```

## 实验结果
- 10 个系统提示词优化任务,每任务 30 次评估预算
- ReElicit 在聚合性能上优于其他 aggregate-only prompt 优化 baselines

## 关键创新
**LLM as Adaptive Semantic Representation Builder**: LLM 不仅生成 prompt,还作为自适应语义表征构建器用于贝叶斯优化

## 建议
- 是否推荐使用: **是**
- 适用场景: AI 产品 Prompt 调优、Agent 系统提示词优化、对话系统行为控制
