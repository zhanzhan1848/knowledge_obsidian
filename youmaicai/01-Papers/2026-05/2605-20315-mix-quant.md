# Mix-Quant: Quantized Prefilling, Precise Decoding for Agentic LLMs

## 元信息
| 标题 | Mix-Quant: Quantized Prefilling, Precise Decoding for Agentic LLMs |
|------|------|
| 作者 | Haiquan Lu, Zigeng Chen, Gongfan Fang, Xinyin Ma, Xinchao Wang |
| 链接 | [原文](https://arxiv.org/abs/2605.20315) |
| arXiv | arXiv:2605.20315 |
| 领域 | LLM 量化, Agent 系统, 推理加速 |

## 核心贡献
1. **Phase-aware 量化框架**: 对 prefilling 阶段使用 NVFP4 量化,对 decoding 阶段保持 BF16 精度
2. **3x Prefilling 加速**: 在保持任务性能的同时实现 prefilling 阶段最高 3 倍加速
3. **解决 Agentic LLM 瓶颈**: 针对多轮交互、长上下文推理中的 prefilling 开销问题

## 核心发现
- 对整个推理过程应用 FP4 量化会导致显著性能下降
- **Prefilling 阶段表现出大量量化冗余**,可以以最小精度损失进行量化,尽管它是计算的主要来源
- Decoding 阶段对量化敏感,需要保持高精度

## 方法
```
Agentic LLM Workflow:
[Planning] → [Tool Use] → [Memory Retrieval] → [Multi-step Interaction]
     ↑
Prefilling 阶段是计算密集型瓶颈 (主导计算来源)

Mix-Quant:
Prefilling: NVFP4 (高吞吐量)
Decoding: BF16 (高精度)
```

## 实验结果
- 在长上下文和 agentic benchmarks 上验证
- 大幅保留任务性能的同时显著提升效率

## 适用场景
- LLM Agent 部署
- 长上下文多轮对话系统
- 边缘设备推理
