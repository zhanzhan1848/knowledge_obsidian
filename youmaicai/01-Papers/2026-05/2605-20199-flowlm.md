# FlowLM: Few-Step Language Modeling via Diffusion-to-Flow Adaptation

## 元信息
| 标题 | FlowLM: Few-Step Language Modeling via Diffusion-to-Flow Adaptation |
|------|------|
| 作者 | Runzhe Zhang, Letian Chen, Wenpeng Zhang, Zhouhan Lin, Peilin Zhao |
| 链接 | [原文](https://arxiv.org/abs/2605.20199) |
| arXiv | arXiv:2605.20199 |
| 领域 | Diffusion Models, Language Generation, Flow Matching |

## 核心贡献
1. **Diffusion-to-Flow Adaptation**: 将预训练的扩散语言模型转换为流匹配语言模型
2. **Few-Step 高质量生成**: 几步采样即可达到 2000 步扩散采样的质量
3. **高效训练**: 微调 FlowLM 只需一半 epochs 即可达到性能饱和

## 核心思想
- 扩散模型的弯曲采样轨迹可以通过 flow matching 重新对齐为直线
- 直线 flow 使得少步生成即可获得高质量输出

## 方法
```math
\text{Diffusion Model} \xrightarrow{\text{Fine-tuning}} \text{FlowLM}
```
通过高效微调将扩散模型的弯曲采样轨迹重新对齐为直线流

**训练目标**: predicting clean data, 引导采样过程走向真实数据分布

## 实验结果
- 微调 FlowLM 仅需一半 epoch 即可达到性能饱和
- 大幅超越原始扩散模型
- 少步生成质量可媲美 2000 步扩散采样

## 关键优势
| 方面 | 优势 |
|------|------|
| 采样速度 | 几步 vs 2000 步 |
| 训练效率 | 一半 epochs 达到饱和 |
| 生成质量 | 媲美大规模扩散采样 |
