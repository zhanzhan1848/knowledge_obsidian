# Generalization in LLM Problem Solving: The Case of the Shortest Path

## 基本信息
- **arXiv**: [2604.15306](https://arxiv.org/abs/2604.15306) [cs.AI]
- **作者**: Yao Tong, Jiayuan Ye, Anastasia Borovykh, Reza Shokri
- **发表**: 2026-04-16
- **链接**: [PDF](https://arxiv.org/pdf/2604.15306)

## 核心贡献

1. **合成控制环境**：基于最短路径规划构建 LLM 系统泛化研究的合成环境，支持两个正交泛化轴——空间迁移（未见地图）和长度扩展（更长时间范围）

2. **因子分离分析**：清晰分离训练数据、训练范式和推理时策略对 LLM 泛化的影响

3. **关键发现**：
   - 模型表现出强空间迁移能力
   - 在长度扩展上持续失败（递归不稳定）
   - RL 提升训练稳定性但不扩展能力边界
   - 推理时 scaling 提升性能但无法拯救长度泛化失败
   - 数据覆盖设定能力上限

## 模型架构

基于最短路径规划的合成环境：
- **空间迁移**：在未见地图上测试
- **长度扩展**：在更长视野问题上测试
- **问题类型**：可组合顺序优化问题

## 核心发现详解

### 递归不稳定性
长度扩展失败的根本原因是递归不稳定——模型在多步推理中无法正确传播和累积中间状态信息。

### 各阶段影响
| 阶段 | 作用 |
|------|------|
| 数据覆盖 | 设定能力上限 |
| 强化学习 | 提升训练稳定性，不扩展能力上限 |
| 推理时 Scaling | 提升性能，无法拯救长度泛化失败 |

## 实验设置
- 合成最短路径规划问题
- 控制训练数据分布
- 测试空间和长度两个泛化维度

## 局限性

- 仅限合成环境，真实世界泛化能力待验证
- 最短路径任务可能无法完全代表 LLM 的其他推理能力
