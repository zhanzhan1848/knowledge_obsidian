## 核心贡献
1. **首个将 token 本身作为矩阵李群元素（bare group element）的注意力机制**，无特征载荷、无外部动作
2. 提出 **Lie-Algebra Attention**：注意力分数为相对姿态的闭式代数范数，而非学习核函数
3. 达到其他方法必须排除的仿射全帧群（affine full-frame groups）：SE(2)、SO(3)、Aff(2) 等
4. 所需 score 参数比学习 MLP 核少 50-80 倍，同时保持相当或更好的性能

## 模型架构

### 核心思想
- Token 是矩阵李群 $G$ 的元素 $g_i$，而非向量
- 相对几何：$g_i^{-1} g_j$ 是规范的（canonical）
- 成对不变量：$w_{ij} = \log(g_i^{-1} g_j)$ 是 intrinsic 而非人为设计的
- 对角 $G$ 作用下的等变性（equivariance）是 tautological 的，cocycle 条件自动满足

### 注意力分数（闭式）
$$s_{ij} = -\|\log(g_i^{-1} g_j)\|_\lambda^2 / \tau$$

这是块加权 Frobenius 内积下的规范邻近核，无需：
- 不可约表示（irreps）
- 球面调和函数（spherical harmonics）
- Clebsch-Gordan 积
- 学习核

### 适用群
- 任何在所选对数图（logarithm chart）内包含相对姿态的矩阵李群
- 包括非紧致、非阿贝尔的仿射群（带 scale 和 shear），这是向量 token 注意力方法无法触及的

## 实验结果
- SE(2) 上：闭式分数匹配同不变量的学习 MLP 核，同时参数少 50-80 倍
- SO(3) 上：性能相当
- Aff(2) 上：同样有效
- 向量 token 基线：不变性被打破 5-12 个数量级

## 局限性
- 仅在序列补全任务上验证，尚未扩展到自然语言任务
- 实现依赖于李群的具体表示选择
