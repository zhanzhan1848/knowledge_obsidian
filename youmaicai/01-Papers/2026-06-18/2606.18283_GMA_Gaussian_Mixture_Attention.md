## 核心贡献

1. **问题**：标准 dot-product attention 的密集 token-to-token 交互模式是将 Transformer 扩展到长上下文的中心瓶颈。

2. **Gaussian Mixture Attention (GMA)**：提出一种概率 attention-style 序列混合器，用 K 个学习到的 Gaussian mixture components 路由替代显式 pairwise query-key 比较。

## 核心方法

### 关键机制
- Queries 和 keys 被映射到共享 latent routing space 上的 posterior responsibility vectors
- 它们的 overlap 定义了 implicit responsibility-space affinity
- Values 被写入 K-slot latent memory 并从中读取
- 利用矩阵乘法的结合律，避免构造 N×N affinity matrix
- 主导激活存储规模为 O(NK) 而非 O(N²)

### 变体
- Bidirectional GMA
- Causal GMA

### 理论分析
- Responsibility-modulated gradient structure
- Constrained non-negative low-rank affinity interpretation
- Local routing stability

## 实验结果

- GMA 展示预期的固定 K 线性内存扩展
- 在长文本分类上与 attention-style baselines 竞争
- Causal GMA 在 WikiText-103 上优于测试的 linear/random-feature attention 变体
- 当前实现仍落后于优化的 causal SDPA 和 Mamba

## 分析
- 学习的 responsibilities 显示广泛的组件使用
- 与 surface-form token categories 有适度对齐
- 支持 GMA 作为概率、可解释的、固定 K 的线性时间 attention-style 替代方案，而非通用替代
