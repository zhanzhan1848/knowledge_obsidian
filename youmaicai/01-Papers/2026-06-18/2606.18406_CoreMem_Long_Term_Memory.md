## 核心贡献

1. **问题**：个性化对话 agent 需要跨多会话维持连贯的长期记忆，但在消费级硬件（如 8GB VRAM 边缘设备）上部署面临严重的内存和计算瓶颈。现有方法使用 isotropic cosine similarity 进行检索和启发式规则进行上下文压缩，缺乏统一理论基础，存在 hubness 问题和句法碎片化。

2. **CoreMem**：提出信息几何统一构建的 resource-efficient edge-cloud memory architecture。

3. **Riemannian Retrieval**：用局部自适应 Fisher-Rao metric 替代 cosine matching，通过 Mahalanobis distance 惩罚 hub memories，实现 O(Ndr) Woodbury 加速的实时搜索。

4. **Fisher-guided Discrete Token Distillation (FDTD)**：层次化 sentence-to-token 压缩机制，从 Fisher information traces 推导敏感性分数，提供原则性的 compression-KL tradeoff，并包含显式结构句法保护。

## 实验结果

在 LOCOMO 和 LongMemEval-S benchmarks 上评估：
- Open-domain reasoning: +4.51 pp
- Temporal reasoning: +4.17 pp
- 在严格 8GB VRAM 预算内运行
