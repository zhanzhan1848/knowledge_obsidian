## 核心贡献

1. **Workflow generation as meta-learning**：将 workflow 生成作为元学习问题，给定任务和算子集，模型学习组合解决策略
2. **两阶段训练**：
   - Stage 1: 在合成 workflow 数据上进行监督微调
   - Stage 2: 使用 RLVR (Reinforcement Learning with Verifiable Rewards)，跨任务实例执行反馈改进端到端成功
3. **强泛化能力**：为训练任务生成有效 workflow，对未训练任务和新算子集展示强零样本泛化

## 应用基准

- Question Answering
- Code Generation
- Mathematical Reasoning

## 关键特点

- 工作流编码了任务级重复算法模式，提供跨实例变体的稳健性
- 可解释的 traces 用于调试
- 跨问题实例的可重用性
- 零样本泛化到新算子集

## 建议
- 是否推荐使用：**是**，工作流生成的 promising 方向
- 适用场景：代码生成、工作流自动化、LLM agent 任务规划
