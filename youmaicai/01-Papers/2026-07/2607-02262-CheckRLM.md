# CheckRLM: RAG 推理中的知识-思维一致性检查

## 基本信息
| 标题 | Effective Knowledge-Thought Coherence Checking in Retrieval-Augmented Reasoning |
|------|-----|
| **arXiv** | [2607.02262](https://arxiv.org/abs/2607.02262) |
| **作者** | Dingling Xu, Ruobing Wang, Qingfei Zhao et al. |
| **类别** | cs.CL |
| **发表** | ACL 2026 |

## 核心贡献

1. **CheckRLM**: 通过 RAG 在推理过程中及时检查和纠正事实错误的框架
2. 从推理链中提取事实主张，识别和定位知识不一致
3. 最小成本精确纠正机制，确保推理链与正确知识之间的一致性

## 技术方法

```
推理链 → 事实主张提取 → 不一致检测 → 最小成本纠正 → 修正后的推理链
```

## 关键创新

- **错误积累缓解**：在长 horizon 推理中特别有效
- **低成本**：低于其他方法的计算成本实现更好性能
- **即时检查**：在推理过程中及时发现和纠正错误

## 实验结果

- 在知识密集型任务上显著优于 baseline
- 有效缓解长链推理中的错误积累问题
