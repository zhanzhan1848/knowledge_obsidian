# Prompt Compression via Activation Aggregation

## 元信息

|------|-------|
| 作者 | Semira Einsele |

## 核心贡献

1. **问题**：大型语言模型通过数十层激活传播处理提示，能否将任务相关信息压缩为单一激活向量并重新注入模型，替代原始 Token 序列？
2. **方案**：使用**学习加权求和**中间层激活并注入早期层，压缩率极高且精度损失 < 2%
3. **发现**：揭示 LLM 激活空间的三个结构特性

## 核心方法

### 压缩机制
```
原始提示 → 通过 Target LLM 前向传播
         → 提取中间层激活（加权求和）
         → 注入 Target LLM 早期层
         → 生成响应
```

### 关键发现（激活空间结构）
1. **中层表征可迁移到早期层**：中间层表征能有效迁移到浅层使用，说明信息编码具有跨层兼容性
2. **单激活向量编码可量化、可恢复的语义信息**：压缩信息可从激活向量中还原
3. **激活加权求和是鲁棒的表征压缩器**：比单层激活更稳定

### 对比传统方法
| 方法 | 压缩粒度 | 信息保留 | 计算开销 |
|------|------|------|------|
| 完整提示处理 | Token 级 | 100% | 高 |
| 提示压缩（传统） | Token 删减 | 部分 | 中 |
| **激活聚合压缩** | **向量级** | **>98%** | **低（固定提示）** |

## 实用性
- 适用于**固定指令提示**的高效复用（无需重复处理 Token 序列）
- 降低每次查询计算成本

## 局限性
- 仅验证了指令提示类型的压缩效果，对话/长文本场景未充分测试
- 学习加权求和需要额外训练阶段
