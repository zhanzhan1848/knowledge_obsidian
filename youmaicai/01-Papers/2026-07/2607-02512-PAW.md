# PAW: Program-as-Weights - 模糊函数编程范式

## 基本信息
| 标题 | Program-as-Weights: A Programming Paradigm for Fuzzy Functions |
|------|-----|
| **arXiv** | [2607.02512](https://arxiv.org/abs/2607.02512) |
| **作者** | Wentao Zhang, Liliana Hotsko, Woojeong Kim, Pengyu Nie, Stuart Shieber, Yuntian Deng |
| **类别** | cs.LG, cs.AI, cs.CL |
| **发表** | 2026-07-02 |

## 核心贡献

1. **Fuzzy-function programming**: 将自然语言规范编译成紧凑的、本地可执行的神经 artifact
2. **PAW (Program-as-Weights)**: 4B 编译器，从 10M 示例的 FuzzyBench 数据集训练
3. 为冻结的轻量级解释器发出参数高效适配器

## 技术亮点

### 性能对比
- **0.6B Qwen3 解释器执行 PAW 程序** ≈ 直接 prompting **Qwen3-32B** 的性能
- 推理内存减少约 **1/50**
- MacBook M3 上运行速度：**30 tokens/s**

### 范式转变
将基础模型从**per-input problem solver**转变为**tool builder**：
- 每个函数定义调用一次
- 产生小型可重用 artifact
- 后续函数调用**便宜且可离线**

## 应用场景

- 日志行重要警报
- 修复格式错误的 JSON
- 按意图排名搜索结果
