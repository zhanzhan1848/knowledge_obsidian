# Instruct-Particulate: Scaling Feed-Forward 3D Object Articulation with Kinematic Control

## 核心方法

输入 3D mesh + 目标运动学规格（part descriptions, connectivity, joint types, point prompts）→ 预测对应的运动学 part segmentation + joint motion parameters。

### 创新点
1. **异构数据集**：150,000+ articulated 3D objects
2. **VLM 自动标注**：使用 vision-language models 自动标注 kinematic labels
3. ** kinematic specification disambiguates**：解决不同粒度标注的兼容问题
4. **测试时可自动获取 kinematic spec**：从大规模 VLM 自动获取

### 技术栈
- 3D mesh 输入
- Kinematic specification (part descriptions, connectivity, joint types, optional point prompts)
- Vision-language models for automatic annotation
- Feed-forward neural network

## 算法复杂度
- 时间复杂度：O(n) per mesh，feed-forward
- 空间复杂度：O(V + E) for mesh + kinematic graph

## 实现难度
- 算法复杂度：中
- 数据依赖：大量标注的 articulated 3D 数据
- 依赖项：VLM (e.g., CLIP), 3D mesh processing

## 核心创新点
1. 首个 category-agnostic 3D 关节动画 feed-forward 方法
2. 超过 150,000 个 articulated 3D objects 的大规模异构数据集
3. 通过 kinematic specification 实现不同粒度标注的兼容
4. 可应用于 AI-generated meshes，泛化能力强
