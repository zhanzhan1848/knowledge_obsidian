# SketchFaceGS: 草图驱动 3D 高斯人脸生成与编辑

## 核心创新

首个 sketch-driven 框架，从 2D 草图实时生成和编辑照片级 3D 高斯人头模型。

### 技术路线

**前馈 + 粗到细架构**：

1. **Transformer-based UV 特征预测模块**
   - 从输入草图重建粗糙但几何一致的 UV 特征图

2. **3D UV 特征增强模块**
   - 补充高频照片级细节
   - 产生高保真 3D 人头

### 编辑能力

**UV Mask Fusion + 分层特征融合策略**：
- 精确、实时、自由视角修改

## 核心挑战与解决

2D 草图的问题：
- 稀疏
- 深度模糊
- 缺少高频外观线索

→ 高斯溅射几何一致的 3D 结构重建困难

## 实验结论

- 生成保真度：优于现有方法
- 编辑灵活性：优于现有方法
- 单次前向传播即可产生高质量可编辑 3D 人头

## 开源

## 算法分析

- **算法复杂度**：中等（Transformer + 3D 特征增强）
- **实时性**：前馈架构，适合实时
- **实现难度**：中等偏高
