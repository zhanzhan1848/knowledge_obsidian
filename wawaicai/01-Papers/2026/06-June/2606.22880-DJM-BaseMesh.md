# Compact Base Meshes for Displacement Mapping using Triangle Jacobians

## 核心方法

DJM (Displacement Jacobian Metric) 提出基于**位移函数 Jacobian** 引导底网格（base mesh）构建，实现比现有方法更好的精度-尺寸权衡。

### 问题定义

将复杂三角形网格转换为位移函数表示（定义在粗底网格上），需要：
1. 底网格三角形数量尽可能少
2. 位移函数能精确近似输入
3. 参数映射双射且低失真

### 关键创新

1. **Triangle Jacobian 评估**：用 Jacobian 衡量位移映射的失真度

2. **改进 QEM 简化**：约束底网格与输入的位移映射为双射且低失真（满足 Jacobian 下界）

3. **显式映射存储**：避免不可靠的射线-网格求交，全程存储输入到基底的映射

4. **鲁棒逆重心坐标求解器**：利用存储映射获得密集的基底-网格对应关系

## 算法复杂度

- 时间复杂度：O(n log n) 简化过程
- 空间复杂度：O(n) 额外映射存储

## 技术细节

### DJM 度量
- 衡量位移映射双射性和低失真性
- Jacobian 下界约束保证 bijectivity

### 与 QEM 对比
| 特性 | QEM | DJM |
|------|-----|-----|
| 误差度量 | 二次距离 | Jacobian 失真 |
| 约束 | 无 | bijectivity + 低失真 |
| 输出 | 简化网格 | 位移映射基底 |
