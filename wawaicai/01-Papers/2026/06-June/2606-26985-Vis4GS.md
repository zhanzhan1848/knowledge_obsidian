# Vis4GS: A Visual Analytic Tool for 3D Gaussian Splatting Reconstruction

## 核心方法
3D Gaussian Splatting (3DGS) 支持快速训练和实时渲染，但其优化过程难以解释。Vis4GS 提供了一个多视角可视化分析工具，用于对 3DGS 重建 artifacts 进行基元级诊断。

### 四个链接视图
1. **Interactive Gaussian Analysis View** - 交互式 Gaussian 分析
2. **Property Timeline View** - 属性时间线
3. **Gaussian Densification Tree View** - Gaussian 增殖树
4. **Log and Control Panel** - 日志和控制面板

### 核心功能
- Gaussian 选择
- 模糊和针状 artifact 评分
- View Coverage 分析
- 多尺度系谱探索 (clone, split, prune, clone-split events)

## 创新点
- 将场景级 artifacts 与基元级证据和优化历史连接
- 支持超越最终图像检查和全局指标的结构化诊断工作流
- 用户研究证明 Vis4GS 比原始 3DGS 查看器提供更强的可用性和 artifact 理解支持

## 复杂度分析
- 时间：O(n) per iteration for n Gaussians
- 空间：O(n) for storing Gaussian properties
