# VolHuMe: High-Resolution Large Scale Dataset of Volumetric Human Meshes

## 数据集概述

提出 **VolHuMe**，使用 64 个 RGB 和 32 个深度相机的先进体素工作室捕获的高质量 4D 人体扫描数据集。

### 数据集规模
- **104 个**独立受试者捕获
- 每个受试者提供丰富的 ground truth

### 数据类型
| 数据类型 | 描述 |
|---------|------|
| SMPL-X | 参数化人体模型 |
| 高分辨率网格 | 精细人体几何 |
| 多视角 RGB/深度图像 | 原始捕获数据 |
|  rigged meshes | 带骨骼绑定的网格 |
| 点云 | 3D 点云数据 |
|  garment segmentation | 服装分割 |
| 手部和面部几何 | 精细局部几何 |

### 捕获设置
- 短距离、高分辨率捕获设置
- 保留细粒度身体部位细节
- 提高几何保真度和纹理分辨率

## 基准测试

在 3D 和 4D 人体重建任务上对 SOTA 方法进行基准测试：
- 展示数据集质量
- 揭示当前评估测试平台的局限性

## 数据集价值

为人体网格重建、点云处理、参数化人体模型研究提供高质量数据支持。
