# Greed for the Spheres: A Signed Distance Interpolation Method

## 核心方法

### 问题定义
从离散采样点插值 Signed Distance Function (SDF) 数据。

### 关键创新
**保证插值结果与输入一致**：
- 新 SDF 数据值与输入完全一致
- 增强后的数据仍对应几何可实现表面

### 技术路线
1. **硬几何约束**：将 SDF 的理论性质表达为硬几何约束
2. **贪婪算法**：构造高效的贪婪算法进行一致性 SDF 插值
3. **GPU 加速**：并行化 GPU 预处理进一步加速

## 应用场景

### 1. 全局 SDF 细化
- 无需 ground truth 知识对 SDF 数据进行上采样

### 2. 网格重建
- 利用粗输入 SDF 的全局信息重建高度详细表面

### 3. Pseudo-SDF 修复
- 处理 CSG Boolean 操作产生的伪 SDF
- 转换为有效 SDF 以便下游处理

## 算法特性

- **一致性保证**：精化后的 SDF 与输入一致
- **无先前方法的一致性保证**

## 实现难度

- **算法复杂度**：中（贪婪算法 + GPU 并行化）
- **数值稳定性**：硬约束保证稳定性
- **依赖项**：GPU 并行计算

## 推荐结论

✅ **推荐实现** - 对网格重建和 SDF 修复有实用价值
