# Holo360D: 全景 3D 重建大规模真实数据集

## 核心贡献

**Holo360D** - 首个大规模真实世界全景数据集，包含：
- 109,495 个全景图
- 配准点云
- 网格
- 对齐相机姿态

### 数据采集

| 设备 | 说明 |
|------|------|
| 3D 激光扫描仪 | 集成 LiDAR、3 个针孔相机、IMU、RTK-GNSS |
| 360° 相机 | 刚性安装在扫描仪顶部 |
| 软件触发 | 扫描仪和相机同步 |

### 数据处理 Pipeline

```
Raw Data → Online SLAM → Offline SLAM → Surface Reconstruction → Post-Processing
```

## 后处理 Pipeline（三步）

| 步骤 | 方法 | 目的 |
|------|------|------|
| 数据去噪 | Remove isolated points | 去除孤立离群点 |
| 网格补全 | Fill glass/occluded regions | 填充玻璃和遮挡区域 |
| 区域特定重网格化 | Region-specific remeshing | 更好保留薄结构 |

## 数据集统计

- 录制时长：>19 小时
- 轨迹距离：>31 km
- 室外场景覆盖：>0.17 km²

## 应用场景

- 灾难恢复文档
- 保护分析
- 虚拟旅游
- 学术研究

## 可行性分析

**推荐度**: ⭐⭐⭐⭐⭐

- 超大规模真实数据，对训练和测试都极有价值
- 完整的后处理 pipeline 可作为参考实现
- 连续轨迹设计对动态重建很有帮助

**实现参考**：
- 网格去噪：参考 Point Cleanup → libigl `remove_duplicates` + outlier removal
- 网格补全：参考 Hole filling → libigl `fill_holes`
- 重网格化：区域特定 remeshing → libigl `remesh`
