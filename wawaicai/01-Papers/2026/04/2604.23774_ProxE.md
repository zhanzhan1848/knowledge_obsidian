# Prox-E: Fine-Grained 3D Shape Editing via Primitive-Based Abstractions

## 核心方法

**Prox-E** 是一个 training-free 框架，通过显式、基于基元的几何抽象实现细粒度 3D 控制。

### 核心流程

1. **几何抽象** - 将输入 3D shape 抽象为紧凑的基元集合（primitives）
2. **VLM 编辑** - 预训练视觉语言模型编辑这个抽象，指定基元级变化
3. **3D 生成引导** - 使用结构化编辑指导 3D 生成模型，实现局部修改同时保留物体整体 identity

## 技术创新点

| 创新 | 描述 |
|------|------|
| Training-free | 无需额外训练 |
| 显式几何抽象 | 基于基元的表示 |
| 细粒度控制 | 基元级编辑 |
| Identity 保存 | 保留未改变区域 |

## 实验评估

- 多个 existing approaches 对比
- 平衡 identity 保存、形状质量、指令保真度
- 优于 2D-based 3D editors 和 training-based 方法

## 可行性分析

**推荐度**: ⭐⭐⭐⭐

- Training-free 方案，易于集成
- 基元抽象与几何处理中的 primitive fitting 技术相关
- 可与 libigl 的 `fit_primitive` 功能结合

**关注点**：
- VLM 依赖部分需评估
- 基元类型选择影响编辑效果
