# LSNIF: Voxel Deformation-Aware Neural Intersection Function

## 核心方法

扩展 **LSNIF (Locally-Subdivided Neural Intersection Function)** 以支持参数化可变形和动画几何。

### 核心技术

| 技术 | 描述 |
|------|------|
| Rest-space / Deformed-space Formulation | 受 meshless rendering 启发 |
| Canonical Space Mapping | 将射线样本映射回规范空间 |
| Single Neural Network | 单一网络表示所有姿势的几何 |

### 关键技术

1. **Scale-invariant distance regression** - 尺度不变距离回归
2. **Uncertainty-weighted multi-task learning** - 不确定性加权多任务学习
3. **Hybrid positional-grid encoding** - 混合位置-网格编码

## 核心创新

- 保持 LSNIF 的紧凑性和效率
- 支持动态几何的鲁棒神经交点预测
- 无需重训练

## 相关研究

- 基于 LSNIF 的扩展
- Meshless rendering 技术结合
