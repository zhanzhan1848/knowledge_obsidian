# Cutscene Agent: LLM Agent Framework for Automated 3D Cutscene Generation

## 核心方法

**Cutscene Agent** - 用于自动化端到端 cutscene 生成的 LLM agent 框架。

### 三项贡献

| 贡献 | 说明 |
|------|------|
| Cutscene Toolkit (MCP) | LLM agents 与游戏引擎的双向集成 |
| Multi-agent System | 导演 agent + 专家 subagents（动画、摄影、声音） |
| CutsceneBench | 分层评估基准 |

### 关键技术

- **双向集成** - agents 不仅调用引擎操作，还持续观察实时场景状态
- **闭环生成** - 可编辑的引擎原生 cinematic assets
- **视觉推理反馈循环** - 感知驱动细化

## CutsceneBench 特点

- 长时域多步骤编排
- 数十个相互依赖的工具调用
- 严格排序约束
- 覆盖现有基准未涵盖的能力维度

## 可行性分析

**推荐度**: ⭐⭐

- 主要针对游戏叙事生成，与核心几何处理关联度较低
- 但 3D 资产生成和动画技术有参考价值
- MCP 协议可能有用于工具集成

**关注点**：
- 引擎原生资产的要求与网格质量相关
- 3D 资产生成的拓扑/UV 要求
