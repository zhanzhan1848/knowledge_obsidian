# LingBot-Map: Geometric Context Transformer for Streaming 3D Reconstruction

## 核心方法

**LingBot-Map** - 基于 Geometric Context Transformer (GCT) 架构的前馈 3D foundation model，用于从流数据重建场景。

### 核心设计

| 组件 | 功能 |
|------|------|
| Anchor Context | 坐标锚定 |
| Pose-Reference Window | 密集几何线索 |
| Trajectory Memory | 长程漂移校正 |

### 关键技术

- **注意力机制设计**：整合三类上下文
- 保持流状态紧凑同时保留丰富几何上下文
- **性能**：约 20 FPS (518×378 分辨率输入)，超过 10,000 帧的长序列

## 算法复杂度

- 时间：O(FPS × 序列长度)
- 空间：紧凑流状态设计

## 实验评估

- 多种 benchmarks 评估
- 优于现有 streaming 和 iterative optimization-based 方法
