# Unifying Real-Time Differentiable Ray Tracing and Rasterization

## 核心方法

提出一种**可微 3D 表示**，统一基于泡沫的光线追踪能力和现代光栅化 pipeline 的效率。

### 技术演进

| 方法 | 问题 |
|------|------|
| Prior Foam Representations | 常数时间射线遍历，但无界 cell 阻碍高效 tile-based 光栅化 |
| **Bounded Power Diagrams** | 广义 Voronoi 泡沫，支持可控 cell 范围 |

### 关键创新

| 创新 | 描述 |
|------|------|
| Bounded Power Diagrams | 无需训练时 Delaunay 三角剖分的空间有界原语 |
| Oriented Surface Formulation | 显式建模内部/外部区域间的接口 |
| Decoupled Geometry/Appearance | 在表面嵌入可微纹理 |

## 性能目标

- 保持最先进光线追踪效率
- 达到与当前 3DGS 竞争的光栅化性能
- 统一实时可微渲染的实际路径
