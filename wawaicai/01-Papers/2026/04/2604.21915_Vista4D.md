# Vista4D: Video Reshooting with 4D Point Clouds

## 核心方法

**Vista4D** - 鲁棒灵活的 video reshooting 框架，将输入视频和目标相机置于 4D 点云中。

### 任务

给定输入视频，从不同相机轨迹和视角重新合成场景，同时保持相同动态。

### 核心创新

| 创新 | 描述 |
|------|------|
| 4D-grounded 点云表示 | 结合静态像素分割和 4D 重建 |
| 显式保留 seen content | 明确保存可见内容 |
| 丰富相机信号 | 提供丰富的相机控制信号 |
| 多视图动态数据训练 | 对真实世界推断中点云伪影的鲁棒性 |

### 应用场景

- Dynamic scene expansion
- 4D scene recomposition

**推荐度**: ⭐⭐⭐
