# Mochi: Rethinking Collision Detection on GPU Ray Tracing Architecture

## 核心方法

**Mochi** - 在光线追踪架构上重新表述离散碰撞检测 (DCD)，利用碰撞关系的对称性。

### 关键创新

| 创新 | 描述 |
|------|------|
| Per-object Proxy Spheres | 解耦 BVH 包围体与碰撞搜索半径 |
| 显著收紧包围盒 | 不牺牲正确性 |
| 统一/非统一球体 | 高效支持 |

### 技术保证

- **Provably sound** - 可证明soundness
- **保证检测所有真实碰撞** - 无漏检

## 性能评估

- 大规模粒子工作负载
- 比 state-of-the-art BVH-based 和 RT-based DCD 实现一致加速
- 推广了 prior RT-based neighbor search 公式
