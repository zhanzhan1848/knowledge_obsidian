# Large-Scale Photogrammetric Documentation of St. John's Co-Cathedral

## 核心方法

圣约翰共教堂（马耳他瓦莱塔，UNESCO 世界遗产）的**大规模摄影测量文档**方法。

### 数据采集

| 数据源 | 规模 |
|--------|------|
| DSLR 相机 | 99,000 张图像 |
| 无人机摄影 | - |
| LIDAR 扫描 | - |

### 3D 重建结果

- **25-30 billion triangles** 的高度详细 3D 重建

### 特殊挑战

| 挑战 | 解决方案 |
|------|----------|
| 反射金属表面 | 混合摄影测量重建 |
| 深色材料 | AI 辅助去噪 |
| 复杂挂毯 | 低光图像分级 |
| 受限访问 | 仅夜间采集（7 晚）|

## Pipeline

```
Multi-modal Data Acquisition → Processing → Image Grading → AI-assisted Denoising → LIDAR Cleanup → Hybrid Photogrammetric Reconstruction (RealityCapture) → Mesh Subdivision → Real-time Visualization
```

## 关键工艺

| 工艺 | 说明 |
|------|------|
