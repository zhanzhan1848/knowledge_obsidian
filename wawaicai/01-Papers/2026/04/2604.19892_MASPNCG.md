# MAS-PNCG: Multilevel Preconditioned Nonlinear Conjugate Gradient for IPC

## 核心方法

**MAS-PNCG** - 释放非线性优化的层次预条件力量。

### 技术创新

| 创新 | 说明 |
|------|------|
| Sparse-Input Woodbury Update | 增量适配细级别 MAS 组件到快速演化的接触集 |
| 近零维护成本 | 捕获接触系统的复杂谱特性 |
| Hessian-aware 2D Subspace Minimization | 最佳结合预条件梯度与前方向 |

### 核心改进

- **避免 Hessian assembly** (PNCG 特性)
- **层次预条件** (原 MAS 问题：每非线性迭代重建成本高)
- **快速 per-subdomain Conservative CCD** - 确保无穿透轨迹

## 性能结果

- 比 GIPC 和 StiffGIPC（MAS 预条件）快 up to **5.66×** 和 **2.07×**
