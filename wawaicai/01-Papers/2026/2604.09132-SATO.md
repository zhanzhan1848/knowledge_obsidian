# Strips as Tokens (SATO): Artist Mesh Generation with Native UV Segmentation

## 核心方法

SATO 提出了一种基于**三角形条带（Triangle Strips）**的网格生成框架，将条带作为 Token 来生成艺术家级别的 3D 网格。

### 核心思想

- **条带式 Token 化**：将网格编码为连续的面片链，显式编码 UV 边界
- **统一表示**：同一 Token 序列可解码为三角形或四边形网格
- **联合训练**：大规模三角形数据提供结构先验，高质量四边形数据增强几何规则性

### 关键技术

1. **Triangle Strip 编码**：保持边缘流动性和结构规则性
2. **UV 边界显式建模**：原生支持 UV 分割
3. **自回归 Transformer**：生成高质量网格序列
