# A Persistent Homology Design Space for 3D Point Cloud Deep Learning

## 核心方法

提出将**持久同调（Persistent Homology, PH）**集成到 3D 点云深度学习中的统一设计框架。

### 核心思想

- **6 个原则性注入点**：
  1. 采样 (Sampling)
  2. 邻域图 (Neighborhood Graphs)
  3. 优化动力学 (Optimization Dynamics)
  4. 自监督 (Self-Supervision)
  5. 输出校准 (Output Calibration)
  6. 内部网络正则化 (Internal Network Regularization)

### 关键技术

1. **Persistence Diagrams/Landscapes/Images**：三种 PH 表示形式
2. **统一设计空间**：3DPHDL 框架
3. **实验验证**：ModelNet40 分类、ShapeNetPart 分割

## 算法复杂度

- 时间复杂度：O(n log n) 复杂系统构建
- 空间复杂度：取决于filtration和表示

## 实现难度

- 算法复杂度：中高
- 数值稳定性：PH 计算相对稳定
- 依赖项：GUDHI, DIPHA, Hera 等 PH 库

## 开源参考

- libigl: 无直接对应（PH 集成需要专用库）
- CGAL: `CGAL::Persistent_homology` 可参考
