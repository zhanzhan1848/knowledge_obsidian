# MeshWeaver

## 核心方法

**自回归网格生成框架**，将网格生成视为表面编织过程，直接预测下一个顶点而非独立坐标：
- **多层次稀疏体素编码器**：以三种方式注入几何上下文
  1. 提供体素特征作为顶点表示
  2. 通过交叉注意力引导 token 预测
  3. 作为结构支架约束表面周围生成
- **层次设计**：单步解码中实现粗到细顶点预测

## 创新点

1. **高 token 效率**：避免长 token 序列，可扩展至高多边形网格
2. **几何感知引导**：条件不仅基于全局形状嵌入，还基于局部表面线索
3. **SOTA 压缩比**：18% 压缩率
4. **高面数生成**：可生成最高 16K 面网格

## 技术细节

- 自回归解码器：遍历输入网格连通性，从当前 1-ring 邻域选择下一顶点
- 局部约束避免无效跳跃穿越表面
