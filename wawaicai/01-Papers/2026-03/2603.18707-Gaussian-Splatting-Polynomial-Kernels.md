# Gaussian Splatting with Polynomial Kernels

## 核心方法

### 问题定义
- **目标**：改进3D Gaussian Splatting的计算效率
- **挑战**：新核函数与现有数据集不兼容

### 核心创新
1. **多项式核函数**
   - 用多项式近似替代原始指数核
   - 结合ReLU激活函数
   - 保持与现有数据集的兼容性

2. **激进剔除策略**
   - 利用多项式核特性
   - 更高效地剔除不可见高斯
   - 减少渲染计算量

### 数学分析

**原始核函数**（指数）：
```
G(x) = exp(-||x||² / (2σ²))
```

**多项式近似**：
```
P(x) = max(0, a_n||x||^n + ... + a_1||x|| + a_0)
```

其中：
- a_i 为多项式系数
- ReLU保证非负性
- n 为多项式阶数

**优势**：
1. 计算更快（避免exp函数）
2. 支持更激进的culling
3. 适合NPU硬件加速

### 算法改进

```
输入：3D高斯集合，相机参数
1. 投影高斯到屏幕空间
2. 使用多项式核评估贡献
3. ReLU激活剔除负值区域
4. Alpha blending合成图像
输出：渲染图像
```

## 技术细节

### 性能提升
- **速度提升**：4-15%
- **图像质量**：几乎无影响
- **兼容性**：保持与现有数据集兼容

### NPU优化
- 多项式计算适合NPU
- 避免指数函数的硬件开销
- 支持向量化运算

### 剔除优化
**原始方法**：
- 基于高斯3σ范围剔除
- 保守策略，计算冗余

**多项式方法**：
- ReLU提供明确边界
- 更精确的可见性判断
- 减少不必要的计算

## 复杂度分析

- **核函数计算**：
  - 指数核：O(1) 但常数大（exp函数）
  - 多项式核：O(n) n为多项式阶数，但常数小

- **渲染复杂度**：
  - 原始：O(k) k为可见高斯数
  - 改进：O(k') k' < k（更激进剔除）

- **实际性能**：4-15% 提升

## 实现难度

- **算法复杂度**：低
  - 核函数替换
  - 无架构修改

- **数值稳定性**：良好
  - 多项式拟合需调参
  - ReLU保证非负性

- **依赖项**：标准3DGS实现

## 推荐结论

✅ **推荐实现**

**理由**：
1. 简单改进，显著收益
2. 无需重新训练数据
3. 适合NPU硬件
4. 即插即用

**注意事项**：
- 多项式系数需要拟合
- 需要平衡精度和速度
- 不同场景可能需要调参
