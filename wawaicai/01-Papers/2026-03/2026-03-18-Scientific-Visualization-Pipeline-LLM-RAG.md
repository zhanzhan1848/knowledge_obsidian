# Toward Reliable Scientific Visualization Pipeline Construction with Structure-Aware Retrieval-Augmented LLMs

## 核心方法

**结构感知检索增强生成工作流**：生成可执行的科学可视化管线
- 针对 vtk.js（Web 可视化库）
- 支持模块选择、参数配置、执行顺序
- 引入 "修正成本" (Correction Cost) 指标

## 技术细节

**挑战**：
- 科学可视化管线有严格的执行依赖
- LLM 生成容易缺失阶段、错误使用算子、顺序不当
- Web 环境需要显式代码级管线组装

**解决方案**：
```
自然语言描述 → 结构感知 RAG → vtk.js 代码示例 → 可视化管线
```

**关键组件**：
1. **管线对齐的代码示例**: 提供上下文指导
2. **结构化领域特定上下文**: 提升可执行性
3. **交互式分析界面**: 支持人在回路检验

## 复杂度分析

- **时间复杂度**: LLM 推理 + 检索
- **空间复杂度**: 代码示例库
- **评估指标**: 
  - 管线可执行性
  - 修正成本（手动干预量）
