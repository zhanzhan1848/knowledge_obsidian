# LoST: Level-of-Semantics Tokenization for 3D Shapes

## 核心方法

**语义层级标记化 (LoST)**：按语义显著性排序标记
- 早期前缀解码为完整、合理的形状（主要语义）
- 后续标记细化实例特定的几何和语义细节

**创新点**：
1. 超越传统几何 LoD 层次（原本为渲染和压缩设计）
2. 提高自回归 (AR) 模型的 token 效率
3. 引入 **RIDA (Relational Inter-Distance Alignment)** 损失

## 技术细节

**RIDA 损失函数**：
- 对齐 3D 形状潜在空间的关系结构
- 与语义 DINO 特征空间对齐
- 实现语义一致性

**标记化策略**：
```
传统 LoD: 空间层次 → Token 低效，缺乏语义连贯性
LoST: 语义显著性 → Token 高效，语义连贯
```

**性能指标**：
- 几何重建：SOTA
- 语义重建：SOTA
- Token 数量：仅为传统 AR 模型的 0.1%-10%

## 复杂度分析

- **时间复杂度**: O(n log n) - 语义排序
- **空间复杂度**: 大幅降低（Token 数量减少）
- **推理速度**: 显著提升
