# OT-UVGS: UV Mapping for Gaussian Splatting as a Capacity Allocation Problem

## 核心方法

将 UV 参数化高斯溅射（UVGS）中的 UV 映射重新解释为**容量分配问题**，提出基于一维最优传输的轻量级映射方法。

### 问题背景
- UVGS 将非结构化 3D 高斯映射到规则 UV 张量，实现紧凑存储
- 传统 UVGS 使用确定性球面投影分配高斯到 UV 位置
- 问题：忽略全局高斯分布，导致 UV 槽空置和密集区域碰撞

### 核心创新
- **OT-UVGS**: 一维最优传输启发的映射方法
- 全局耦合分配，同时保持原始 UVGS 表示
- 实现：基于排序的排名算法，O(N log N) 复杂度

### 算法流程
1. 将高斯按一维最优传输原则分配到 UV 位置
2. 使用 rank-based sorting
3. 可作为球面 UVGS 的直接替代

## 算法复杂度
- 时间复杂度：O(N log N)，N 为高斯数量
- 空间复杂度：O(N)

## 实现难度
- 算法复杂度：低
- 数值稳定性：稳定，最优传输保证全局最优
- 依赖项：无特殊依赖
