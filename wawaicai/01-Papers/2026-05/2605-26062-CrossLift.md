# CrossLift: Lifting Cross Fields From 2D Visual Priors

## 核心方法

使用文本到图像的生成模型作为 2D 视觉先验，从合成图像中提取与特征对齐的交叉场方向，然后将其反向投影到网格表面。采用两阶段光滑插值：先在每个视角内插值，再跨视角聚合。使用自定义置信度权重解决同一面上候选方向冲突问题。

## 算法流程
1. 使用 text-to-image priors 合成特征对齐的 quad mesh 图像
2. 从 2D 图像中提取每像素方向作为候选方向
3. 将 2D 方向反投影到网格表面
4. 两阶段光滑插值聚合候选方向
5. 生成对齐的四角网格

## 应用场景
- 纹理对齐的四角网格生成
- 交互式交叉场设计（用户绘制粗略线条作为信号）
- 语义对齐的四角网格生成
