# R-DMesh: Video-Guided 3D Animation via Rectified Dynamic Mesh Flow

## 核心方法

本文提出 **Rectified Dynamic Mesh (R-DMesh)**，一个统一框架，通过视频引导生成高保真4D网格，解决"姿态不对齐"问题。

### 问题定义

实际应用中，用户提供的静态网格初始姿态很少与参考视频的起始帧对齐。强制让网格跟随不匹配的轨迹会导致严重的几何变形或动画失败。

### 核心创新点

1. **姿态 rectification**：将任意姿态的输入网格自动对齐到视频初始帧，再进行动画
2. **三流注意力机制 (Triflow Attention)**：利用顶点级几何特征调制三个正交流
3. **Rectified Flow 扩散变换器**：条件于预训练视频潜变量，有效迁移时空先验到3D领域

### 关键技术架构

```
输入静态网格 → VAE 解耦
     ├─ 条件基础网格 (conditional base mesh)
     ├─ 相对运动轨迹 (relative motion trajectories)
     └─ rectification jump offset（关键：自动对齐到视频初始姿态）
     
三流注意力机制 (Triflow Attention)
     → 顶点级几何特征调制
     → 确保物理一致性和局部刚性
     
Rectified Flow 扩散变换器（条件于预训练视频潜变量）
     → 迁移丰富的时空先验到3D领域
```

## 数据集

**Video-RDMesh**：50万+动态网格序列，专门模拟姿态不对齐场景。

## 实验结果

1. 有效解决姿态不对齐问题
2. 支持姿态重定向 (pose retargeting)
3. 支持整体4D生成 (holistic 4D generation)
