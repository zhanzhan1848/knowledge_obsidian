# PointLLM-R: Enhancing 3D Point Cloud Reasoning via Chain-of-Thought

## 核心方法

数据驱动框架，为 3D 点云理解构建大规模 CoT 监督。

## 两阶段 pipeline

1. **质量评估与细化**: 通过视觉-语言模型 (VLM) 评估和参考引导细化点-文本指令数据
2. **HiLPO**: 人类参与的提示优化 (Human-in-the-Loop Prompt Optimization) 合成高质量推理路径

## 数据集

**PoCoTI**: CoT增强的点-文本指令跟随数据集
- 55K 样本
- 包含显式推理路径

## 模型

Fine-tuning PointLLM → **PointLLM-R**
- 推理能力3D多模态语言模型

## 实验验证

- 生成式3D分类
- 描述任务
- 泛化到真实扫描点云和多轮对话
