# SGP 2026 - Symposium on Geometry Processing 程序概要

## 会议信息

## Keynotes

### Geometric not-so-deep learning
- **讲者**: Julie Digne (LIRIS - CNRS - Université Claude Bernard Lyon 1)
- **核心**: 轻量级方法、计算高效、直接在单个形状上操作、利用统计几何特征

### Computational Optimal Transport: From Low to High Dimension and Back
- **讲者**: Justin Solomon (MIT CSAIL)
- **核心**: 最优传输在几何处理中的应用、从低维到高维的转变

### Resource-Efficient Visual Computing
- **讲者**: Bernhard Kerbl (University of Copenhagen / TU Wien)
- **核心**: 实时、資源高效视觉计算、AI 在视觉计算中的应用

## Graduate School 课程

### Shape Spaces
- 讲者：Josua Sassen, Florine Hartwig
- 主题：Riemannian shape spaces 在几何处理中的应用

### Geometry Processing from 2D Image Priors
- 讲者：Dale Decatur, Richard Liu, Nam Anh Dinh
- 主题：从 2D 图像先验进行 3D 几何处理
- **重要**: 2D foundation models 到 3D 几何处理的迁移

### Cone-Nets: Theory and Interactive Design
- 讲者：Klara Mundilova, Michele Vidulis
- 主题：锥面网理论及交互设计工具

### Differentiable Geometry Processing in Python
- 讲者：Ana Dodik, Ahmed Mahmoud
- **重要**: 可微分几何处理 Python 实现

## 重要趋势观察

1. **不确定性感知几何处理**：Gaussian Process Implicit Surfaces (GPIS) 成为热点
2. **轻量级/高效方法**：不完全依赖深度学习大数据集
3. **2D 到 3D 迁移**：利用 2D 图像先验进行几何处理
4. **可微分几何处理**：Python 实现成为教学和实践重点
5. **最优传输**：几何处理与机器学习的交叉点
