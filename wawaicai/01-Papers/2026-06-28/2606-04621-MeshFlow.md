# MeshFlow: Efficient Artistic Mesh Generation via MeshVAE and Flow-based Diffusion Transformer

## 核心方法

### MeshVAE + Rectified Flow Transformer

1. **连续-离散统一表示**
   - 使用 VAE 将连续顶点位置和离散连通性编码到统一连续潜空间
   - 对比损失 (contrastive loss) 监督
   - 潜空间比 token-based 方法更紧凑

2. **3D Rectified Flow Transformer 生成器**
   - 并行生成所有网格顶点和边
   - 替代自回归 (AR) 方法的逐 token 预测
   - 避免二次复杂度

3. **关键优势**
   - 比最快 AR 生成器快 **18x**
   - 避免顶点坐标量化误差
   - 支持高达 16K 面的高多边形网格
