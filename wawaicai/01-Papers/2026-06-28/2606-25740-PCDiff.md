# PCDiff: Point Cloud Diffusion with Global and Local Reconstruction for Instance-Level 3D Anomaly Detection

## 核心方法

### 问题背景
3D 异常检测在精密工业制造中至关重要，现有重建方法面临两大挑战：
1. 前景弱缺陷区域（如划痕）难以重建和检测，偏差可小至 **10⁻³**
2. 背景非缺陷区域易产生位置偏差，导致误报

### PCDiff 框架

**生成阶段 (Generation Phase)**：
- 实例级多模态注意力嵌入生成框架
- 异常以纹理梯度、图像块、文本、mask 为条件
- 实现弱缺陷异常的高质量生成

**检测阶段 (Detection Phase)**：
- 联合局部-全局重建算法
- 局部异常恢复 + 全局几何一致性
- 保留背景正常结构的同时恢复前景缺陷

## 算法复杂度

- 点云精度要求：10⁻³ 级别
- 涉及：Diffusion Model + 多模态注意力 + 联合重建

## 创新点

- 首次将扩散模型应用于点云实例级异常检测
- 实例级多模态条件生成弱缺陷
- 联合局部-全局重建平衡精度

## 可行性分析

| 维度 | 评估 |
|------|------|
| 算法复杂度 | 高（Diffusion + 多模态 + 点云处理） |
| 数值稳定性 | 需精细调参（10⁻³ 级别精度要求） |
| 依赖项 | 点云处理库 (Open3D, PCC) |
| 推荐度 | ⭐⭐⭐⭐ 工业应用价值高，扩散+几何结合 |
