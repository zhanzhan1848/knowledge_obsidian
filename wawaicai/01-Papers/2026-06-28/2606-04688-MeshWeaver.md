# MeshWeaver: Sparse-Voxel-Guided Surface Weaving for Autoregressive Mesh Generation

## 核心方法

### 自回归网格生成的核心问题
1. **低 Token 化效率** → 长 token 序列，无法扩展到高多边形网格
2. **缺乏几何感知引导** → 仅依赖全局形状嵌入，缺少局部表面线索

### MeshWeaver 解决方案
将网格生成视为**表面编织 (Surface Weaving)** 过程，直接预测下一顶点而非独立坐标。

**多层次稀疏体素编码器** (Multi-level Sparse-voxel Encoder)：
1. 提供体素特征作为顶点表示
2. 通过交叉注意力引导 token 预测
3. 作为结构支架约束生成

**层次化设计**：
- 粗到细的顶点预测（单步解码）
- 生成模型与 3D 几何紧密耦合
- 最高支持 **16K 面**网格
- SOTA 压缩率 **18%**
