## 核心贡献

> 将空间和时间自适应集成到流体上同调框架，解决非单连通曲面上不可压缩欧拉方程的数值不稳定和调和分量丢失问题

1. **自适应流体上同调**: 动态空间细化 + 时间步进集成到流体模拟
2. **后验误差估计**: 自适应调整空间分辨率
3. **拓扑感知谐波基插值**: 在网格变异期间稳健传递调和形式，保持流场同调类
4. **内存降低 86%**: 与全局精细网格相比，同时保持高精度

---

## 技术方案

### 核心思想

非单连通曲面上的流体模拟需要处理流场的局部和全局行为。传统方法：
- 涡量-流函数 formulations 适合欧几里得域
- 调和分量曾被视为静态，但 Yin et al. 2023 发现其具有关键动力学

**挑战**:
- 高旋度/快速速度变化区域需要高分辨率
- 网格变异时 naive 插值导致数值不稳定和同调类丢失

**解决方案**:
```
1. 空间自适应: intrinsic triangulations + AFEM 后验误差估计
2. 时间自适应: Dormand-Prince 5(4) Runge-Kutta
3. 拓扑感知插值: 同调理论保证谐波基在网格变异后保持同调类
```

### 关键技术

| 技术 | 说明 |
|------|------|
| Fluid Cohomology | 处理非单连通流形的全局拓扑分量 |
| 固有三角剖分 | 网格细化的几何基础 |
| 后验误差估计 | 自适应确定细化区域 |
| Dormand-Prince 5(4) | 自适应时间步进 |
| 拓扑感知插值 | 同调类保持的谐波基迁移 |
| L²-投影对比 | naive 方法失效，拓扑感知方法更稳定 |

### 公式

**欧拉方程** (二维不可压缩):
$$\nabla \cdot \mathbf{u} = 0$$
$$\frac{\partial \mathbf{u}}{\partial t} + (\mathbf{u} \cdot \nabla)\mathbf{u} = -\nabla p$$

**调和分量** $H$: 定义在流形上满足 $\Delta H = 0$ 的速度分量

## 实验结论

- **数据集**: torus 等非单连通曲面上的流体模拟
- **指标**: 内存占用、计算时间、数值稳定性
- **结果**: 
  - 内存降低 86% (vs 全局精细网格)
  - 静态方法失败的低质量网格上保持稳定
  - 准确重现高分辨率模拟的动力学

## 局限性

- 当前评估仅聚焦 2D 曲面流
- 3D 扩展理论给出但未充分验证
- 依赖良好法向量和网格质量
