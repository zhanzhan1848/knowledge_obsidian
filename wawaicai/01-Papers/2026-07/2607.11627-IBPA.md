## 核心贡献

> 将传统 Ball Pivoting Algorithm (BPA) 改造为增量式、实时、自由形态的流形网格重建方法，支持孔洞检测

1. **增量 BPA (IBPA)**: 从流式点云数据逐步构建可定向流形 mesh，无需假设点云重叠或空间分布
2. **自由形态表面**: 支持悬垂、垂直墙等复杂拓扑，不受 DTM/DSM 高度场限制
3. **孔洞检测机制**: 实时识别并高亮不完全网格区域，为 ROV/AUV 提供可操作的反馈
4. **流形保证**: 同时满足顶点流形和边流形性质

---

## 技术方案

### 核心思想

传统 BPA 是非增量的，不保证流形性，且假设完整点云输入。IBPA 将 BPA 改造为增量处理模式：

1. **增量球滚动**: 点云逐帧/逐块输入时，维护活跃 ball pool
2. **流形网格构建**: 追踪边-面邻接关系，确保每个顶点度和边唯一性
3. **孔洞检测**: 通过面片边界追踪识别未覆盖区域
4. **方向一致性**: 基于法向量传播保证网格可定向性

### 关键技术

| 技术 | 说明 |
|------|------|
| Ball Pivoting Algorithm | 基于球半径滚动构建三角形面片 |
| 流形约束 | 顶点度和边唯一性检查 |
| 增量重建 | 流式点云处理，支持动态添加点 |
| 孔洞检测 | 边界边追踪 + 探索 fronts |

### 算法流程

```
输入: 点云流 + 法向量 + 球半径集合
输出: 流形三角形网格 + 孔洞区域

1. 初始化空网格和活跃 ball pool
2. for each 点云帧:
   a. 添加新点到 kd-tree
   b. 更新 ball pool
   c. 执行 ball pivoting 生成新面片
   d. 应用流形约束验证
   e. 追踪孔洞边界
   f. 输出当前网格状态
```

## 实验结论

- **应用场景**: 水下地形重建 (ROV/AUV 测深数据)
- **数据集**: 包含沉船 (aircraft) 的水下声学点云
- **对比**: 优于 DTM/DSM 方法，可处理悬垂结构
- **实时性**: 增量处理支持实时可视化

---

## 局限性

- 依赖法向量输入 (需提前计算或从传感器获取)
- 球半径需要人工设置或自适应估计
- 对噪声点云的鲁棒性有限
