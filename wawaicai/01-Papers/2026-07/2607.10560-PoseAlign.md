## 核心贡献

> 提出两阶段文本引导网格变形框架，通过 Laplacian 微分表示和注意力共享 SDS Loss 实现姿态保持和文本对齐

1. **两阶段变形**: 全局姿态缩放 + 局部细节雕刻
2. **Laplacian 表示**: 替代 Jacobian 的更平滑微分表示，用于全局变形
3. **Pose-aligned SDS Loss**: 引入注意力共享机制的 SDS 损失，保持语义对应
4. **CGI 2026 Best Paper**: 获得会议最佳论文奖

---

## 技术方案

### 核心思想

现有文本引导变形方法存在两个问题：
- **Jacobian 独立性**: 相邻面片的 Jacobians 独立计算，导致变形不连贯
- **视觉先验偏差**: 扩散模型倾向于覆盖原始姿态以匹配常见姿态

**两阶段方案**:

```
Stage 1 - 全局姿态缩放:
  CLIP Loss + Laplacian 正则 → 初步对齐文本语义

Stage 2 - 局部细节雕刻:
  Pose-aligned SDS Loss + Attention Sharing → 精细细节 + 姿态保持
```

### 关键技术

| 技术 | 说明 |
|------|------|
| Laplacian 表示 | 微分坐标，比 Jacobian 更平滑，适合大尺度变形 |
| CLIP Loss | 计算高效，适合全局结构先验编码 |
| SDS Loss | Score Distillation Sampling，扩散模型监督 |
| Attention Sharing | 保持输入输出之间的语义对应，preserving pose |
| Per-face Jacobian | 局部细节变形后的几何表示 |

### Loss 函数对比

| Loss | 效率 | 平滑性 | 姿态保持 |
|------|------|--------|----------|
| CLIP | 高 | 差 | 中 |
| SDS | 低 | 好 | 差 (without attention sharing) |
| Pose-aligned SDS | 低 | 好 | 好 |

## 实验结论

- **数据集**: 多类别 3D 网格 (动物、人物等)
- **对比**: TextDeformer, MeshUp
- **结果**: 同时实现更好的文本对齐、更精确的姿态保持、更少伪影
- **用户研究**: 定性评估显示显著优势

---

## 局限性

- 依赖预训练扩散模型，计算成本较高
- 需要初始网格作为输入
- 复杂姿态变形效果可能受限
