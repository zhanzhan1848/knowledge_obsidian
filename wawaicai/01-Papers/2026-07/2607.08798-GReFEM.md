## 核心贡献

> 探索多模态 LLM 作为零样本语义助手，基于物理引导的文本提示定位应力关键区域，指导有限元网格细化

1. **GReFEM 框架**: 利用 MLLM 视觉定位应力关键区域作为网格细化锚点
2. **orthoViews 模块**: 视图选择模块最大化关键几何特征的可观测性
3. **零样本物理 grounding**: MLLM 展示强大的零样本能力，精确遵循复杂空间-物理指令
4. **精确度优于盲 heuristic**: 在匹配细化预算下，比几何 heuristic 更高 precision

---

## 技术方案

### 核心思想

传统自适应体积网格生成依赖 PDE 求解器计算误差指示器，耦合求解器和计算成本。GReFEM 探索：**能否利用 MLLM 的高级语义理解作为有限元网格细化的零样本几何代理？**

**关键挑战**:
- MLLM 预训练于 2D，难以直接处理 3D 几何
- 像素级定位不可靠 (图 2a)
- 需要桥接 2D MLLM 推理和 3D 网格细化

**Pipeline**:
```
输入: 3D CAD 模型 + 物理引导文本提示 (e.g., "应力集中区域")

1. orthoViews 视图选择:
   - 生成多视角正交投影
   - 选择最大化几何特征可观测性的视图组合

2. 区域提议 + 特征检测:
   - MLLM 在 2D 视图上定位关键区域 (网格单元)
   - 网格化定位比像素级更稳定

3. 表面投影 (2D→3D):
   - 将 2D 区域提议反投影到 3D 表面
   - 生成网格细化锚点

4. 体积网格生成:
   - 基于锚点引导的体积细化
```

### 关键技术

| 技术 | 说明 |
|------|------|
| MLLM 零样本 grounding | GPT-4V 等模型直接理解几何+物理指令 |
| orthoViews | 正交视图选择，最大化特征可见性 |
| 网格化区域定位 | 比像素定位更稳定 |
| 物理引导提示 | "应力集中"、"几何特征" 等语义指令 |
| 2D→3D 投影 | 表面反投影锚点生成 |

## 实验结论

- **数据集**: 多样化 CAD 几何、不同加载条件、SOTA MLLMs
- **对比基线**: 调优的几何 heuristic
- **结果**:
  - MLLM 在 precision 上显著优于盲 heuristic
  - 能够隔离应力关键特征以节省细化预算
  - 同时识别成功和当前局限，为未来工作定义边界

---

## 局限性

- 依赖正交视图，复杂 3D 几何可能丢失信息
- 网格化定位精度有限
- 纯几何 heuristic 在某些场景仍有优势
- 评估基于 CAD 模型，未涵盖原始扫描数据
