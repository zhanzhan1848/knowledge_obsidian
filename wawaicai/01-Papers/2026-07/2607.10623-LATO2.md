## 核心贡献

> 将网格生成分解为顶点流 (V-Flow) 和连通性流 (T-Flow) 两个独立阶段，解决传统联合建模导致的顶点漂移和表面破损问题

1. **因子分解流匹配**: 分离顶点几何和连通性为两个独立流匹配阶段
2. **共享 voxel scaffold**: 两个阶段共享粗粒度体素骨架，保证几何-拓扑一致性
3. **分块生成**: 支持将 scaffold 分区，各部分在完整潜在容量下合成，提高分辨率
4. **拓扑自适应编辑**: 操作第一阶段顶点自动诱导对应连通性，无需重优化

---

## 技术方案

### 核心思想

传统方法将顶点和连通性联合编码在单一潜空间，导致连续几何与离散拓扑相互干扰。LATO.2 的关键洞察是：**一旦因子分解，连通性可以在已知顶点条件下表达为连续概率**。

**两阶段架构**:

```
Stage 1 - V-Flow (顶点流):
  Voxel scaffold → VAE 压缩顶点位置 → 潜空间流匹配 → 子像素精度顶点

Stage 2 - T-Flow (连通性流):
  已知顶点 → 构建逐顶点特征 → 潜空间流匹配 → 成对边概率 → 网格连通性
```

### 关键技术

| 技术 | 说明 |
|------|------|
| Flow Matching | 连续正则化流生成模型，比 diffusion 更快速 |
| 因子分解 | 顶点流 (连续) + 连通性流 (条件连续) |
| Voxel Scaffold VAE | 将网格压缩到共享体素网格表示 |
| 逐顶点特征 | 连接边概率解码的连续潜变量 |
| Part-wise Generation | 分区生成支持更高分辨率 |

### 公式

**顶点流**: 学习从噪声分布到顶点位置分布的映射
**连通性流**: 给定顶点 $V$，预测边概率 $p(e_{ij}|v_i, v_j)$

## 实验结论

- **对比基线**: PolyDiff, MeshCraft, MeshFlow, LATO
- **指标**: 几何保真度 (Chamfer Distance), 连通性质量 (面片数、边流形率)
- **结果**: SOTA 拓扑感知网格生成，在几何精度和连通性质量上均优于先前方法

---

## 局限性

- 依赖粗粒度 voxel scaffold，分辨率受其限制
- 分块生成需要预先知道部件划分
- 当前仅验证于合成数据
