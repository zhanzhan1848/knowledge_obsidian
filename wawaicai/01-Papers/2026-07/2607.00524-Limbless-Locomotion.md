# Geometric Shape Optimization for Limbless Locomotion

## 核心方法

提出**微分几何框架**用于模拟**细长软体生物**（如蛇）的运动。

### 核心方法
1. **曲线表示**: 使用 **Fourier-Chebyshev 多项式基**将身体表示为 3D 参数曲线
2. **优化求解**: 求解优化问题确定曲线与环境交互，估算多项式系数
3. **能量正则化**: 弯曲和扭转能量项确保物理合理性和非自交

### 运动仿真流程
```
参数曲线 → 优化求解 → 曲面驱动 → 可视化
```

## 技术创新
- **Fourier-Chebyshev 多项式**: 高效曲线表示
- **微分几何能量**: 物理约束
- **无自交保证**: 防止穿模

## 应用场景
- 计算机图形学 (动画/游戏)
- 软体机器人
- 计算建模
