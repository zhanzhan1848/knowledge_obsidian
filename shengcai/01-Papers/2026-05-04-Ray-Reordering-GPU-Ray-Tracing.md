# On Ray Reordering Techniques for Faster GPU Ray Tracing

## 基本信息

| 属性 | 值 |
|------|-----|
| 作者 | Jakub Boksansky 等 |
| 发表 | arXiv cs.GR |
| 年份 | 2025 |
| 链接 | [arXiv:2506.11273](https://arxiv.org/abs/2506.11273) |

## 核心贡献

1. **光线重排序研究**：系统研究射线重排序作为提升现有 GPU 光线追踪实现性能的工具
2. **终止点估计方法**：提出一种使用终止点估计（termination point estimation）的新型修改方法，特别适合追踪次级射线
3. **性能提升数据**：在最新 GPU 上光线重排序可实现 **1.3-2.0x** 的追踪速度提升，但恢复重排序开销在硬件加速追踪阶段存在困难

## 技术方案

### 核心思想
研究完全依赖于特定追踪内核的光线重排序方法，总结现有计算光线排序键的方法并讨论其特性。

### 关键发现
- 光线重排序在最新 GPU 上可显著提升追踪速度
- 在 RTX 追踪内核的波前路径追踪场景下验证
- 在硬件加速追踪阶段恢复重排序开销存在问题

## 局限性

- 主要聚焦波前路径追踪（wavefront path tracing）
- 在硬件加速追踪阶段开销回收困难
