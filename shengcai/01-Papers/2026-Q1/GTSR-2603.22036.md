## 核心贡献

> 首个基于 3DGS 管道重建半透明物体表面几何的方法

1. **双高斯集合**：表面高斯 + 内部高斯，分别建模表面和散射颜色
2. **Fresnel 混合渲染**：使用 Fresnel 项混合两组高斯渲染半透明物体外观
3. **Disney BSDF + Deferred Rendering**：增强法线和深度约束，改善非轮廓区域重建细节
4. **实时渲染性能**：在 NeuralTO Syn 数据集上优于基线重建方法

---

## 技术方案

### 核心思想

从多视角图像重建半透明物体是难题。现有方法（可微分路径追踪、神经隐式场）计算成本高。3DGS 管道对不透明物体效果好，但对半透明物体困难，因为未考虑光学特性。GTSR 通过双高斯集建模表面和内部散射。

### 关键技术

| 技术 | 说明 |
|------|------|
| 表面高斯 | 建模半透明物体表面 |
| 内部高斯 | 建模光通过半透明物体时的散射颜色 |
| Fresnel 项 | 混合两组高斯 |
| Disney BSDF | 增强法线和深度约束 |
| Deferred Rendering | 在 3DGS 管道中集成 PBR |

### 渲染公式

```
最终渲染 = Fresnel(surface_normal, view_direction) × surface_gaussian 
         + (1 - Fresnel(...)) × interior_gaussian
```

## 实验结论

- **数据集**：NeuralTO Syn dataset + 新增半透明物体
- **基线**：现有重建方法
- **结果**：在 NeuralTO Syn 数据集上优于基线
- **性能**：实时渲染性能

---

## 局限性

- 需要预先知道半透明度
- 对复杂材质适应性有限
- Disney BSDF 计算开销

## 实现建议

- **实现难度**：中
- **预期性能**：实时渲染
- **适用场景**：半透明物体重建、医疗成像、玉石/蜡质材质建模
