## 核心贡献

> 首个完全无姿态的事件流 3DGS 框架，仅从事件流重建 3D 场景

1. **边缘引导结构感知**：利用事件流中的边缘信息提供丰富的结构线索
2. **无姿态框架**：无需相机姿态输入，从事件流完全重建
3. **基于 patch 的时间相干分析**：提取边缘同时抑制噪声
4. **边缘加权损失**：在初始化、跟踪和束调整全程使用

---

## 技术方案

### 核心思想

NeRF 和 3DGS 需要高质量 RGB 输入和精确姿态，限制了在快速相机运动或不良光照条件下的鲁棒性。事件相机以高时间分辨率和宽动态范围捕获亮度变化。E2EGS 利用边缘信息实现无姿态 3D 重建。

### 关键技术

| 技术 | 说明 |
|------|------|
| 事件相机 | 捕获每像素亮度变化，高时间分辨率，宽动态范围 |
| 边缘提取 | 利用边缘/非边缘区域的不同 S-T 特性 |
| patch 级时间相干分析 | 测量局部方差，提取边缘并抑制噪声 |
| 结构感知 Gaussian 初始化 | 基于边缘引导 |
| 边缘加权损失 | 全程优化 |

### 事件特性

```
相机移动 → 边缘处产生一致事件
非边缘区域 → 稀疏噪声
→ patch 方差分析可有效区分
```

## 局限性

- 需要事件相机硬件
- 对低纹理场景可能受限
- 边缘提取质量影响整体性能

## 实现建议
