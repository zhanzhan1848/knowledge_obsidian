## 核心贡献

> 将量子碰撞模型与经典光线追踪结合，实现相干光-物质相互作用的渲染，突破传统渲染对材料光学性质静态假设的限制。

1. 提出基于量子碰撞模型(QCM)的光-物质相互作用框架，将光线追踪扩展到相干吸收和非线性光-物质效应
2. 推导出适合模拟二维材料界面的对称约束幺正算符
3. 提出多层交互框架，通过重复碰撞产生彩虹色效应
4. 提供可用于生产渲染管线的实用着色器实现

---

## 技术方案

### 核心思想

传统光渲染技术将材料的光学性质视为静态，但在响应入射光照而动态演变的系统中，这种假设失效。本文提出将经典光线追踪与量子碰撞模型结合，探索相干光-物质相互作用在渲染中的效果。

通过将入射光和材料激发视为量子化的模式，将次表面散射建模为一系列对称约束的幺正碰撞。这种公式允许整合非可积动力学和多层干涉效应引起的混沌光学响应。

### 关键技术

| 技术 | 说明 |
|------|------|
| 量子碰撞模型(QCM) | 将光线和材料激发建模为量子粒子，在空间和时间上局部相互作用 |
| 次表面散射建模 | 作为对称约束幺正碰撞的序列，保持材料记忆效应 |
| 多层干涉框架 | 通过重复碰撞产生彩虹色效应 |
| BSDF预计算 | 使用近-term量子计算机预计算碰撞算子生成标准BSDF |

## 公式

### 单次碰撞状态变换

光线与表面碰撞后，权重根据复散射振幅 r, t, a 在三个出射模式间分配：

```math
|F⟩ = r|1_r 0_s 0_t⟩ + a|0_r 1_s 0_t⟩ + t|0_r 0_s 1_t⟩
```

其中 r 为反射、t 为透射、a 为吸收振幅。

### 幺正演化算符

系统经过短时间 δt 后演化：

```math
|I⟩ \rightarrow e^{-iH\delta t}|I⟩
```

## 实验结论

- **应用场景**: 渲染新型物理启发材料，具有独特光学特征
- **展示效果**: 翡翠雕像涂覆新型材料节点，随层数和光线数增加产生更深色彩对比
- **量子计算**: 多层情况下量子计算仍可行，仅需预计算光学性质
