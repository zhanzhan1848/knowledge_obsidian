## 核心贡献
1. **LiDAR 中心化 3DGS 框架**：将度量几何先验显式集成到整个高斯优化过程中
2. **几何-纹理感知分配策略**：选择性地将高斯基元分配到具有高结构或外观复杂度的区域
3. **曲率自适应细化机制**：在训练期间动态引导高斯分裂向几何复杂区域
4. **置信感知度量深度正则化**：使用 LiDAR 测量将重建几何锚定到绝对尺度

## 技术方案
传统 3DGS 依赖 SfM 稀疏点云，存在尺度模糊、几何一致性有限和强视图依赖问题。本工作将 LiDAR 数据作为主动几何先验，而非被动初始化源。

**核心创新**：
- 将 3DGS 优化重构为固定表示预算下的几何条件分配和细化问题
- 几何-纹理感知分配：根据复杂度智能分配高斯基元
- 曲率自适应细化：动态调整高斯分布
- 置信感知深度正则化：确保度量精度

## 公式
```math
# 3D Gaussian Splatting 渲染
C(x) = Σ(c_i * α_i * ∏(1 - α_j))

# 深度正则化
L_depth = Σ|D_pred - D_lidar| * confidence

# 其中：
# C(x) - 像素颜色
# c_i - 高斯颜色
# α_i - 透明度
# D_pred - 预测深度
# D_lidar - LiDAR 深度
```

## 代码
```python
# 概念性伪代码
class LiDAR3DGS:
    def __init__(self, budget):
        self.budget = budget  # 高斯基元预算
        self.gaussians = []
    
    def geometry_texture_allocation(self, scene):
        # 计算几何-纹理复杂度
        complexity = self.compute_complexity(scene)
        # 分配高斯基元
        self.allocate_gaussians(complexity)
    
    def curvature_adaptive_refinement(self):
        # 计算曲率
        curvature = self.compute_curvature()
        # 动态细化
        self.refine_gaussians(curvature)
    
    def depth_regularization(self, lidar_depth):
        # 置信感知深度约束
        loss = self.depth_loss(lidar_depth)
        return loss
```

## 实验结论
- **数据集**：ScanNet++ 和自定义真实世界数据集
- **性能**：在度量尺度重建和几何保真度方面达到 SOTA
- **优势**：
  - 解决了尺度模糊问题
  - 提高了几何一致性
  - 减少了视图依赖性

## 局限性
- 依赖 LiDAR 设备
- 对于纯纹理区域可能仍有限制
- 计算开销可能增加
