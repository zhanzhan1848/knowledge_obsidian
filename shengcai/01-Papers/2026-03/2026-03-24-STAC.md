# STAC: Plug-and-Play Spatio-Temporal Aware Cache Compression

## 元信息
| 标题 | Plug-and-Play Spatio-Temporal Aware Cache Compression for Streaming 3D Reconstruction |
| 作者 | Runze Wang et al. |
| 链接 | [arXiv:2603.20284](https://arxiv.org/abs/2603.20284) |
| PDF | [PDF](https://arxiv.org/pdf/2603.20284) |
| 会议 | CVPR 2026 |

## 核心贡献
1. **时空稀疏性洞察**: 发现因果 Transformer 在 3D 重建中具有内在的时空稀疏性
2. **Working Temporal Token Caching**: 使用衰减累积注意力分数保留长期信息标记
3. **Long-term Spatial Token Caching**: 将空间冗余标记压缩为体素对齐表示
4. **Chunk-based Multi-frame Optimization**: 联合处理连续帧以提高时间一致性

## 技术方案

### 核心问题
- **流式 3D 重建**: 需要长期时间一致性和高效内存使用
- **KV Cache 问题**: 随流长度线性增长，造成内存瓶颈
- **早期驱逐**: 在有限内存预算下显著降低重建质量

### 三大组件

#### 1. Working Temporal Token Caching
- 使用衰减累积注意力分数
- 保留长期信息标记
- 自动识别重要时间标记

#### 2. Long-term Spatial Token Caching
- 压缩空间冗余标记
- 体素对齐表示
- 内存高效存储

#### 3. Chunk-based Multi-frame Optimization
- 联合处理连续帧
- 提高时间一致性
- 提高 GPU 效率

## 实验结论
- **内存减少**: ~10x
- **推理加速**: 4x
- **重建质量**: SOTA
- 显著提高实时 3D 重建的可扩展性

## 与渲染的关联
- **流式 3D 重建**: 实时渲染的关键技术
- **内存优化**: 对大规模场景渲染有价值
- **时间一致性**: 保证渲染质量
