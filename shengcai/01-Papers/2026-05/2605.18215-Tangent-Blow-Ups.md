# Tangent Blow-Ups for Processing Non-Manifold Geometry

## 元信息
| 标题 | 值 |
|------|-----|
| 作者 | Alice Petrov |
| 链接 | [arXiv:2605.18215](https://arxiv.org/abs/2605.18215) |
| 领域 | Graphics (cs.GR) |

## 核心贡献
1. 引入 **tangent blow-up**，将几何提升到环境空间与切平面Grassmannian的乘积空间
2. 在奇点处恢复结构，区分位置相同但切方向/曲率不同的高阶接触点
3. 定义离散微分算子（梯度、散度、Laplacian）于提升域中

## 技术方案
- **问题**：几何处理管线假设输入是流形，但实际数据包含边缘、角点、自相交、分支连接等奇点
- **方法**：通过切空间丛的笛卡尔积恢复奇点处结构
- **应用**：测地线计算、分割、曲面参数化、曲率估计

## 公式
```math
\text{TB}(X) = \{(x, V) \in X \times \text{Gr}(k, T_xX)\}
```
