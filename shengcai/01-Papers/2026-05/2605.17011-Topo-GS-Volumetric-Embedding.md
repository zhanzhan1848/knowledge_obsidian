# Topo-GS: Continuous Volumetric Embedding of High-Dimensional Data via Topological Gaussian Splatting

## 元信息
| 标题 | 值 |
|------|-----|
| 作者 | João Paulo Gois PhD |
| 链接 | [arXiv:2605.17011](https://arxiv.org/abs/2605.17011) |
| 领域 | Graphics (cs.GR) |

## 核心贡献
1. 用 **3D Gaussian Splatting** 将高维数据投影重铸为无网格体积重建
2. 局部几何约束替代标准光度损失
3. 正交 Procrustes 目标强制 As-Rigid-As-Possible 先验
4. 拓扑感知策略保留 1D 轨迹或 2D 曲面

## 技术方案
- **问题**：传统降维依赖离散点云，存在视觉遮挡和人为不连续性
- **方法**：将离散散点图转换为连续体积表示
- **创新**：对齐每个高斯空间协方差到局部切空间
