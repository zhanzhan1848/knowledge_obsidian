# Skinned Motion Retargeting with Spatially Adaptive Interaction Guidance

## 核心贡献

1. **空间自适应锚点重定向**: 动态重定位锚点到目标角色可达区域
2. **Transformer 锚点细化策略**: 预测锚点位移并约束在目标几何上
3. **可微软投影**: 可微分投影保持锚点在目标角色几何上
4. **图自编码器**: 基于图的结构预测目标骨骼运动

## 技术方案

### 问题背景

- 跨不同身形的运动重定向困难
- 保持交互语义（自接触、近身接近）挑战
- 静态对应方法难以处理夸张身形比例

### 方法流程

```
源角色 → 姿态依赖空间结构 → Transformer 锚点细化 → 可达区域锚点
                                                        ↓
目标角色 ← 图自编码器骨骼运动 ← 锚点引导
```

### 核心组件

1. **空间自适应锚点**
   - 动态重定位锚点
   - 考虑目标角色几何
   - 保持交互语义

2. **Transformer 锚点细化**
   - 预测锚点位移
   - 软投影到目标几何
   - 可微优化

3. **图自编码器**
   - 条件于锚点
   - 预测目标运动
   - 保持空间配置

### 训练策略

交替训练方案:
```
锚点适配 ←→ 运动重定向
(交替优化)
```

## 实验结论

- 在保持交互保真度方面优于 SOTA
- 跨不同角色几何有效
- 适用于夸张身形比例
