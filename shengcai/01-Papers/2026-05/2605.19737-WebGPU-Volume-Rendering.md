# Decentralized Direct Volume Rendering: Browser-Native GPU Architecture for MRI Digital Twins

## 元信息
| 标题 | 值 |
|------|-----|
| 作者 | Oserebameh Beckley |
| 链接 | [arXiv:2605.19737](https://arxiv.org/abs/2605.19737) |
| 领域 | Graphics (cs.GR) |

## 核心贡献
1. 提出基于 **WebGPU** 的去中心化客户端渲染架构
2. 单程 **deterministic ray marching** 直接在浏览器执行
3. 首次证明复杂3D医学模拟可在浏览器原生运行，无需深度学习或外部计算依赖

## 技术方案
- **问题**：医学数字孪生依赖服务端渲染或昂贵工作站
- **方法**：WebGPU架构在低功耗集成GPU上执行单 pass raymarching 和 morphological gradient 计算
- **性能**：TTFP < 920ms，>= 82 FPS

## 代码 (伪代码)
```glsl
// Single-pass raymarching in WebGPU
void main() {
    vec3 rayDir = getRayDirection(uv);
    vec3 p = cameraPos;
    for (int i = 0; i < MAX_STEPS; i++) {
        float density = sampleMRI(p);
        if (density > THRESHOLD) {
            color = computeShading(p, density);
            break;
        }
        p += rayDir * stepSize;
    }
}
```

## 实验结论
- Time to First Pixel (TTFP) < 920ms
- 持续交互保真度 >= 82 FPS
- 演示: https://webgpu-mri.vercel.app/
