# PersistGS: Differentiable Physics for Object Permanence in 4D Gaussian Splatting

## 核心贡献

1. **PersistGS 方法**: 将可微分刚体模拟与 3D Gaussian Splatting 耦合，解决物体被遮挡时的持久性问题
2. **场景分解**: 将场景分解为每个物体的 Gaussians 和碰撞网格
3. **物理预测**: 从遮挡前的观测轨迹通过可微分模拟估计摩擦和速度
4. **SE(3) 轨迹**: 使用生成的 SE(3) 轨迹在遮挡期间定位物体 Gaussians
5. **质心轮廓损失**: 隔离位置梯度与外观噪声，轨迹误差降低 40%

## 技术方案

### 问题
动态 3D Gaussian Splatting (3DGS) 从多相机视频重建动态场景。当运动物体被所有训练相机完全遮挡时，监督信号消失，代表该物体的 Gaussians 退化。现有方法依赖学习到的生成先验，优先考虑视觉可信度而非物理正确性。

### 方法
```
观测轨迹 → 可微分物理模拟 → SE(3) 轨迹预测 → 遮挡期 Gaussians 定位
```

1. 从预遮挡轨迹估计摩擦系数和速度
2. 使用可微分刚体模拟预测遮挡期轨迹
3. 轨迹满足刚体动力学方程，忠实地捕捉接触事件（反弹、摩擦减速、方向变化）
4. 质心轮廓损失隔离位置梯度

### 关键公式
- 轨迹误差降低: 40%
- PSNR 提升: +2.46dB (相对于常速外推)
- 与 ground truth 上界差距: 仅 0.19dB

## 实验结论

- 在合成场景中优于常速外推: +2.46dB PSNR
- 与 ground truth 轨迹上界差距仅 0.19dB
- 质心轮廓损失使轨迹误差降低 40%

## 局限性

- 依赖预遮挡期的观测数据
- 仅处理刚体运动
- 需要碰撞网格

## 实用性评估

- **实现难度**: 中 (需要可微分物理模拟器)
- **性能表现**: 轨迹精度高，接近 ground truth
- **适用场景**: 动态场景重建、机器人感知、自动驾驶
