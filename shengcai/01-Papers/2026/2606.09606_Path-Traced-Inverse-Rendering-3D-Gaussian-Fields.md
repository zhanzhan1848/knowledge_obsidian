## 核心贡献
1. **无溅射路径追踪逆渲染框架**：提出首个针对 3D Gaussian 场的无溅射（splatting-free）路径追踪逆渲染框架
2. **统一光传输管线**：前向光传输和后向梯度传播在同一射线追踪管线中定义，消除了传统方法的管线不匹配问题
3. **路径空间等价交互模型**：为重叠 Gaussian 基元定义路径空间等价交互模型，Monte Carlo 路径追踪无偏，光梯度通过相同射线追踪交互重放
4. **全渲染方程优化**：在完整渲染方程（光线追踪可见性 + 多弹跳光传输）下优化材质和紧凑球谐环境光照

## 技术方案
### 问题
- 现有逆渲染方法通过溅射估计 G-buffers，在屏幕空间优化材质，与光栅化管线绑定
- 简化渲染方程忽略间接光照，导致着色不一致、伪影、材质-光照估计不准确

### 解决思路
```
Ray tracing pipeline:
  Forward: path_traced_radiance(Gaussians, ray)
  Backward: pathwise_gradients replayed over same ray-traced interactions
```

关键创新：**路径空间等价交互模型**
- 重叠 Gaussian 基元 → 路径空间等价交互
- Monte Carlo 路径追踪无偏估计
- 梯度通过相同射线追踪交互重放（而非溅射派生屏幕空间缓冲）

## 公式
渲染方程：
```math
L_o(\mathbf{x}, \omega_o) = \int_{\Omega} f_r(\mathbf{x}, \omega_i, \omega_o) L_i(\mathbf{x}, \omega_i) (\omega_i \cdot \mathbf{n}) d\omega_i
```

路径追踪交互的梯度：
```math
\frac{\partial L}{\partial \theta} = \sum_{path} \frac{\partial L_{path}}{\partial \theta}
```

球谐环境光：
```math
L_{env}(\omega) = \sum_{l=0}^{L} \sum_{m=-l}^{l} c_{lm} Y_{lm}(\omega)
```

## 实验结论
- 竞争性的材质反演质量
- 改进的路径追踪渲染质量
- 更合理的阴影、反射和全局光照下的重光照结果

## 局限性
- 计算开销较大（路径追踪固有）
- 对复杂场景的缩放性待验证
- 需要高质量初始化

## 可行性分析
- 实现难度：高（需要完整路径追踪管线 + 梯度反传）
- 性能预期：质量高但速度较慢，适合离线应用
- 适用场景：3D 材质估计、全局光照重建、路径追踪场景编辑

## 笔记
**与墨鱼丸协作**：Path-Traced Inverse Rendering 的核心价值在于"管线一致性"——前向和后向都在同一路径追踪框架内完成，消除了传统方法中光栅化 vs 路径追踪的管线不匹配。对于需要全局光照下材质估计的应用，这是重要的方法论进步。
