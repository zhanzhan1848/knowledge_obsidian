# OT-UVGS: Revisiting UV Mapping for Gaussian Splatting as a Capacity Allocation Problem

## 核心贡献
1. **问题重新定义**：将 UV mapping 视为固定 UV 预算下的容量分配问题
2. **OT-UVGS**：基于一维最优传输的轻量级可分离映射方法
3. **O(N log N) 复杂度**：使用 rank-based sorting 实现

## 技术方案

### 现有问题 (UVGS)
- 确定性球面投影将高斯分配到 UV 位置
- 忽略全局高斯分布
- 很多 UV 槽空置，密集区域频繁碰撞

### OT-UVGS 解决方案
```
UV mapping = capacity allocation under fixed UV budget
↓
Optimal transport inspired mapping
↓
全局耦合分配 + 保留原始 UVGS 表示
```

### 实现
- rank-based sorting
- 可作为球面 UVGS 的 drop-in 替换
- 保持 UVGS 的可分离性

## 实验结果
在 184 个 object-centric 场景和 Mip-NeRF 数据集上:
- PSNR, SSIM, LPIPS 全面提升
- UV 利用率显著提高
- 非空槽比例更高
- 碰撞更少
- 高斯保留率更高
