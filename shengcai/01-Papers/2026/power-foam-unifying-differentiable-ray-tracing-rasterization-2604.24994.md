# Power Foam: Unifying Real-Time Differentiable Ray Tracing and Rasterization

## 元信息
| 标题 | Power Foam: Unifying Real-Time Differentiable Ray Tracing and Rasterization |
|------|------|
| 作者 | Shrisudhan Govindarajan 等 |
| 链接 | [arXiv:2604.24994](https://arxiv.org/abs/2604.24994) |
| 会议 | arXiv cs.GR |
| 日期 | 2026-04-27 |

## 核心贡献

1. **统一可微渲染表示**：将基于泡沫的射线追踪能力与现代光栅化管线的效率统一
2. **有界幂图 (Bounded Power Diagrams)**：将Voronoi泡沫推广到有界幂图，解决无界单元导致的光栅化效率问题
3. **可训练无需Delaunay三角化**：通过可控单元范围，避免了训练期间昂贵的Delaunay三角化
4. **定向表面公式**：显式建模内部/外部区域间的界面，解耦几何与外观
5. **可微纹理嵌入**：将可微纹理直接嵌入表面

## 技术方案

### 问题
- 之前泡沫表示支持常数时间射线遍历
- 但无界单元阻碍了高效基于tile的光栅化

### 解决方案
```
泡沫表示 → 有界幂图
    ↓
可控单元范围 + 定向表面公式
    ↓
保留射线追踪效率 + 光栅化性能
```

### 关键公式

幂图单元定义：
```math
\text{power}(x, c_i) = \|x - c_i\|^2 - w_i
```
其中 $c_i$ 是站点，$w_i$ 是权重。

有界幂图通过约束权重使得单元空间有界。

定向表面：
```math
S = \{x \in \mathbb{R}^3 \mid \phi(x) = 0\}
```
其中 $\phi$ 是有界幂图的距离函数。

## 与3DGS对比

| 特性 | 3DGS | Power Foam |
|------|------|------------|
| 射线遍历 | 全局深度排序 | 每射线排序，消除伪影 |
| 光栅化效率 | 高 | 竞争性高 |
| 可微渲染 | 支持 | 完整支持 |
| 单元边界 | 有界 | 有界（通过幂图） |

## 优势

1. **保留射线追踪效率**：常数时间射线遍历
2. **光栅化性能竞争**：与现代3DGS竞争
3. **统一实时可微渲染**：实用路径

## 局限性

- 需要控制单元格范围
- 实现复杂度较高

## 实验结论

- 在保持SOTA射线追踪效率的同时实现竞争性光栅化性能
- 为统一实时可微渲染提供实用路径
