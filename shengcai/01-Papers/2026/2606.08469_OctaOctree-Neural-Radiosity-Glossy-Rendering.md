## 核心贡献
1. **OctaOctree 表示**：提出高效空间-角度辐射表示，自适应八叉树组织 3D 空间，每空间节点关联八面体方向图
2. **反射感知先验嵌入**：将反射感知空间-角度先验直接嵌入辐射表示，减少神经网络从位置特征恢复高频视相关效应的负担
3. **单次网络查询高质量全局光照**：在主光线交点单次网络查询即可产生高质量、方向感知全局光照
4. **实时性能**：在实时性能下实现改进的逼真度

## 技术方案
### 问题
- 现有神经辐射缓存方法依赖位置特征编码或空间组织缓存
- 难以表示高频方向辐射变化（ glossy/specular 材质）
- 增加模型复杂度或采样成本才能覆盖

### 解决思路
```
OctaOctree 结构:
  - 3D Space: 自适应八叉树（细粒度局部光照/可见性变化）
  - Direction: 每空间节点 → 八面体方向图
  - 空间层次 + 角度分辨率耦合：细空间 + 粗角度 vs 粗空间 + 富角度
```

**反射感知先验**：
- 空间节点粗糙 → 角度分辨率高（捕捉 glossy/specular）
- 空间节点精细 → 局部光照和可见性变化
- 神经网络只需"补全"而非"重建"高频方向效应

## 公式
辐射表示：
```math
R(\mathbf{x}, \omega) \approx \text{MLP}(F_{oct}(\mathbf{x}), \omega)
```

其中 $F_{oct}(\mathbf{x})$ 是 OctaOctree 提取的空间特征。

空间-角度耦合损失：
```math
\mathcal{L} = \|R(\mathbf{x}, \omega) - \hat{R}_{gt}(\mathbf{x}, \omega)\|^2
```

## 实验结论
- 从漫反射互反射到清晰 glossy/specular 反射的广泛间接光照效果
- 单次网络查询即可达到高质量
- 相比基线神经辐射缓存方法改进的逼真度和实时性能

## 局限性
- 对极端复杂几何的缩放性待验证
- 八叉树构建需要额外预处理
- 对动态场景需要重建树结构

## 可行性分析
- 实现难度：中（需要八叉树 + 神经辐射场）
- 性能预期：实时性能（单次查询）
- 适用场景：实时全局光照、 glossy/specular 材质渲染、神经辐射缓存
