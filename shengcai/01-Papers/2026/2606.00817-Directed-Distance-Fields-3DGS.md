## 核心贡献
1. 将训练好的 3DGS 场景蒸馏为 Directed Distance Function (DDF)，实现常数时间的射线查询
2. DDF 是小型神经场，通过单次前向传播返回射线与表面的距离及是否命中
3. DDF 大小为 52MB，与高斯数量无关，查询成本和内存不随场景增长

## 技术方案

### 核心思想
3D Gaussian Splatting (3DGS) 只能回答主射线（从相机发出的射线），无法追踪阴影、环境光遮蔽和全局光照所需的次级射线。本文通过蒸馏 DDF 将 3DGS 场景转化为射线预言机。

### 关键技术
- **DDF 定义**：输入射线（原点和方向），输出到第一个表面的距离及是否命中
- **监督策略**：高斯渲染的深度过于模糊，无法教导薄特征；清洁的距离监督可恢复
- **性能**：DDF 比 sphere tracing SDF 快 26-72 倍，查询时间和内存不随场景增长
- **无需网格流程**：图像→3DGS 场景→神经表面→清洁距离→DDF 学习

### 公式
DDF 查询：
```math
d_{DDF}(\mathbf{o}, \mathbf{d}) \rightarrow (t, h)
```
其中 $t$ 是距离，$h$ 表示是否命中。

### 代码
```python
# DDF 前向查询示例
def query_ddf(ddf, origin, direction):
    ray = torch.stack([origin, direction], dim=-1)
    dist, hit = ddf(ray)
    return dist, hit
```

## 实验结论
- 全局光照次级射线预言机
- 阴影重建：30.3 dB
- 环境光遮蔽：21.3 dB
- 跨 142 个物体和真实捕获场景验证

## 局限性
- 需要预先训练 3DGS 场景
- DDF 训练需要干净的距离监督

## 可行性分析
- 实现难度：中（需要 3DGS 和神经场基础）
- 性能预期：查询速度常数级，不依赖场景复杂度
- 适用场景：实时全局光照、阴影、环境光遮蔽
