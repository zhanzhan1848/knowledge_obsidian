## 核心贡献
1. 提出Minimal Retroreflective Microfacet (MRM)模型，只需单一代换即可将任何现有microfacet BSDF转换为物理合理的回射材质
2. 核心替换：将视线方向替换为其关于表面法线的反射，再评估标准模型
3. 基于已发表的back-vector公式，只需最小代码修改
4. 已被OpenPBR和MaterialX材质标准采用
5. 证明在反射对称NDF假设下满足互易性和能量守恒
6. 用测量回射材质数据验证模型

## 技术方案
### 核心公式
```
将 view direction 替换为 reflect(wo, n)
然后评估标准 microfacet model
```

### 理论基础
- 基于back-vector formulation
-只需最小代码更改
- 互易性证明：适用于反射对称NDF（所有常用分布都满足）
- 能量守恒证明：同样条件下成立

### 代码实现
```cpp
// Minimal Retroreflective Microfacet
Vector3 wo_reflected = reflect(wo, n);
f_microfacet(wo_reflected, wi, n, ...);
```

## 实验结论
- 已验证与测量回射材质数据一致
- 已被OpenPBR和MaterialX采用
- 证明互易性和能量守恒

## 局限性
- 假设反射对称NDF
- 需要现有microfacet模型作为基础

## 可行性分析
- 实现难度：低（代码修改极小）
- 性能预期：无额外性能开销
- 适用场景：回射材质渲染、OpenPBR/MaterialX标准实现
