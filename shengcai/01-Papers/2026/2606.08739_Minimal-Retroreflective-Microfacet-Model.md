## 核心贡献
1. **极简回向反射微面元模型**：通过单一替换——将视线方向关于表面法线反射后再评估标准模型——将任意现有微面元 BSDF 变为物理合理的回向反射模型
2. **基于 back-vector 形式论**：基于已发表的 back-vector 公式，代码改动极小
3. **已被行业标准采纳**：已纳入 OpenPBR 和 MaterialX 材料标准
4. **严格证明**：在满足反射对称 NDF 分布假设（所有常用分布均满足）下证明了互反性和能量守恒

## 技术方案
### 标准微面元 BSDF
```math
f_r(\omega_i, \omega_o) = \frac{F(\omega_i, h) G(\omega_i, \omega_o, h) D(h)}{4 (\omega_i \cdot n) (\omega_o \cdot n)}
```

### MRM 替换
```math
f_r^{MRM}(\omega_i, \omega_o) = f_r(\omega_i, R_{n}(\omega_o))
```

其中 $R_{n}(\omega_o)$ 是将 $\omega_o$ 关于表面法线 $n$ 反射。

### 核心思想
- 回向反射 = 光线接近入射方向时反射增强
- 替换 view direction → reflected view direction 即可
- NDF 保持反射对称性：$D(h) = D(R_n(h))$

## 公式
互反性证明条件：
```math
D(h) = D(R_n(h)) \quad \Rightarrow \quad f_r^{MRM}(\omega_i, \omega_o) = f_r^{MRM}(\omega_o, \omega_i)
```

能量守恒：
```math
\int_{\Omega} f_r^{MRM}(\omega_i, \omega_o) (\omega_o \cdot n) d\omega_o \leq 1
```

## 代码
```cpp
// Minimal change to existing microfacet BRDF
// Instead of: f_r(omega_i, omega_o)
// Use: f_r(omega_i, reflect(omega_o, n))
Vector3 reflected_wo = reflect(-omega_o, normal);
float brdf_mrm = evaluate_microfacet(omega_i, reflected_wo, n, params);
```

## 实验结论
- 在测量数据上验证了模型正确性
- 已被 OpenPBR 和 MaterialX 采纳为标准
- 代码实现极简，只需替换一个方向

## 局限性
- 假设 NDF 反射对称性（但对常用分布成立）
- 对非 retroreflective 材质不适用

## 可行性分析
- 实现难度：低（只需一行代码替换）
- 性能预期：无额外开销
- 适用场景：汽车涂层、运动服装、安全装备等回向反射材质模拟
