# PBR-Conditioned Video Diffusion Refinement for Controllable and Consistent Relighting

## 基本信息
- **作者**: (待补充)

- **链接**: [arXiv](https://arxiv.org/abs/2607.08016)

## 核心贡献
1. 提出 LightCrafter: 混合视频重光照管线
2. 将视频重光照重新定义为PBR渲染的视频翻译
3. 使用CogVideoX后训练捕捉全局光照等光度先验
4. 在真实世界重光照基准上达到SOTA

## 技术方案
### 问题背景
- 视频重光照需要平衡时间一致性与光传输物理理解
- 现有方法: 逆向渲染后正向渲染 OR 视频到视频翻译
- 各自缺陷: 噪声重建/难以建模效果 OR 控制有限/不稳定

### 核心思想
```
输入视频 → PBR渲染(目标光照) → 翻译 → 最终输出
     ↑                              ↑
   原始光照                    CogVideoX后训练
                              捕捉全局光照
```

### 方法流程
```
1. PBR渲染代理视频(目标光照下)
2. CogVideoX视频翻译PBR渲染到目标
3. 合成视频对+真实非配对视频后训练
4. 输出长期时间一致性结果
```

## 实验结论
- PBR渲染本身已超越部分先前工作
- 全局光照等效果需CogVideoX捕捉
- 在真实和合成基准上均超越SOTA
