## 核心贡献
1. **3D Gaussian 射线追踪 + 神经渲染的统一框架**：结合物理渲染（3DGS ray tracing）和神经渲染，突破单一范式的局限性
2. **内禀分解先验引导材质正则化**：从学习到的逆渲染模型提取内禀分解先验，正则化 Gaussian 场的材质属性
3. **光线示踪器作为辐射引导而非最终像素**：将射线追踪输出作为结构化 3D 支架，为轻量级神经渲染器提供"辐射引导"
4. **多阶段训练策略**：大规模预训练 + 目标微调，数据集包含 2.1M 渲染帧（合成+真实世界 3D 重建）

## 技术方案
### 问题
- 纯 PBR 的 Gaussian 表示方法因重建几何不完美、材质估计和光传输估计的缺陷，难以实现真实感重光照
- 神经渲染方法缺少显式场景表示，限制了交互式编辑能力

### 解决思路
```
3DGS + Ray Tracer → Radiometric Guidance → Neural Renderer → Photorealistic Output
```

1. **内禀分解先验**：从学习逆渲染模型提取材质属性正则化 Gaussian 场
2. **射线追踪器作为引导**：不直接输出像素，而是提供辐射度量引导
3. **轻量级神经渲染器桥接**：弥合有 shading-model 约束的估计与真实感输出之间的域差距
4. **多阶段训练**：预训练 → 目标场景微调

## 公式
核心渲染方程（扩展形式）：
```math
L_o(\mathbf{x}, \omega_o) = \int_{\Omega} f_r(\mathbf{x}, \omega_i, \omega_o) L_i(\mathbf{x}, \omega_i) (\omega_i \cdot \mathbf{n}) d\omega_i
```

神经渲染器输出：
```math
\hat{L}_{neur} = G_{\theta}(G_{gs}, E_{intrinsic}, \omega_o)
```

## 实验结论
- 在真实感重光照任务上优于基于 Gaussian 的方法
- 在编辑可控性和速度上优于先前神经渲染器
- 首个支持捕获 3D 场景中交互式应用的方法（动态几何/光照/材质）

## 局限性
- 依赖预训练的大规模数据集
- 神经渲染器增加了推理开销
- 对极端光照条件下的泛化能力待验证

## 可行性分析
- 实现难度：高（需要 3DGS + 射线追踪 + 神经渲染的联合实现）
- 性能预期：交互式性能，支持动态场景编辑
- 适用场景：3D 场景重建、真实感人像/物体编辑、AR/VR 场景
