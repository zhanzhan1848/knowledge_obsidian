## 核心贡献
1. **生成式像素对齐几何表示**：提出"世界追踪"，为每个输入像素预测有序 3D 点栈（第一层=可见表面，后续层=遮挡表面的前到后交点）
2. **WT-DiT 模型**：世界追踪扩散 transformer，将多层几何作为去噪 token，通过分解注意力+全局注意力耦合
3. **像素空间流匹配 + 混合噪声调度**：平衡可见表面重建和遮挡几何生成
4. **保持 2D→3D 对应**：支持文本驱动 3D 场景编辑、几何条件新视角视频合成、与纹理网格生成器的免训练集成

## 技术方案
### 表示
```
每个像素 → 有序 3D 点栈
  layer[0]: 可见表面（与输入像素对其）
  layer[1..N]: 遮挡表面（由近到远排序）
```

### WT-DiT
- 多层几何 → 分离去噪 token
- 分解注意力：层内注意力（局部）
- 全局注意力：层间注意力（跨层协调）
- 混合噪声调度：平衡可见/遮挡信号

## 公式
流匹配目标：
```math
\tau(\mathbf{x}, t, c) = (1-t) \mathbf{x}_{clean} + t \mathbf{x}_{noisy}
```

多层表示（像素 p）：
```math
\{\mathbf{x}_p^{(0)}, \mathbf{x}_p^{(1)}, \ldots, \mathbf{x}_p^{(K)}\}
```

其中 $\mathbf{x}_p^{(0)}$ 是可见表面，$\mathbf{x}_p^{(k>0)}$ 是遮挡交点。

## 实验结论
- 在 visible-surface 重建和 complete geometry 生成上均表现优异
- 优于深度预测器和 image-to-3D 生成器
- 支持 text-driven 3D 编辑和几何条件新视角合成

## 局限性
- 扩散模型训练成本高
- 推理速度受限于 DiT
- 对极端遮挡场景的泛化能力待验证

## 可行性分析
- 实现难度：高（扩散模型 + 3D 生成）
- 性能预期：推理较慢，适合离线生成
- 适用场景：3D 资产生成、场景重建、AR/VR 内容创建

## 笔记
**与墨鱼丸协作**：World Tracing 更偏向几何重建而非传统渲染，但"像素对齐 + 多层几何"的表示方法对渲染中的可见性处理有启发。其 diffusion transformer 架构（WT-DiT）对于需要处理复杂遮挡场景的渲染应用值得关注。
