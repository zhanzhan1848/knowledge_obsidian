## 核心贡献
1. **光追踪估计复杂灯具外观**：用光追踪构建从发光体到出口表面的路径，将外观估计转化为分布学习问题
2. **归一化流网络建模出口表面辐射 PDF**：用大型归一化流网络建模出口表面出射辐射的概率密度函数，恢复辐射 = PDF × flux
3. **蒸馏到轻量级 MLP**：将学习到的外观蒸馏到轻量级 MLP，直接在出口表面估计辐射
4. **采样网络 + 混合网络**：训练采样网络用于从灯具有效直接光照计算，混合网络将灯具合成到场景

## 技术方案
### 问题
- 复杂灯具（小型发光体被多层specular层包围）难以用（双向）路径追踪渲染
- 需要极多样本才能准确捕捉复杂光传输

### 解决思路
```
Light tracing → paths from emitter to exit surface
    ↓
Model PDF of outgoing radiance on exit surface (normalizing flow)
    ↓
Radiance = estimated_pdf × flux
    ↓
Distill to lightweight MLP for fast inference
    ↓
Sampling network for direct illumination + blending network for scene composition
```

## 公式
出口表面辐射：
```math
L_o(\mathbf{x}, \omega) = p_{\theta}(\omega) \cdot \Phi(\mathbf{x})
```

其中 $p_{\theta}(\omega)$ 是归一化流网络估计的方向分布，$\Phi(\mathbf{x})$ 是表面光通量。

损失函数：
```math
\mathcal{L} = \mathbb{E}_{\omega \sim p_{gt}} [-\log p_{\theta}(\omega)] + \lambda \|L_o - \hat{L}_o\|^2
```

## 实验结论
- 低样本数即可渲染复杂灯具
- 可在任意场景中渲染挑战性灯具
- 归一化流 → MLP 蒸馏有效

## 局限性
- 需要灯具的预定义几何
- 对动态光照场景需要重新训练
- 归一化流训练复杂度高

## 可行性分析
- 实现难度：高（需要光追踪 + 归一化流 + 蒸馏）
- 性能预期：推理快（MLP），训练复杂
- 适用场景：灯具设计可视化、建筑渲染、汽车照明

## 笔记
**与墨鱼丸协作**：PureLight 用"分布学习"的方式解决复杂灯具渲染问题，是一个优雅的思路。其将归一化流蒸馏到 MLP 的做法对其他"先复杂后简化"的渲染方法有参考价值。对于需要高质量灯具渲染的应用值得关注。
